## 15-Minute Deep Dive

```markdown
# Quick Study: 15 Minutes

**Concept**: [개념 이름]

## Mission
I have 15 minutes. Give me the essence of this concept.

## Output Format

### 1. One-Sentence Definition (30 sec read)
> [Simple definition anyone can understand]

### 2. Why It Matters (1 min read)
> [Real-world relevance and importance]

### 3. How It Works (3 min read)
> [Core mechanism in simple terms]
> [Visual if helpful]

### 4. Common Misconceptions (2 min read)
> [What people get wrong]

### 5. One Example (3 min read)
> [Concrete, relatable example]

### 6. Quick Test (5 min)
> [3 questions to verify basic understanding]

## File Updates
- Update 00_CORE_CONCEPT.md with definition
- Log session in STUDY_LOG.md
- Commit: "Quick Study: [concept]"
```

## 5-Minute Refresher

```markdown
# Refresher: 5 Minutes

**Concept**: [개념 이름]

Already learned this before. Need a quick refresh.

## Output
1. Core definition (1 sentence)
2. Key mechanism (1 paragraph)
3. Most important thing to remember (1 sentence)
4. Quick self-test (1 question)

Keep it ultra-brief. No file updates needed.
```

## Instant Answer

```markdown
# Quick Answer

**Question**: [질문]

Give me:
1. Direct answer (1-2 sentences)
2. Brief explanation (1 paragraph)
3. If wrong assumption in question, correct it

No lengthy explanations. Just answer.
```

## Cheat Sheet Generator

```markdown
# Create Cheat Sheet

**Topic**: [주제]

Generate a one-page cheat sheet with:
- Key definitions (5 max)
- Essential formulas/rules (5 max)
- Common patterns (3 max)
- Gotchas to avoid (3 max)

Format for quick scanning. Use tables and bullet points.
Output to: knowledge/CHEATSHEET_[topic].md
```
