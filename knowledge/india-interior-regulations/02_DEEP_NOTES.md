## 🔬 Layer 1: AIS 상세 시험 기준

### AIS-035: 내장재 난연성 (Flammability)

```
적용 범위:
═══════════════════════════════════════════════════════════════

모든 내장재 (운전석 뒤쪽 기준):
· 시트 커버/폼
· 헤드라이닝
· 도어 트림
· 카펫
· 선바이저
· 암레스트
· 기타 내장 표피재
```

| 시험 항목 | 기준 | 시험 방법 |
|----------|------|----------|
| **연소 속도** | ≤ 100mm/min | ISO 3795 |
| **시편 크기** | 356 x 100mm | - |
| **시편 두께** | 실제 사용 두께 | 최대 13mm |
| **화염 노출** | 15초 | 38mm 버너 |
| **측정 구간** | 38mm ~ 292mm | 254mm 구간 |

```
시험 절차:

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   1. 시편 준비 (356 x 100mm, 5개)                           │
│         │                                                   │
│         ↓                                                   │
│   2. 조건 처리 (23±2°C, 50±5%RH, 24시간)                    │
│         │                                                   │
│         ↓                                                   │
│   3. 수평 연소 시험                                          │
│                                                             │
│      [버너] ─────→ [시편]                                   │
│           15초       ════════════════════                   │
│                      ↑                ↑                     │
│                    시작점           종료점                   │
│                    (38mm)          (292mm)                  │
│         │                                                   │
│         ↓                                                   │
│   4. 연소 속도 계산                                          │
│      V = (거리 / 시간) × 60 [mm/min]                        │
│         │                                                   │
│         ↓                                                   │
│   5. 판정: V ≤ 100mm/min → 합격                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### AIS-021: 내장 돌출물 (Interior Protrusion)

```
목적: 충돌 시 승객 부상 최소화

적용 부위:
═══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   Zone A: 헤드 임팩트 영역                                   │
│   ─────────────────────────────                             │
│   · IP 상단                                                 │
│   · 필러 트림 상부                                          │
│   · 헤드라이닝                                              │
│   · 선바이저                                                │
│                                                             │
│   Zone B: 무릎/하지 영역                                     │
│   ─────────────────────────────                             │
│   · IP 하단 (니 볼스터)                                      │
│   · 글로브박스                                              │
│   · 콘솔 하부                                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

| 항목 | 요구사항 | 비고 |
|------|----------|------|
| **돌출 높이** | ≤ 9.5mm (Zone A) | 25.4mm 이상은 패딩 |
| **R값** | ≥ 3.2mm | 날카로운 모서리 금지 |
| **에너지 흡수** | HIC ≤ 1000 | 헤드폼 시험 |
| **변형** | 영구 변형 한계 | 재질별 기준 |

```
헤드폼 시험 (Head Impact):

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   시험 조건:                                                 │
│   · 헤드폼 질량: 6.8kg (Adult)                              │
│   · 충돌 속도: 24.1 km/h                                    │
│   · 측정: 가속도 (3축)                                       │
│                                                             │
│   HIC 계산:                                                 │
│                                                             │
│   HIC = max [(t₂-t₁) × (1/(t₂-t₁) ∫a dt)^2.5]             │
│                                                             │
│   합격 기준: HIC ≤ 1000                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🏭 Layer 2: 인증 기관 및 절차

### ARAI vs iCAT 비교

| 항목 | ARAI | iCAT |
|------|------|------|
| **정식 명칭** | Automotive Research Association of India | International Centre for Automotive Technology |
| **설립** | 1966년 | 2006년 |
| **위치** | 푸네 (Pune) | 마네사르 (Manesar) |
| **주요 역할** | 4륜차 형식 승인 | 4륜차/2륜차 형식 승인 |
| **시험 시설** | 충돌, 배기, NVH 등 | 충돌, 배기, EMC 등 |
| **장점** | 역사/경험 풍부 | 최신 시설 |
