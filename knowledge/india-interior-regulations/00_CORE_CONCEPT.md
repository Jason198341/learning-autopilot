# CORE CONCEPT: 인도 내장재 법규

> India Automotive Interior Regulations - 인도 자동차 내장 부품 법규 총정리

---

## 🎯 왜 인도 법규를 알아야 하나요?

### 인도 시장의 중요성

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   🇮🇳 인도 자동차 시장                           │
│                                                 │
│   · 세계 4위 자동차 시장 (2024)                  │
│   · 연간 판매: 약 400만 대                       │
│   · 성장률: 연 8-10%                            │
│   · 현대/기아 점유율: 약 20%                     │
│                                                 │
│   → 인도 진출 = 법규 필수 이해!                  │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 인도 법규의 특징

```
인도 자동차 법규 체계:

┌─────────────────────────────────────────────────┐
│                                                 │
│   1️⃣ CMVR (Central Motor Vehicle Rules)        │
│      중앙 자동차 규칙 - 최상위 법규              │
│                                                 │
│   2️⃣ AIS (Automotive Industry Standards)       │
│      자동차 산업 표준 - 기술 기준                │
│                                                 │
│   3️⃣ BIS (Bureau of Indian Standards)          │
│      인도 표준국 - 제품 인증                     │
│                                                 │
│   4️⃣ TAC (Type Approval Certificate)           │
│      형식 승인 - 차량 인증                       │
│                                                 │
└─────────────────────────────────────────────────┘
```

---

## 📋 One-Sentence Definition

> **인도 내장재 법규**란, CMVR/AIS/BIS 기준에 따라 인도에서 판매되는 차량의 내장 부품이 충족해야 하는 안전, 품질, 환경 요구사항이다.

## 📜 주요 법규 체계

### CMVR (Central Motor Vehicle Rules, 1989)

```
CMVR = 인도 자동차 법규의 헌법

┌─────────────────────────────────────────────────┐
│                                                 │
│   Rule 93: 안전 부품 요구사항                    │
│   Rule 115: 배기가스 기준                        │
│   Rule 118: 소음 기준                           │
│   Rule 124: 형식 승인 절차                       │
│   Rule 125: 수입차 인증                          │
│   Rule 126: 부품 인증                           │
│                                                 │
│   ※ AIS는 CMVR의 기술 세부 기준을 정의           │
│                                                 │
└─────────────────────────────────────────────────┘
```

### AIS (Automotive Industry Standards)

```
AIS = 자동차 기술 표준 (ARAI 발행)

주요 내장재 관련 AIS:

┌──────────┬─────────────────────────────────────┐
│   AIS    │              내용                   │
├──────────┼─────────────────────────────────────┤
│ AIS-001  │ 내부 백미러 (Interior Mirror)       │
│ AIS-002  │ 외부 백미러 (Exterior Mirror)       │
│ AIS-005  │ 도어래치, 힌지, 시트벨트 앵커리지    │
│ AIS-017  │ 스티어링 임팩트 (충돌 시 안전)       │
│ AIS-020  │ 헤드레스트 (Head Restraint)         │
│ AIS-021  │ 내장 돌출물 (Interior Protrusion)   │
│ AIS-023  │ 시트 강도 (Seat Strength)           │
│ AIS-035  │ 난연성 (Flammability)               │
│ AIS-072  │ 아동용 구속장치 (Child Restraint)   │
│ AIS-098  │ 에어백 (Frontal Airbag)             │
│ AIS-099  │ 시트벨트 어셈블리                    │
│ AIS-135  │ ISOFIX 앵커리지 시스템               │
│ AIS-145  │ 사이드/커튼 에어백                   │
└──────────┴─────────────────────────────────────┘
```

## 🔺 필수 장착 장비 상세

### 1. 삼각대 (Warning Triangle)

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   📐 삼각대 규격 (AIS-018 / IS 4606)             │
│                                                 │
│              ▲                                  │
│             ╱ ╲                                 │
│            ╱   ╲    높이: 최소 400mm            │
│           ╱     ╲   변 길이: 최소 400mm         │
│          ╱───────╲  반사체 폭: 최소 25mm        │
│                                                 │
│   요구사항:                                      │
│   · 적색 반사 삼각형                             │
│   · 접이식 가능                                  │
│   · 자립 가능                                    │
│   · 차량 후방 최소 50m에 설치                    │
│                                                 │
│   의무 대상: 모든 M, N 카테고리 차량             │
│   (승용차, 버스, 트럭 등)                        │
│                                                 │
└─────────────────────────────────────────────────┘
```

## 🏷️ Key Terms (핵심 용어)

| 용어 | 영문 | 설명 |
|------|------|------|
| **CMVR** | Central Motor Vehicle Rules | 인도 중앙 자동차 규칙 (최상위 법규) |
| **AIS** | Automotive Industry Standards | 자동차 산업 표준 (기술 기준) |
| **BIS** | Bureau of Indian Standards | 인도 표준국 (인증 기관) |
| **ARAI** | Automotive Research Association of India | 인도 자동차 연구협회 (시험/인증) |
| **iCAT** | International Centre for Automotive Technology | 국제 자동차 기술 센터 |
| **TAC** | Type Approval Certificate | 형식 승인 인증서 |
| **COP** | Conformity of Production | 양산 적합성 |
| **M1** | Category M1 | 승용차 (9인 이하) |
| **M2/M3** | Category M2/M3 | 버스 (9인 초과) |
| **N1/N2/N3** | Category N | 화물차 |

## 💡 기억하세요!

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   인도 내장재 법규 핵심 3가지:                    │
│                                                 │
│   1. CMVR이 최상위, AIS가 기술 기준              │
│      → CMVR Rule 93이 안전 부품 근거             │
│                                                 │
│   2. 삼각대/구급상자/소화기 = 필수 장비           │
│      → 차종에 따라 의무/권장 구분                 │
│                                                 │
│   3. ARAI/iCAT에서 시험 및 인증                  │
│      → TAC 없으면 인도 판매 불가!                │
│                                                 │
└─────────────────────────────────────────────────┘
```
