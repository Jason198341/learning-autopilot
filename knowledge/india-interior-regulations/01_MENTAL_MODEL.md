## 🏛️ 인도 법규 체계 = 피라미드 구조

### 법규 계층 구조

```
인도 자동차 법규 피라미드:

                    ▲
                   ╱ ╲
                  ╱   ╲
                 ╱ MVA ╲         ← Motor Vehicle Act (법률)
                ╱───────╲           국회에서 제정
               ╱         ╲
              ╱   CMVR    ╲      ← Central Motor Vehicle Rules
             ╱─────────────╲        교통부에서 제정 (규칙)
            ╱               ╲
           ╱      AIS        ╲   ← Automotive Industry Standards
          ╱───────────────────╲     ARAI에서 제정 (기술 표준)
         ╱                     ╲
        ╱        BIS/IS         ╲ ← Bureau of Indian Standards
       ╱─────────────────────────╲   제품 규격 (인증 기준)
      ╱___________________________╲

위에서 아래로: 법적 구속력 ↓, 기술 상세도 ↑
```

## 🧭 Mental Shortcuts

### 1️⃣ "CMVR = 헌법, AIS = 시행령"

```
법규 이해 공식:

CMVR Rule 93 = "차량은 안전해야 한다" (원칙)
        ↓
AIS-0XX = "이렇게 시험하라" (방법)
        ↓
BIS/IS = "이 기준을 통과하라" (합격선)
```

### 2️⃣ "삼소구 = 필수 3종"

```
인도 필수 장비 암기:

삼 = 삼각대 (Warning Triangle)
소 = 소화기 (Fire Extinguisher) - 버스/트럭
구 = 구급상자 (First Aid Kit)

※ 승용차는 삼각대 필수, 나머지 권장
※ 버스/트럭은 3종 모두 필수
```

### 3️⃣ "ARAI가 시험, TAC가 증명"

```
인증 흐름:

ARAI/iCAT = 시험/심사 기관 (시험소)
TAC = 형식 승인서 (합격증)
COP = 양산 적합성 (공장 심사)

ARAI에서 시험 → 합격하면 TAC 발급 → COP로 양산 관리
```

## 📐 차량 카테고리 이해

### M/N 카테고리 분류

```
인도 차량 분류 체계:

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   M 카테고리 (승객 수송)                                      │
│   ═══════════════════════                                   │
│                                                             │
│   M1: 승용차 (9인 이하, 운전자 포함)                          │
│       → 세단, SUV, 해치백                                    │
│                                                             │
│   M2: 소형 버스 (9인 초과, GVW ≤ 5톤)                        │
│       → 미니버스, 밴                                         │
│                                                             │
│   M3: 대형 버스 (9인 초과, GVW > 5톤)                        │
│       → 시내버스, 관광버스                                   │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   N 카테고리 (화물 수송)                                      │
│   ═══════════════════════                                   │
│                                                             │
│   N1: 소형 화물 (GVW ≤ 3.5톤)                               │
│       → 픽업, 소형 트럭                                      │
│                                                             │
│   N2: 중형 화물 (3.5톤 < GVW ≤ 12톤)                        │
│       → 중형 트럭                                            │
│                                                             │
│   N3: 대형 화물 (GVW > 12톤)                                │
│       → 대형 트럭, 트레일러                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘

※ GVW = Gross Vehicle Weight (총중량)
```
