```
현재 상황:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   일반 차량 법규:                                            │
│   · AIS-035 난연성 → EV에도 동일 적용                       │
│   · AIS-021 돌출물 → EV에도 동일 적용                       │
│                                                             │
│   EV 추가 고려사항:                                          │
│   · 배터리 화재 시 내장재 연소 특성?                         │
│   · 고전압 절연 요구사항?                                   │
│   · 전자파 차폐 요구사항?                                   │
│                                                             │
│   미해결 과제:                                               │
│   1. 배터리 열폭주 시나리오 대응                            │
│   2. 내장재 전도성 관리                                     │
```

**탐구 방향**:
- AIS-156 (EV 안전) 상세 내용 확인

### Resolved 2: 삼각대 의무 적용 범위

**질문**: 어떤 차종에 삼각대가 의무인가?

**답변**: 모든 M 카테고리(승용/버스)와 N 카테고리(화물) 차량에 의무

```
삼각대 적용 범위:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   ✅ 의무 (Mandatory):                                      │
│   · M1: 승용차                                              │
│   · M2, M3: 버스                                            │
│   · N1, N2, N3: 화물차                                      │
│                                                             │
│   ❌ 비적용:                                                 │
│   · L 카테고리 (이륜/삼륜) - 별도 규정                       │
│                                                             │
│   근거: CMVR Rule 138 + AIS-018                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
