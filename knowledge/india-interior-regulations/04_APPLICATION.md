## 📝 Practice Problems

### Level 1: Basic

**Problem 1.1**: 필수 장비 확인

> 인도에서 판매하는 10인승 소형 버스(M2)에 필수로 장착해야 하는 장비는?

<details>
<summary>💡 정답 보기</summary>

```
M2 카테고리 (9인 초과, GVW ≤ 5톤) 필수 장비:

✅ 삼각대 (Warning Triangle)
   · IS 4606 규격
   · 1개 이상

✅ 구급상자 (First Aid Kit)
   · IS 16651 Category B
   · 승용차보다 내용물 많음

✅ 소화기 (Fire Extinguisher)
   · 2kg ABC 타입
   · BIS 인증 (IS 15683)
   · 1개 이상

⚠️ 주의: 승용차(M1)는 삼각대만 필수,
   버스(M2, M3)는 3종 모두 필수!
```

</details>

### Level 2: Intermediate

**Problem 2.1**: 난연성 시험 불합격 대응

> 도어 트림 소재의 AIS-035 시험 결과가 120mm/min으로 불합격되었습니다. 어떻게 대응해야 할까요?

<details>
<summary>💡 정답 보기</summary>

```
문제: 연소 속도 120mm/min > 기준 100mm/min

대응 방안:

1. 소재 변경
   ─────────────────
   · 난연제 첨가량 증가
   · 난연 등급 높은 원단 사용
   · 백킹재 변경 (난연 폼)

2. 구조 변경
   ─────────────────
   · 두께 증가 (연소 속도↓)
   · 난연 배리어 추가
   · 표면 코팅 적용

3. 재시험
   ─────────────────
   · 변경 샘플 제작
   · ARAI 재시험 신청
   · 합격 확인

실무 팁:
· 목표는 80mm/min 이하로 설계 (마진 확보)
· 인도 고온 환경 고려 (60°C 노출 후 시험)
· 공급업체와 사전 스펙 협의 필수
```

</details>

### Level 3: Advanced

**Problem 3.1**: 신규 법규 대응 전략

> 2025년부터 인도에서 후방 카메라/센서가 의무화됩니다. 기존 라인업 전체에 대응하기 위한 전략을 수립하세요.

<details>
<summary>💡 정답 보기</summary>

```
후방 카메라/센서 의무화 대응 전략:

Step 1: 영향 분석
─────────────────────────────────────────
· 적용 대상: 전 M1 카테고리
· 시행일: 2025.XX.XX (신규 모델)
· 기존 모델: 2026.XX.XX

· 현재 라인업:
  - 모델 A: 후방 카메라 기적용 ✓
  - 모델 B: 후방 센서만 적용 → 카메라 추가 필요
  - 모델 C: 미적용 → 카메라+센서 추가 필요

Step 2: 기술 대응
─────────────────────────────────────────
공통 모듈 개발:
· 범용 후방 카메라 모듈
· 디스플레이 연동 (기존 AVN 활용)
· 배선 하네스 표준화

차종별 적용:
· 모델 B: 카메라 모듈 추가
· 모델 C: 카메라 + 디스플레이 패키지

Step 3: 인증 대응
─────────────────────────────────────────
· AIS-XXX (후방 시야) 시험
· TAC 변경 신청 (일괄 처리)
· 예상 비용: 모델당 50만 INR

Step 4: 양산 대응
─────────────────────────────────────────
· 2025년 상반기: 신규 모델 적용
· 2025년 하반기: 기존 모델 순차 적용
· 원가 증가: 약 5,000~8,000 INR/대

핵심 포인트:
· 공통 모듈화로 개발비 절감
· 일괄 인증으로 시험비 절감
· 조기 대응으로 시장 경쟁력 확보
```

</details>

## ✅ 법규 대응 체크리스트

### 신차 개발 시

```
□ 차량 카테고리 확인 (M1, M2, M3, N1, N2, N3)
□ 적용 AIS 리스트업
□ 필수 장비 확인 (삼소구)
□ 안전 부품 TAC 요건 확인
□ 내장재 난연성 목표 설정
□ 돌출물/HIC 시뮬레이션
□ ARAI/iCAT 시험 일정 협의
□ 인증 비용/기간 예산 반영
□ 공급업체 시험 성적서 확보
□ COP 대응 품질 시스템 구축
```

### 법규 변경 시

```
□ 변경 법규 내용 파악
□ 적용 대상 차종 확인
□ 시행일 확인 (신규/기존 구분)
□ 기술 대응 방안 수립
□ 인증 일정 계획
□ 원가 영향 분석
□ 양산 전환 계획
□ 재고 처리 계획
□ 딜러/고객 커뮤니케이션
```

## ⚠️ Common Mistakes

| 실수 | 왜 문제인가 | 올바른 접근 |
|------|------------|------------|
| M1에 소화기 장착 안 함 | M1은 권장, M2/M3는 필수 - 혼동 주의 | 차종별 요건 명확히 구분 |
| 난연 마진 없이 설계 | 양산 편차로 불합격 위험 | 기준의 80% 이하로 설계 |
| 시험 일정 촉박하게 계획 | ARAI 대기 시간 길음 | 최소 6개월 전 시험 신청 |
| COP 준비 소홀 | 양산 불가/TAC 취소 위험 | 품질 시스템 사전 구축 |
| 법규 변경 모니터링 안 함 | 갑작스런 대응으로 비용 증가 | 정기적 법규 동향 확인 |
