## 🎯 Level Definitions

### Level 1: 입문 (0-30%)

```
이 수준에서 알아야 할 것:

□ 인도 자동차 법규 존재 인지
□ CMVR, AIS 용어 이해
□ 인도 시장 판매 시 인증 필요 인지
□ 삼각대 필수 장비 인지

확인 질문:
"인도에서 차 팔려면 뭐가 필요해요?"
→ "TAC 인증이 필요합니다" 답변 가능?
```

### Level 2: 초급 (31-50%)

```
이 수준에서 알아야 할 것:

□ MVA → CMVR → AIS → BIS 계층 이해
□ M/N 카테고리 분류
□ 삼소구 차종별 적용 구분
□ 주요 AIS 번호와 적용 부품 연결

확인 질문:
"M2 버스에 필요한 필수 장비는?"
→ "삼각대, 구급상자, 소화기 모두 필수" 답변 가능?
```

### Level 3: 중급 (51-70%) ← 현재 수준

```
이 수준에서 알아야 할 것:

■ AIS-035 난연성 기준 (100mm/min)
■ AIS-021 돌출물 기준
□ 시험 절차 상세 (시편, 조건)
□ ARAI/iCAT 인증 프로세스
□ TAC 취득 비용/기간 추정

확인 질문:
"난연성 시험 불합격 시 대응 방안은?"
→ 소재 변경, 구조 변경 등 구체적 방안 제시 가능?
```

### Level 4: 고급 (71-90%)

```
이 수준에서 알아야 할 것:

□ 모든 내장재 관련 AIS 상세 내용
□ 시험 비용 및 기간 정확한 추정
□ COP 심사 대응 방법
□ 법규 변경 모니터링 및 대응 전략
□ 실제 TAC 서류 작성 능력

확인 질문:
"신차 인도 출시 법규 대응 일정을 수립하세요"
→ 6개월 상세 일정표 작성 가능?
```

### Level 5: 전문가 (91-100%)

```
이 수준에서 알아야 할 것:

□ ARAI/교통부 담당자와 직접 소통 경험
□ TAC 취득 프로젝트 리드 경험
□ 법규 변경 사전 대응 시스템 구축
□ 공급업체 법규 교육 가능
□ 인도 법규 전문가 네트워크

확인 질문:
"인도 법규 인증팀을 구축하세요"
→ 조직, 프로세스, 시스템 설계 가능?
```
