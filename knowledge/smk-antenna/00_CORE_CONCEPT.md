## 1. SMK 안테나 개요

### 1.1 정의

```
┌─────────────────────────────────────────────────────────────────┐
│                    SMK (Smart Key) 안테나                        │
├─────────────────────────────────────────────────────────────────┤
│  정의: 차량과 스마트키(FOB) 간 무선 통신을 위한 송수신 장치       │
│                                                                 │
│  핵심 기능:                                                      │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐         │
│  │  키 위치    │ → │  인증 통신   │ → │  잠금/시동   │         │
│  │  감지       │    │  (암호화)    │    │  제어       │         │
│  └─────────────┘    └─────────────┘    └─────────────┘         │
└─────────────────────────────────────────────────────────────────┘
```

### 1.3 주파수 대역별 역할

| 주파수 | 대역 | 용도 | 통신 방향 | 거리 |
|--------|------|------|-----------|------|
| **125kHz** | LF (저주파) | 키 위치 감지, Wake-up | 차량 → 키 | 0.5~2m |
| **134.2kHz** | LF | 이모빌라이저 | 차량 ↔ 키 | 수 cm |
| **315MHz** | UHF (북미) | RKE 명령 | 키 → 차량 | 30~100m |
| **433.92MHz** | UHF (유럽/아시아) | RKE 명령 | 키 → 차량 | 30~100m |
| **2.4GHz** | UWB | 정밀 위치 측정 | 양방향 | 10~30m |

### 2.4 LF 안테나 전기적 특성

```
┌─────────────────────────────────────────────────────────────────────┐
│                  LF 안테나 전기적 파라미터                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [등가회로]                                                          │
│                                                                     │
│      ┌───────┐    ┌───────┐    ┌───────┐                           │
│  ────┤   L   ├────┤   R   ├────┤   C   ├────                       │
│      │인덕턴스│    │ 저항  │    │ 커패시│                           │
│      └───────┘    └───────┘    └───────┘                           │
│                                                                     │
│  [핵심 파라미터]                                                     │
│                                                                     │
│  1. 인덕턴스 (L)                                                     │
│     ┌──────────────────────────────────────────────────────────┐   │
│     │ L = μ₀ × μᵣ × N² × A / l                                  │   │
│     │                                                           │   │
│     │ μ₀ : 진공 투자율 (4π × 10⁻⁷ H/m)                         │   │
│     │ μᵣ : 코어 비투자율 (페라이트: 100~2000)                   │   │
│     │ N  : 권선 수 (turns)                                      │   │
│     │ A  : 코어 단면적 (m²)                                     │   │
│     │ l  : 자로 길이 (m)                                        │   │
│     └──────────────────────────────────────────────────────────┘   │
│                                                                     │
│  2. Q 팩터 (Quality Factor)                                         │
│     ┌──────────────────────────────────────────────────────────┐   │
│     │ Q = ωL / R = 2πfL / R                                     │   │
│     │                                                           │   │
│     │ 높은 Q값의 의미:                                           │   │
│     │ - 에너지 손실 적음                                         │   │
│     │ - 통신 거리 증가                                           │   │
│     │ - 선택도(Selectivity) 향상                                 │   │
│     │                                                           │   │
│     │ SMK 안테나 Q값 범위: 20~100 (페라이트 코어 기준)            │   │
│     └──────────────────────────────────────────────────────────┘   │
│                                                                     │
│  3. 공진 주파수 (Resonant Frequency)                                 │
│     ┌──────────────────────────────────────────────────────────┐   │
│     │ f₀ = 1 / (2π√(LC))                                        │   │
│     │                                                           │   │
│     │ 125kHz 공진을 위한 L-C 조합 예:                            │   │
│     │ L = 5mH, C = 324pF → f₀ = 125kHz                         │   │
│     │ L = 10mH, C = 162pF → f₀ = 125kHz                        │   │
│     └──────────────────────────────────────────────────────────┘   │
│                                                                     │
│  [통신 거리 계산]                                                    │
│                                                                     │
│     ┌──────────────────────────────────────────────────────────┐   │
│     │ H = (N × I × A) / (4π × r³)    [A/m]                      │   │
│     │                                                           │   │
│     │ H : 자계 강도                                              │   │
│     │ N : 권선 수                                                │   │
│     │ I : 전류                                                   │   │
│     │ A : 안테나 면적                                            │   │
│     │ r : 거리                                                   │   │
│     │                                                           │   │
│     │ → 거리 r³에 반비례 (Near-field 특성)                      │   │
│     │ → 1m에서 125kHz 수신 최소 자계 강도: ~1 mA/m              │   │
│     └──────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────┘
```

## 9. 용어 사전

| 용어 | 영문 | 설명 |
|------|------|------|
| SMK | Smart Key | 키를 주머니에 넣은 채 잠금/시동 가능한 시스템 |
| PEPS | Passive Entry Passive Start | 버튼 조작만으로 입출입 및 시동 |
| RKE | Remote Keyless Entry | 리모컨 버튼으로 원격 잠금/해제 |
| LF | Low Frequency | 저주파 (125kHz), 근거리 통신용 |
| UHF | Ultra High Frequency | 극초단파 (315/433MHz), 원거리 통신용 |
| UWB | Ultra-Wideband | 초광대역, 정밀 위치 측정용 |
| Q Factor | Quality Factor | 안테나 효율 지표 (높을수록 좋음) |
| Ferrite | 페라이트 | 고투자율 자성 재료 |
| FOB | Key Fob | 스마트키 단말기 (열쇠고리형) |
| ToF | Time of Flight | 비행 시간 (거리 측정용) |
| BCM | Body Control Module | 차체 제어 모듈 |
| ECU | Electronic Control Unit | 전자 제어 장치 |
| Relay Attack | 릴레이 공격 | 중계기를 이용한 해킹 방법 |
