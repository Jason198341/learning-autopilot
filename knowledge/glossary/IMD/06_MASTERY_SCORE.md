## 🎯 Level Definitions

### Level 1: 입문 (0-30%)

```
이 수준에서 알아야 할 것:

□ IMD가 뭔지 설명 가능
□ IMD vs 도장의 기본 차이
□ "금형 내 장식"의 의미 이해
□ IMD가 왜 친환경인지 설명

확인 질문:
"IMD가 뭐예요?" → 1분 내 설명 가능?
```

### Level 2: 초급 (31-50%)

```
이 수준에서 알아야 할 것:

□ IML, IMR, IMF 차이점
□ 필름 5층 구조 설명
□ IMD 공정 4단계 순서
□ IMD 적용 부품 예시 5개 이상

확인 질문:
"IMD 필름 구조를 그려보세요" → 그릴 수 있음?
```

### Level 3: 중급 (51-70%) ← 현재 수준

```
이 수준에서 알아야 할 것:

■ 필름 소재별 특성 (PET, PC, PMMA)
■ 인쇄 방식별 특성 (그라비아, 스크린)
□ 사출 조건 최적화 (온도, 속도, 보압)
□ 주요 불량 유형 5가지
□ 경제성 분석 (BEP 계산)

확인 질문:
"이 불량의 원인은?" → 분석 가능?
```

### Level 4: 고급 (71-90%)

```
이 수준에서 알아야 할 것:

□ 금형 설계 포인트 (게이트, 이젝터)
□ 필름 피딩 시스템 설계
□ 품질 검사 항목 및 기준
□ 원가 구조 분석
□ 신규 부품 적용 타당성 검토

확인 질문:
"이 부품에 IMD 도입해야 할까요?" → 판단 가능?
```

### Level 5: 전문가 (91-100%)

```
이 수준에서 알아야 할 것:

□ 터치 센서 통합 IMD 설계
□ 조명 통합 IMD 설계
□ 신규 기술 트렌드 파악
□ 문제 발생 시 근본 원인 분석
□ 타인에게 IMD 기술 교육 가능

확인 질문:
"IMD 프로젝트를 처음부터 리드할 수 있나요?"
```
