# CORE CONCEPT: IMD

> In-Mold Decoration - 금형 내 장식 기술

---

## 🎯 IMD가 뭐예요?

### 한 줄 정의

> **IMD (In-Mold Decoration)**는 플라스틱 사출 성형 시 **금형 안에서 필름으로 장식까지 동시에** 하는 기술이에요.

### 쉽게 말하면?

```
일반 방식:                        IMD 방식:
═══════════                      ═══════════

1. 플라스틱 성형                   1. 필름을 금형에 넣음
      ↓                                ↓
2. 꺼냄                           2. 플라스틱 사출
      ↓                                ↓
3. 도장 공장으로                   3. 꺼내면 완성!
      ↓                              (한 번에 끝!)
4. 도장
      ↓
5. 건조
      ↓
6. 완성

공정 5단계 → 공정 1단계로 단축!
```

---

## One-Sentence Definition

> **IMD**란, 인쇄된 필름을 금형에 삽입하고 플라스틱을 사출하여 **성형과 장식을 동시에** 완료하는 표면 처리 기술이다.

---

## 🚨 왜 IMD가 등장했나? (도입 배경)

### 문제의 시작: 도장의 한계

```
자동차 내장재를 예쁘게 만들고 싶다!
        ↓
기존 방법: 도장 (Painting)
        ↓
┌─────────────────────────────────────────────────┐
│                                                 │
│   🔴 문제 1: 환경 규제                           │
│   ─────────────────────────────────────────     │
│   · 도장 = 유기용제(신나) 사용                   │
│   · VOC (휘발성 유기화합물) 대량 배출             │
│   · EU REACH, 미국 EPA 규제 강화                 │
│   · "VOC 줄여라!" ← 정부 압박                    │
│                                                 │
│   🔴 문제 2: 원가 부담                           │
│   ─────────────────────────────────────────     │
│   · 도장 라인 = 수백억 투자                      │
│   · 공정 6단계 = 시간/인력 비용                  │
│   · 불량률 5~10% = 재작업 비용                   │
│   · "원가 줄여라!" ← 경영진 압박                 │
│                                                 │
│   🔴 문제 3: 품질 문제                           │
│   ─────────────────────────────────────────     │
│   · 도장 박리, 오렌지필, 먼지 불량               │
│   · 색상 편차 (Lot마다 다름)                     │
│   · "품질 올려라!" ← 고객 클레임                 │
│                                                 │
└─────────────────────────────────────────────────┘
        ↓
    해결책이 필요해!
        ↓
    💡 IMD 등장!
```

### 도장 vs IMD: 왜 바꿔야 하나?

```
도장 공장의 현실:
═══════════════════════════════════════════════════

                    ┌─────────────┐
                    │   도장 부스  │
                    │  (클린룸)   │
                    └─────────────┘
                          │
    ┌─────────────────────┼─────────────────────┐
    │                     │                     │
    ↓                     ↓                     ↓
💨 VOC 배출            💰 비용               ⚠️ 불량
· 톨루엔, 자일렌       · 설비 300억+         · 먼지 흡착
· 작업자 건강 문제     · 운영비 높음         · 흘러내림
· 환경부 규제 대상     · 에너지 소비 큼       · 색상 편차


IMD의 해결:
═══════════════════════════════════════════════════

                    ┌─────────────┐
                    │  사출기 1대  │
                    │  + 필름     │
                    └─────────────┘
                          │
    ┌─────────────────────┼─────────────────────┐
    │                     │                     │
    ↓                     ↓                     ↓
🌱 VOC 90%↓           💵 원가 30%↓          ✅ 불량 2%
· 용제 거의 없음       · 공정 통합           · 필름 품질 균일
· 친환경 인증 가능     · 설비 단순           · 박리 없음
· 규제 걱정 없음       · 인력 절감           · 색상 일정
```

### 업계가 IMD를 선택한 이유

| 압박 요인 | 도장의 문제 | IMD의 해결 |
|----------|------------|-----------|
| **환경 규제** | VOC 배출로 벌금/제재 위험 | VOC 90% 감소, 규제 통과 |
| **원가 절감** | 6공정, 고비용 설비 | 1공정, 저비용 |
| **품질 요구** | 불량률 5-10% | 불량률 1-3% |
| **납기 단축** | 도장 건조 대기 시간 | 즉시 출하 가능 |
| **디자인 다양화** | 색상 변경 = 라인 교체 | 필름만 바꾸면 OK |

### 결론: IMD는 선택이 아닌 필수

```
┌─────────────────────────────────────────────────┐
│                                                 │
│   과거: "도장이 더 예쁘니까 도장!"               │
│                  ↓                              │
│   현재: "환경규제 + 원가 + 품질 = IMD 필수!"     │
│                  ↓                              │
│   미래: "기능 통합 (터치/조명) = IMD 진화!"       │
│                                                 │
│   ※ 도장은 점점 줄고, IMD는 점점 늘어나는 중     │
│                                                 │
└─────────────────────────────────────────────────┘
```

## Why It Matters (왜 중요한가요?)

### 🏭 자동차 산업에서의 위치

```
자동차 내장재 표면 처리 방법:

┌─────────────────────────────────────────────────┐
│                                                 │
│   1. 도장 (Painting)                            │
│      · 전통적 방식                               │
│      · VOC 발생 (환경 문제)                      │
│      · 공정 많음                                 │
│                                                 │
│   2. IMD (In-Mold Decoration) ← 떠오르는 기술!   │
│      · 친환경 (VOC 감소)                         │
│      · 공정 단순                                 │
│      · 디자인 자유도                             │
│                                                 │
│   3. 성형색 (무도장)                             │
│      · 가장 저렴                                 │
│      · 디자인 제한                               │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 📊 중요한 이유 4가지

| 관점 | 왜 중요? | 수치 |
|------|----------|------|
| **환경** | VOC 배출 감소 | 도장 대비 90%↓ |
| **원가** | 공정 단축으로 비용 절감 | 20-30% 절감 |
| **품질** | 균일한 외관, 박리 없음 | 불량률↓ |
| **디자인** | 복잡한 패턴/질감 가능 | 자유도↑ |

## IMD의 종류

### 🎬 IMD vs IML vs IMF

> 💡 비슷한 이름이 많아서 헷갈려요! 정리해드릴게요.

```
┌──────────────────────────────────────────────────────┐
│               In-Mold 기술 가족                       │
├──────────────────────────────────────────────────────┤
│                                                      │
│   IMD (In-Mold Decoration)                          │
│   └── 넓은 의미: 금형 내 장식 기술 전체               │
│       └── 좁은 의미: 필름이 남는 방식                 │
│                                                      │
│   IML (In-Mold Labeling)                            │
│   └── 라벨(인쇄 필름)이 표면에 완전 부착              │
│   └── 필름이 제품의 일부가 됨                        │
│                                                      │
│   IMF (In-Mold Film)                                │
│   └── 3D 성형된 필름 사용                           │
│   └── 복잡한 형상에 적용                            │
│                                                      │
│   IMR (In-Mold Release)                             │
│   └── 잉크만 전사, 필름은 분리                       │
│   └── 얇은 잉크층만 표면에                          │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 비교표

| 방식 | 필름 상태 | 두께 | 적용 |
|------|----------|------|------|
| **IML** | 제품에 부착 | 두꺼움 (0.2-0.5mm) | 가전, 자동차 |
| **IMR** | 분리 (잉크만 전사) | 매우 얇음 | 휴대폰 케이스 |
| **IMF** | 3D 프리폼 | 중간 | 복잡한 형상 |

## Key Terms (핵심 용어)

| 용어 | 영문 | 설명 |
|------|------|------|
| **캐리어 필름** | Carrier Film | 인쇄를 담는 베이스 필름 |
| **잉크층** | Ink Layer | 실제 무늬/색상 인쇄 |
| **하드코트** | Hard Coat | 표면 보호층 (스크래치 방지) |
| **접착층** | Adhesive Layer | 플라스틱과 붙는 층 |
| **전사** | Transfer | 무늬를 옮기는 것 |
| **프리폼** | Preform | 미리 형태를 만든 필름 |
| **게이트** | Gate | 플라스틱이 들어오는 입구 |
| **웰드라인** | Weld Line | 플라스틱 흐름 합류선 |
