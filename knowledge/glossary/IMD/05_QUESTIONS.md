## 🔒 Resolved Questions (답을 찾은 질문)

### Resolved 1: PP 재질에 IMD 적용 가능한가?

**질문**: 비극성인 PP에 IMD를 적용할 수 있을까?

**답변**: 예, 가능합니다. 단, 전용 필름이 필요합니다.

```
PP IMD 솔루션:
┌─────────────────────────────────────────────────┐
│                                                 │
│   PP의 문제:                                     │
│   · 비극성 → 일반 접착 불가                      │
│   · 표면 에너지 낮음                             │
│                                                 │
│   해결책:                                        │
│   1. PP 전용 접착층                              │
│      · 염화 PP (CPP) 계열                        │
│      · 반응성 접착제                             │
│                                                 │
│   2. 표면 처리                                   │
│      · 코로나 처리                               │
│      · 플라즈마 처리                             │
│                                                 │
│   3. 올레핀계 필름                               │
│      · PP와 동종 → 상용성 우수                   │
│                                                 │
│   결론: PP IMD 상용화 완료!                      │
│         (도어 트림, 필러 트림 등 적용 중)         │
│                                                 │
└─────────────────────────────────────────────────┘
```

**참고 자료**: 필름 제조사 기술 자료, 양산 적용 사례

### Resolved 2: IMD로 3D 곡면 적용이 가능한가?

**질문**: 복잡한 3D 형상에도 IMD를 적용할 수 있을까?

**답변**: IMF (In-Mold Film) 기술로 가능합니다.

```
평면 IMD vs 3D IMF:
┌─────────────────────────────────────────────────┐
│                                                 │
│   IMD (평면/완만한 곡면):                        │
│                                                 │
│      ┌─────────────┐                            │
│      │  평면 필름   │ → 금형에 삽입 → 사출       │
│      └─────────────┘                            │
│                                                 │
│      적용: 센터 페시아, 클러스터 베젤 등          │
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│   IMF (3D 곡면):                                │
│                                                 │
│      ┌─────────────┐     ┌───────────┐         │
│      │  평면 필름   │ →  │ 프리폼     │ → 사출   │
│      └─────────────┘     │ (3D 성형) │         │
│                          └───────────┘         │
│                                                 │
│      추가 공정: 열성형 (프리폼)                  │
│      적용: 도어 핸들 커버, 스피커 그릴 등         │
│                                                 │
└─────────────────────────────────────────────────┘
```

**핵심**: 프리폼(열성형) 공정 추가로 3D 형상 대응 가능

### Resolved 3: IMD 필름의 스크래치 내성은?

**질문**: IMD 표면이 쉽게 긁히지 않을까?

**답변**: 하드코트층으로 충분한 내스크래치성 확보 가능

```
IMD 하드코트 성능:
┌─────────────────────────────────────────────────┐
│                                                 │
│   하드코트 종류:                                 │
│                                                 │
│   1. 열경화형                                   │
│      · 경도: H~2H                               │
│      · 일반적 사용                               │
│                                                 │
│   2. UV 경화형                                  │
│      · 경도: 2H~3H                              │
│      · 고급 적용                                 │
│                                                 │
│   3. 실리콘 하드코트                             │
│      · 경도: 3H 이상                            │
│      · 최고급 적용                               │
│                                                 │
│   시험 결과:                                     │
│   · 연필 경도: H~2H (합격)                       │
│   · 러빙 테스트: 100회 이상 (합격)               │
│   · 도장 대비: 동등 이상                         │
│                                                 │
└─────────────────────────────────────────────────┘
```
