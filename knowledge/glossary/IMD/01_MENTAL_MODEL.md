## 🖼️ IMD = 스티커 붙이기의 업그레이드 버전

### 일상 비유

```
어릴 때 스티커 붙여본 적 있죠?

일반 스티커 방식:              IMD 방식:
═══════════════              ═══════════════

┌─────────────┐              ┌─────────────┐
│   스티커    │              │    필름     │
│    ↓       │              │     ↓      │
│  물건 위에  │              │  금형 안에  │
│   붙임     │              │   넣음     │
└─────────────┘              └─────────────┘

문제:                        장점:
· 모서리 벗겨짐               · 플라스틱과 일체화
· 공기방울                   · 절대 벗겨지지 않음
· 오래되면 들뜸               · 영구적

"스티커를 플라스틱 안에 녹여 넣은 것!"
```

    │   └─────────────────────┘    │
    │                               │

## 📐 필름 단면 구조

### 필름 레이어 분석

```
IMD 필름의 층 구조 (단면):

┌─────────────────────────────────────────────┐
│  1. 캐리어 필름 (Carrier)                    │
│     · PET 필름 (투명)                        │
│     · 사출 후 분리되거나 남음                 │
├─────────────────────────────────────────────┤
│  2. 이형층 (Release Layer)                   │
│     · 캐리어와 분리를 위한 층 (IMR 방식)      │
├─────────────────────────────────────────────┤
│  3. 하드코트 (Hard Coat)                     │
│     · 표면 보호 (스크래치 방지)               │
│     · 광택 조절                              │
├─────────────────────────────────────────────┤
│  4. 인쇄층 (Print Layer)                     │
│     · 무늬, 색상, 패턴                       │
│     · 그라비아/스크린 인쇄                    │
├─────────────────────────────────────────────┤
│  5. 접착층 (Adhesive Layer)                  │
│     · 플라스틱 기재와 결합                    │
│     · 열에 의해 활성화                       │
└─────────────────────────────────────────────┘
         │
         ↓
   ┌──────────────┐
   │  플라스틱 기재 │ ← 사출되는 부분
   │  (ABS, PC 등) │
   └──────────────┘
```

## 🆚 IMD vs 도장 비교

### 프로세스 비교

```
도장 방식 (기존):
══════════════════════════════════════════════

사출 → 탈지 → 프라이머 → 베이스 → 클리어 → 건조
 │      │       │         │        │       │
 ↓      ↓       ↓         ↓        ↓       ↓
1일    2시간   1시간     1시간    1시간   4시간

총 6단계, 약 1~2일 소요
VOC 발생 ⚠️


IMD 방식 (새로운):
══════════════════════════════════════════════

필름 삽입 + 사출 → 완성!
        │
        ↓
      30초~1분

총 1단계, 30초~1분 소요
VOC 거의 없음 ✓
```

### 비교표

| 항목 | 도장 | IMD |
|------|------|-----|
| 공정 수 | 6+ | 1 |
| 시간 | 1~2일 | 1분 |
| VOC | 많음 | 거의 없음 |
| 불량률 | 5~10% | 1~3% |
| 초기 투자 | 적음 | 큼 (필름/금형) |
| 유연성 | 색상 변경 쉬움 | 변경 어려움 |

## 🧭 Mental Shortcuts

### 1️⃣ "IMD = 문신"
```
스티커 = 붙였다 떼는 것 (임시)
IMD = 피부에 새긴 문신 (영구)

→ 플라스틱 안에 무늬가 새겨진 것!
```

### 2️⃣ "필름 + 플라스틱 = 완성품"
```
IMD 공식:

인쇄된 필름 + 용융 플라스틱 = 장식된 부품
    (A)    +     (B)     =    (A+B)

한 번에 끝!
```

### 3️⃣ "도장의 대체재"
```
도장이 왜 문제?
· 환경 오염 (VOC)
· 공정 많음 (비용)
· 불량 많음 (품질)

IMD가 왜 해결책?
· 친환경 (VOC 없음)
· 공정 단순 (비용↓)
· 일체화 (불량↓)
```
