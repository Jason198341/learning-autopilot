## 🎯 Primary Analogy: 물에 뜨는 문신 스티커

### This concept is like...

```
[IMD] ≈ [어릴 때 물 묻혀 붙이는 임시 문신]
```

### 왜 이 비유가 맞을까요?

```
임시 문신 스티커                    IMD 필름
═════════════════                  ═════════════

1. 물을 묻힌다                      1. 열을 가한다
        │                                 │
        ↓                                 ↓
2. 피부에 대고 누른다                2. 금형에 넣고 사출
        │                                 │
        ↓                                 ↓
3. 종이를 떼면                      3. 캐리어를 떼면
   무늬가 피부에!                      무늬가 플라스틱에!

공통점:
· 무늬가 "전사"됨
· 베이스(종이/캐리어)는 분리
· 열/물로 접착 활성화
```

### 차이점

| 임시 문신 | IMD |
|----------|-----|
| 물로 활성화 | 열/압력으로 활성화 |
| 며칠 후 지워짐 | 영구적 |
| 피부 위에 | 플라스틱과 일체화 |

## 🏗️ IMD = 건축의 대리석 무늬 필름

### 비유: 인테리어 시트

```
집 인테리어                         자동차 인테리어
═══════════                        ═══════════════

┌─────────────────┐               ┌─────────────────┐
│  가구 리폼 시트   │               │    IMD 필름     │
│                 │               │                 │
│  · 대리석 무늬    │               │  · 우드그레인    │
│  · 우드 무늬     │               │  · 카본룩       │
│  · 콘크리트 무늬  │               │  · 메탈릭       │

가구 시트의 문제:                   IMD의 장점:
· 모서리 들뜸                      · 영구 밀착
· 기포 생김                        · 기포 없음
· 오래되면 벗겨짐                   · 벗겨지지 않음

왜? IMD는 플라스틱 "안에" 들어가니까!
```

## 🔗 Pattern Recognition

### 패턴 1: "표면 장식의 진화"

```
시대별 표면 장식 방법:

1세대: 도장 (Painting)
       ═══════════════
       액체를 뿌림 → 건조
       문제: VOC, 공정 많음

2세대: 수압 전사 (Water Transfer)
       ═══════════════
       물에 떠 있는 필름으로 전사
       문제: 복잡, 불균일

3세대: IMD (In-Mold Decoration)
       ═══════════════
       금형 안에서 한 번에
       장점: 친환경, 단순

4세대: 디지털 프린팅 (개인화)
       ═══════════════
       맞춤형 소량 생산
       트렌드: On-demand 제조
```

### 패턴 2: "공정 통합의 트렌드"

```
과거:  성형 → 이동 → 후가공 → 이동 → 조립
현재:  성형 + 후가공 = 동시 (IMD)
미래:  성형 + 장식 + 기능 = 동시 (스마트 IMD)

모든 산업에서 반복되는 패턴:
"공정을 합치면 비용이 줄고 품질이 올라간다"
```

## 💡 Metaphors

### Technical Metaphor
> "IMD는 플라스틱에 **영구적인 화장**을 하는 것과 같다. 도장이 매일 하는 화장이라면, IMD는 반영구 화장(문신)이다. 한 번 하면 지워지지 않고, 더 균일하고, 더 오래간다."

### Everyday Metaphor
> "IMD는 **책에 비닐 커버 씌우기**와 비슷해요. 그런데 커버가 책 표지와 완전히 합쳐져서 절대 벗겨지지 않는 거예요. 그리고 커버에 예쁜 무늬가 이미 프린팅되어 있어서 책이 더 고급스러워 보이죠."

### Teaching Metaphor
> "샌드위치를 생각해보세요. 빵 사이에 햄, 치즈, 야채가 있죠? IMD는 플라스틱(빵) 사이에 예쁜 필름(햄)을 넣는 거예요. 먹으면(성형하면) 다 합쳐져서 하나가 되는 거죠!"

## 🎭 Where Analogies Break

### 비유가 안 맞는 부분

| 비유 | 한계점 |
|------|--------|
| **임시 문신** | IMD는 영구적 (문신은 임시) |
| **열전사** | IMD는 3D 형상 가능 |
| **래핑 필름** | IMD는 일체화 (래핑은 표면만) |

### IMD만의 특별한 점

```
다른 기술과 다른 IMD만의 특징:

1. 성형과 동시에
   → 별도 공정 불필요
   → 다른 기술은 후가공

2. 3D 형상에 적용
   → 복잡한 곡면 OK
   → 2D 프린팅의 한계 극복

3. 플라스틱과 일체화
   → 밀착력 최고
   → 박리 걱정 없음

4. 양산 효율
   → 사이클 타임 그대로
   → 대량 생산에 최적
```

## 🧩 Transfer Learning

### IMD에서 배우는 범용 원리

```
1. 공정 통합의 가치
   ─────────────────
   "따로 하던 걸 합치면 비용↓ 품질↑"
   → 모든 제조업에 적용 가능

2. 친환경 = 경쟁력
   ─────────────────
   "환경 규제 대응 = 비즈니스 기회"
   → VOC 감소가 IMD의 킬러 앱

3. 필름/코팅의 힘
   ─────────────────
   "표면이 제품의 인상을 결정"
   → 표면 처리 기술의 중요성
```
