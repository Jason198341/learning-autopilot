## 🚙 Real-World Use Cases

### Use Case 1: 센터 페시아 우드그레인

**Context**: 고급 세단의 센터 페시아에 우드그레인 패턴 적용

```
요구사항:
· 천연 나무 같은 외관
· 고급스러운 질감
· 균일한 품질
· 대량 생산 (월 2만대)

옵션 비교:

┌────────────────────────────────────────────────────────┐
│  옵션 1: 천연 우드                                      │
│  ─────────────────                                     │
│  · 진짜 나무 베니어 적용                                │
│  · 비용: ★★★★★ (매우 비쌈)                            │
│  · 균일성: ★☆☆☆☆ (나무마다 다름)                      │
│  · 결론: 최고급차에만 적용                              │
├────────────────────────────────────────────────────────┤
│  옵션 2: 수압 전사                                      │
│  ─────────────────                                     │
│  · 물에 뜬 필름으로 전사                                │
│  · 비용: ★★★☆☆ (보통)                                │
│  · 균일성: ★★★☆☆ (공정 변수)                         │
│  · 결론: 중급차에 적용                                  │
├────────────────────────────────────────────────────────┤
│  옵션 3: IMD ★ 선택                                    │
│  ─────────────────                                     │
│  · 인쇄 필름 금형 내 성형                               │
│  · 비용: ★★☆☆☆ (저렴)                                │
│  · 균일성: ★★★★★ (최고)                              │
│  · 결론: 양산차에 최적!                                 │
└────────────────────────────────────────────────────────┘
```

**적용 결과:**
- 원가 30% 절감 (도장 대비)
- 불량률 2% (도장 8% 대비)
- 환경 규제 완벽 대응

### Use Case 2: 에어컨 조작 베젤

**Context**: 터치 패널 느낌의 에어컨 조작부

```
기존 방식 (도장 + 레이저):

   ┌─────────────────────────────┐
   │    TEMP  FAN  MODE          │
   │    [↑]  [🌀]  [❄]          │
   │    [↓]                      │
   └─────────────────────────────┘

   공정: 성형 → 도장 → 레이저 마킹
   문제: 도장 박리, 레이저 변색


IMD 방식 (필름 일체형):

   ┌─────────────────────────────┐
   │    TEMP  FAN  MODE          │
   │    [↑]  [🌀]  [❄]          │
   │    [↓]                      │

   공정: 필름+사출 (원샷)
   장점: 박리 없음, 조명 통합 가능
```

**고급 버전 - 터치 센서 통합:**
```
┌─────────────────────────────────────────┐

│   │ 인쇄층 (무늬)                    │  │

│   │ ITO 패턴 (터치 센서) ← NEW!      │  │
│   ├─────────────────────────────────┤  │
│   │ 접착층                          │  │
│   └─────────────────────────────────┘  │
│                                         │
│   결과: 버튼 없는 슬릭한 터치 패널!       │
│                                         │
└─────────────────────────────────────────┘
```

## 📝 Practice Problems

### Level 1: Basic

**Problem 1.1**: IMD vs 도장 선택

> 월 5천 개 생산하는 내장 부품에 우드그레인을 적용하려 합니다. IMD와 도장 중 무엇을 선택해야 할까요?

<details>
<summary>💡 정답 보기</summary>

```
분석:
· 월 5천 개 = 연 6만 개
· IMD 손익분기점: 약 3만 개

결론: IMD 선택!

이유:
1. 6만 개 > 3만 개 → 투자비 회수 가능
2. 공정 단순 → 품질 안정
3. 친환경 → 규제 대응

다만, 디자인 변경이 잦다면 도장이 나을 수 있음
(필름 금형/인쇄판 변경 비용 고려)
```

</details>

### Level 2: Intermediate

**Problem 2.1**: 불량 원인 분석

> IMD 부품에서 아래 불량이 발생했습니다. 원인과 대책은?

```
불량 현상:
┌─────────────────────────────────┐
│                                 │
│    ~~~~  ← 무늬가 밀려있음       │
│    ____     (정상 위치 아님)     │
│                                 │
└─────────────────────────────────┘
```

<details>
<summary>💡 정답 보기</summary>

```
불량: 필름 밀림 (Film Shift)

가능한 원인:
1. 필름 고정 불량
   · 진공 흡착 압력 부족
   · 정전기 발생

2. 사출 조건 문제
   · 사출 속도 너무 빠름
   · 게이트 위치 부적절

3. 금형 문제
   · 필름 가이드 마모
   · 클램핑 불량

대책:
1. 진공압 확인 및 증가 (-0.08MPa 이상)
2. 초기 사출 속도 감소 (30% 이하)
3. 필름 고정 핀/가이드 점검
4. 정전기 제거 장치 추가
```

</details>

### Level 3: Advanced

**Problem 3.1**: 신규 IMD 프로젝트 검토

> 다음 조건의 부품에 IMD 적용을 검토하세요.

```
부품: 도어 트림 우드 가니쉬
재질: PP-TD20
크기: 300 x 50 x 5mm
물량: 월 3만 개
현재: 수압 전사 (부품당 800원)
```

<details>
<summary>💡 정답 보기</summary>

```
Step 1: 기술 검토
─────────────────
· PP 재질: IMD 접착 가능 (PP 전용 필름 필요)
· 크기: 적정 (대형 아님)
· 형상: 2D에 가까움 (IMD 적합)
→ 기술적으로 가능!

Step 2: 경제성 검토
─────────────────
초기 투자:
· 필름 금형 개조: 1,500만원
· 피딩 장치: 3,000만원
· 필름 개발: 1,000만원
총 투자: 5,500만원

운영 비용:
· 수압 전사: 800원/개
· IMD 예상: 500원/개
· 절감: 300원/개

손익 분기:
5,500만원 ÷ 300원 = 18.3만 개
월 3만 개 → 6.1개월 회수!

Step 3: 결론
─────────────────
✓ IMD 전환 권장
· 6개월 투자 회수
· 품질 향상 기대
· 친환경 대응
```

</details>

## ✅ IMD 적용 검토 체크리스트

### 기술 검토
```
□ 기재 재질 확인 (ABS, PC 최적 / PP 가능)
□ 부품 형상 복잡도 (3D 심하면 IMF 검토)
□ 표면 요구사항 (광택, 질감, 내구성)
□ 내열/내약품 요구사항
□ 조명/터치 통합 여부
```

### 경제성 검토
```
□ 예상 물량 (연 3만 개 이상 권장)
□ 현재 공법 비용
□ 초기 투자비 산출
□ 손익 분기점 계산
□ 디자인 변경 빈도
```

### 품질 검토
```
□ 필름-기재 접착력 시험
□ 내광성/내열성 시험
□ 스크래치/마모 시험
□ 외관 품질 기준 (이물, 기포, 밀림)
□ 양산 불량률 목표
```

## ⚠️ Common Mistakes

| 실수 | 왜 문제인가 | 올바른 접근 |
|------|------------|------------|
| PP에 범용 필름 적용 | 접착 불량 | PP 전용 필름 사용 |
| 3D 곡면에 평면 필름 | 주름 발생 | 프리폼 (IMF) 검토 |
| 게이트를 필름 측에 | 잉크 밀림 | 게이트 반대 측 배치 |
| 금형 온도 너무 높음 | 필름 변형 | 60°C 이하 유지 |
| 소량에 IMD 적용 | 투자비 미회수 | 물량 확인 후 결정 |
