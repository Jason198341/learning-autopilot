# DEEP NOTES: IMD

## 🔬 Layer 1: 필름 소재 및 구조

### 캐리어 필름 종류

| 소재 | 두께 | 특징 | 적용 |
|------|------|------|------|
| **PET** | 50-125μm | 투명, 저렴, 범용 | 대부분 IMD |
| **PC** | 125-250μm | 내열, 내충격 | 고급 IMD |
| **PMMA** | 200-500μm | 고광택, 투명 | 고급 외관 |
| **ABS** | 300-500μm | 성형성 좋음 | IMF용 |

### 하드코트 물성

| 물성 | 기준 | 측정법 |
|------|------|--------|
| **연필 경도** | H~2H | ASTM D3363 |
| **내마모성** | 500회 이상 | Taber 마모 |
| **내약품성** | 변색 없음 | 핸드크림 48h |
| **내광성** | ΔE < 1.5 | 제논 300h |
| **광택도** | 50-90 GU | 60° 광택 |

## 🏭 Layer 2: 공정 기술

### IMD 공정 변수

```
사출 조건 최적화:

┌─────────────────────────────────────────────────┐
│                                                 │
│   금형 온도                                      │
│   ├── 고정측: 40-60°C (필름 측)                 │
│   └── 가동측: 50-70°C (기재 측)                 │
│       ※ 필름 측이 너무 뜨거우면 변형!            │
│                                                 │
│   사출 속도                                      │
│   ├── 초기: 느리게 (필름 밀림 방지)              │
│   └── 충전: 보통~빠르게                         │
│       ※ 웰드라인 위치 고려                      │
│                                                 │
│   보압/냉각                                      │
│   ├── 보압: 적정 (과하면 필름 변형)              │
│   └── 냉각: 충분히 (변형 방지)                  │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 금형 설계 포인트

| 항목 | 권장 사항 | 이유 |
|------|----------|------|
| **게이트 위치** | 필름 반대측 | 필름 밀림 방지 |
| **게이트 타입** | 핀포인트/서브마린 | 게이트 자국 최소화 |
| **이젝터** | 필름 없는 부위 | 필름 손상 방지 |
| **언더컷** | 최소화 | 필름 찢김 방지 |
| **R값** | R0.5 이상 | 필름 밀착 |

### 필름 피딩 시스템

```
자동 피딩 방식:

1. 롤 피딩 (Roll Feeding)
   ═══════════════════════════════════

   [롤] → [텐션 조절] → [위치 정렬] → [금형]

   장점: 연속 생산, 대량 생산
   단점: 필름 낭비, 정밀도 한계

2. 시트 피딩 (Sheet Feeding)
   ═══════════════════════════════════

   [적재] → [픽업] → [위치 결정] → [금형]

   장점: 정밀 위치, 소량 생산
   단점: 속도 느림, 비용 높음

3. 로봇 피딩 (Robot Feeding)
   ═══════════════════════════════════

   [스택] → [로봇 픽업] → [비전 정렬] → [삽입]

   장점: 유연성, 복잡 형상
   단점: 투자비 높음
```

## ⚙️ Layer 3: 품질 관리

### 주요 불량 유형

```
IMD 불량 유형과 원인:

┌─────────────────────────────────────────────────┐
│  1. 필름 밀림 (Film Shift)                       │
│     ─────────────────────                       │
│     원인: 사출 압력, 필름 고정 불량               │
│     대책: 진공 흡착, 게이트 위치 변경             │
│                                                 │
│  2. 주름 (Wrinkle)                              │
│     ─────────────────────                       │
│     원인: 필름 텐션, 열변형                      │
│     대책: 필름 예열, 텐션 조절                   │
│                                                 │
│  3. 기포 (Air Trap)                             │
│     ─────────────────────                       │
│     원인: 필름-금형 사이 공기                    │
│     대책: 진공 흡착, 사출 속도                   │
│                                                 │
│  4. 잉크 흐름 (Ink Wash-out)                    │
│     ─────────────────────                       │
│     원인: 고온/고압에 잉크 밀림                  │
│     대책: 잉크 내열성, 사출 조건                 │
│                                                 │
│  5. 접착 불량 (Delamination)                    │
│     ─────────────────────                       │
│     원인: 접착층 불량, 기재 호환성               │
│     대책: 필름-기재 매칭, 접착층 개선            │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 검사 항목

| 검사 항목 | 방법 | 기준 |
|----------|------|------|
| **외관** | 육안/조명 | 이물, 흠집 없음 |
| **밀착성** | 크로스컷 | 5B (박리 없음) |
| **경도** | 연필 경도 | H 이상 |
| **내마모** | 러빙 테스트 | 100회 이상 |
| **내약품** | 침적 테스트 | 변색/팽윤 없음 |
| **내열** | 고온 노출 | 80°C 24h OK |
| **위치 정밀도** | 치수 측정 | ±0.3mm |

## 🔮 Layer 5: 최신 기술 트렌드

### 고기능 IMD

```
1. 터치 센서 통합 IMD
   ═══════════════════════════════════
   필름에 ITO 패턴 → 터치 기능 통합
   · 버튼 없는 슬릭한 디자인
   · 에어컨 조작부, 오디오 패널

2. 조명 통합 IMD
   ═══════════════════════════════════
   필름에 광섬유/도광 → 은은한 조명
   · 엠비언트 라이팅
   · 로고 조명

3. 히팅 통합 IMD
   ═══════════════════════════════════
   필름에 발열체 → 열선 기능
   · 스티어링 휠 열선
   · 도어 핸들 열선

4. 안테나 통합 IMD
   ═══════════════════════════════════
   필름에 안테나 패턴 → 통신 기능
   · 스마트키 수신
   · NFC 결제
```

### 친환경 IMD

| 트렌드 | 내용 | 진행 상황 |
|--------|------|----------|
| **바이오 필름** | PLA 기반 필름 | 연구 단계 |
| **재활용 가능** | 올레핀 필름 | 적용 시작 |
| **수성 잉크** | VOC 제로 잉크 | 일부 적용 |
| **무용제 접착** | UV 경화 접착 | 개발 중 |
