# BNCAP 2.0 심층 노트

## 1. 5가지 평가 축 상세

### 1.1 충돌 보호 (Crash Protection) - 55점

**성인 탑승자 보호 (Adult Occupant Protection)**
| 시험 | 평가 항목 | 배점 |
|---|---|---|
| 정면 오프셋 64km/h | 머리, 목, 가슴, 무릎, 대퇴골, 하퇴골 | 16점 |
| 정면 풀폭 50km/h | 머리, 목, 가슴 상해치 | 8점 |
| 측면 배리어 50km/h | 머리, 가슴, 복부, 골반 | 8점 |
| 측면 폴 32km/h | 머리, 가슴 상해치 | 6점 |
| 후방 충돌 50km/h | 경추 상해 (Whiplash) | 2점 |

**아동 탑승자 보호 (Child Occupant Protection)**
- CRS(아동보호장치) 설치 용이성: 5점
- 정면/측면 충돌 시 아동 더미 상해치: 10점

### 1.2 VRU 보호 (Vulnerable Road User) - 20점

**보행자 충격 시험**
| 구역 | 시험 방법 | 배점 |
|---|---|---|
| 머리 충격 | 성인/아동 머리 모형 충격 | 8점 |
| 상체 충격 | 골반/대퇴부 충격 | 4점 |
| 하체 충격 | 다리 충격체 (Legform) | 4점 |

**자전거 탑승자 보호**
- AEB 자전거 탐지: 4점
- 측면/후방 자전거 회피 성능

### 1.3 안전 주행 (Safe Driving) - 10점

**기본 안전장치 장착 여부**
| 장치 | 배점 | 비고 |
|---|---|---|
| ESC | 2점 | 2027년 의무화 |
| 커튼 에어백 | 2점 | 전좌석 필수 |
| TPMS | 1점 | 직접 방식 권장 |
| 시트벨트 리마인더 | 1점 | 전좌석 |
| 사이드 에어백 | 1점 | 1열 기본 |
| ISOFIX | 1점 | 2열 2개소 |
| 센터 에어백 | 1점 | 선택 |
| 무릎 에어백 | 1점 | 운전석 |

### 1.4 사고 회피 (Accident Avoidance) - 10점

**ADAS 평가**
| 기능 | 배점 | 시험 조건 |
|---|---|---|
| AEB 차량 | 3점 | 정지/이동 차량 인식 |
| AEB 보행자 | 3점 | 주/야간 보행자 |
| AEB 자전거 | 2점 | 횡단 자전거 |
| LKA/LDW | 1점 | 차선 이탈 경고/보조 |
| SAS | 1점 | 속도 제한 보조 |

**AEB 시험 조건**
- 차량-차량: 10~80 km/h
- 차량-보행자: 20~60 km/h
- 야간 조건 포함 (40 lux 이하)

### 1.5 사고 후 안전 (Post-Crash Safety) - 5점

| 항목 | 배점 | 내용 |
|---|---|---|
| 자동 긴급 호출 (eCall) | 2점 | 충돌 감지 시 자동 신고 |
| 연료 차단 | 1점 | 충돌 후 연료 펌프 차단 |
| 도어 열림 | 1점 | 충돌 후 잠금 해제 |
| 경고등 자동 점멸 | 1점 | 2차 사고 방지 |

## 2. 충돌 시험 상세

### 2.1 정면 오프셋 충돌 (ODB)
```
속도: 64 km/h
오프셋: 40% (운전석측)
배리어: 변형 배리어 (Deformable Barrier)
더미: Hybrid III 50th (운전석), 5th (조수석)
측정: 머리/목/가슴/대퇴골/경골 상해치
```

**Euro NCAP 대비**: 동일 프로토콜

### 2.2 정면 풀폭 충돌 (FWRB)
```
속도: 50 km/h
오프셋: 100% (전폭 충돌)
배리어: 강체 배리어 (Rigid Wall)
더미: THOR 50th (운전석)
측정: 가슴 처짐량, 경추 모멘트
```

**목적**: 시트벨트/에어백 성능 평가

### 2.3 측면 배리어 충돌 (MDB)
```
속도: 50 km/h
방향: 운전석측 90° 충돌
배리어: 이동 변형 배리어 (1,500 kg)
더미: ES-2re (운전석), Q10 아동 (2열)
측정: 머리/가슴/복부/골반 상해치
```

### 2.4 측면 폴 충돌 (Oblique Pole)
```
속도: 32 km/h
각도: 75° 경사
폴: 직경 254mm 강체 폴
더미: WorldSID 50th
측정: 머리/가슴 상해치
```

**핵심**: 커튼 에어백 필수 (폴 충돌 대응)

### 2.5 후방 충돌 시험
```
속도: 50 km/h
방향: 후방에서 충돌
더미: BioRID II (경추 상해 평가)
평가: 연료 누출, 화재 위험, 도어 열림
```

## 5. 인증 프로세스

### 5.1 신청 절차
```
1. OEM → MoRTH 신청서 제출
2. 시험 차량 제출 (4대: 정면2, 측면2, 폴1)
3. 지정 시험소 배정 (ICAT, ARAI, CIRT 등)
4. 시험 실시 (약 4-6주)
5. 결과 분석 및 등급 산정
6. 인증서 발급
```
