## 🚙 Real-World Use Cases

### Use Case 1: OEM 포트폴리오 전략 수립

**Context**: 한국 OEM이 인도 시장에서 CAFE III 대응 전략을 수립해야 함

```
현재 상황 분석:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   현재 포트폴리오 (예시):                                    │
│   ─────────────────────────────                             │
│   모델 A (소형 세단): 95 g/km, 연 5만대                     │
│   모델 B (중형 SUV): 140 g/km, 연 8만대                     │
│   모델 C (대형 SUV): 165 g/km, 연 3만대                     │
│   모델 D (EV): 0 g/km, 연 1만대                             │
│                                                             │
│   현재 기업 평균 계산:                                       │
│   ─────────────────────────────                             │
│   (95×50,000 + 140×80,000 + 165×30,000 + 0×10,000×3)       │
│   ────────────────────────────────────────────────          │
│   (50,000 + 80,000 + 30,000 + 30,000)                       │
│                                                             │
│   = (4,750,000 + 11,200,000 + 4,950,000 + 0) / 190,000     │
│   = 20,900,000 / 190,000                                    │
│   = 110 g/km                                                │
│                                                             │
│   CAFE III 목표: 91.7 g/km                                  │
│   Gap: 18.3 g/km 초과 ❌                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**대응 전략:**

```
Strategy 1: EV 판매 확대 (Super Credit 활용)
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   현재: EV 1만대 (실제) → 3만대 (계산)                       │
│   목표: EV 3만대 (실제) → 9만대 (계산)                       │
│                                                             │
│   변경 후 계산:                                              │
│   (95×50K + 140×80K + 165×30K + 0×90K) / 250K              │
│   = 20,900,000 / 250,000                                    │
│   = 83.6 g/km ✓                                            │
│                                                             │
│   필요 조치:                                                 │
│   · EV 라인업 2개 → 4개 확대                               │
│   · 인도 내 EV 생산 설비 투자                               │
│   · 충전 인프라 파트너십                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘

Strategy 2: 차량 경량화
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   목표: 차종당 평균 10% 경량화                               │
│                                                             │
│   내장재 경량화 계획:                                        │
│   ────────────────────                                      │
│   · IP: PP-TD20 → PP-LGF30 (15% 경량화)                    │
│   · 도어 트림: ABS → PP+NFRC (12% 경량화)                   │
│   · 시트: 스틸 → Mg 프레임 (30% 경량화)                     │
│   · 카펫: 기존 PET → LWRT (20% 경량화)                      │
│                                                             │
│   예상 효과:                                                 │
│   · 차량당 30-50 kg 경량화                                  │
│   · CO2 3-5 g/km 감소                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘

Strategy 3: HEV 도입
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   모델 B SUV: ICE → HEV 전환                                │
│                                                             │
│   변경 전: 140 g/km, 8만대                                  │
│   변경 후: 85 g/km, 8만대 (2배 가중 = 16만대)               │
│                                                             │
│   효과:                                                     │
│   · CO2 기여: 11,200,000 → 6,800,000 (39% 감소)            │
│   · Super Credit 2배 효과                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 📝 Practice Problems

### Level 1: Basic

**Problem 1.1**: CAFE 기업 평균 계산

> A사가 다음과 같이 판매했습니다. 기업 평균 CO2를 계산하세요.
> - 소형차: 90 g/km, 10만대
> - SUV: 150 g/km, 5만대
> - EV: 0 g/km, 2만대 (Super Credit 3배)

<details>
<summary>💡 정답 보기</summary>

```
계산:
· 소형차: 90 × 100,000 = 9,000,000
· SUV: 150 × 50,000 = 7,500,000
· EV: 0 × 20,000 × 3 = 0 (60,000대 효과)

총 CO2: 16,500,000
총 판매 (계산용): 100,000 + 50,000 + 60,000 = 210,000대

기업 평균 = 16,500,000 / 210,000 = 78.6 g/km

CAFE III 기준 (91.7 g/km) 충족 ✓
```

</details>

### Level 2: Intermediate

**Problem 2.1**: 경량화 필요량 계산

> 현재 기업 평균이 100 g/km입니다. CAFE III (91.7 g/km) 충족을 위해 경량화만으로 대응하려면 평균 몇 kg 경량화가 필요할까요? (10kg 경량화 = 1 g/km CO2 감소 가정)

<details>
<summary>💡 정답 보기</summary>

```
필요 CO2 감소: 100 - 91.7 = 8.3 g/km

경량화 필요량: 8.3 × 10 = 83 kg/대

현실성 검토:
· 일반 승용차 중량: 1,200~1,500 kg
· 83 kg = 약 6% 경량화
· 내장재 중량: 약 180~250 kg (전체의 15-20%)
· 내장재에서만 83 kg 감량: 약 40% 감량 필요 → 어려움

결론: 경량화만으로는 부족, EV/HEV 확대 병행 필요
```

</details>

### Level 3: Advanced

**Problem 3.1**: 포트폴리오 최적화

> B사의 현재 포트폴리오입니다. CAFE III 충족을 위한 최적 전략을 수립하세요.

| 모델 | CO2 (g/km) | 판매량 | 마진 (₹) |
|------|-----------|--------|---------|
| 경차 | 85 | 50,000 | 50,000 |
| 세단 | 110 | 80,000 | 100,000 |
| SUV | 145 | 100,000 | 200,000 |
| EV | 0 | 10,000 | 80,000 |

<details>
<summary>💡 정답 보기</summary>

```
현재 상태 분석:
──────────────────────────────────────────
· 총 CO2: 85×50K + 110×80K + 145×100K + 0×30K
        = 4,250,000 + 8,800,000 + 14,500,000
        = 27,550,000
· 총 판매 (계산): 50K + 80K + 100K + 30K = 260,000
· 기업 평균: 27,550,000 / 260,000 = 106 g/km
· Gap: 14.3 g/km

전략 옵션 평가:
──────────────────────────────────────────
Option 1: SUV 판매 축소 (100K → 70K)
· 새 평균: (27,550,000 - 145×30K) / 230,000
         = 23,200,000 / 230,000 = 100.9 g/km
· 마진 손실: 30K × ₹200K = ₹600 crore
· 결과: 아직 미달 ❌

Option 2: EV 판매 확대 (10K → 30K)
· EV 추가 효과: 20K × 3 = 60K (계산용)
· 새 평균: 27,550,000 / 320,000 = 86.1 g/km
· 추가 투자: 생산설비 ₹300 crore
· 결과: 충족 ✓

Option 3: 세단 HEV 전환 (50% 물량)
· 세단 HEV: 70 g/km, 40K대 (2배 = 80K 계산)
· 세단 ICE: 110 g/km, 40K대
· 새 CO2: 4.25M + 2.8M + 2.2M + 14.5M + 0
        = 23.75M
· 새 판매: 50K + 40K + 80K + 100K + 30K = 300K
· 새 평균: 79.2 g/km ✓

최적 전략: Option 2 + Option 3 조합
──────────────────────────────────────────
· EV 20K 확대 + 세단 HEV 40K 전환
· 투자: ₹500 crore (EV + HEV)
· 마진 유지 + 장기 경쟁력 확보
· 기업 평균: 약 75 g/km (여유 확보)
```

</details>

## ✅ CAFE III 대응 체크리스트

### OEM용

```
□ 현재 포트폴리오 CO2 분석 완료
□ 2027년 목표 Gap 계산
□ EV/HEV 로드맵 수립
□ 경량화 목표 차종별 설정
□ 투자 vs 벌금 비교 분석
□ 공급업체 경량화 요구 전달
□ Pool 참여 여부 결정
□ WLTP 시험 체계 구축
```

### Tier 1 공급업체용

```
□ OEM별 경량화 요구사항 파악
□ 경량화 기술 로드맵 수립
□ 소재 변경 검토 (비용/물성)
□ 발포 사출 설비 투자 검토
□ 박육 설계 CAE 역량 확보
□ 시험 인증 대응 (AIS-035 등)
□ 원가 영향 분석 및 협상
```

## ⚠️ Common Mistakes

| 실수 | 왜 문제인가 | 올바른 접근 |
|------|------------|------------|
| EV만 믿고 대응 | 충전 인프라 부족 시 판매 한계 | EV + HEV + 경량화 종합 대응 |
| 벌금 감수 전략 | 브랜드 이미지 손상, 누적 비용 | 선제적 투자가 장기적 유리 |
| 단기 대응만 계획 | 2032년 이후 CAFE IV 더 엄격 | 10년 로드맵 필요 |
| 경량화만 의존 | 한계 있음 (안전, 비용) | 파워트레인 전환 병행 |
