# CORE CONCEPT: CAFE Phase 3 (인도)

> Corporate Average Fuel Efficiency Phase III - 인도의 2027년 연비/배출가스 규제

---

## 🎯 한 문장 정의

**CAFE Phase 3**는 2027년 4월부터 시행되는 인도의 기업 평균 연비 규제로, 자동차 제조사가 **1년간 판매한 모든 차량(ICE+HEV+BEV 전부 포함)**의 평균 CO2 배출량을 **91.7 g/km** 이하로 유지해야 하는 법규입니다.

## 🔢 Super Credit: "크레딧 3배"가 도대체 뭔 말인가?

### 핵심 개념

```
Super Credit = CO2 계산 시 "판매대수를 뻥튀기"해주는 제도

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   일반 차량: 1대 팔면 → 1대로 계산                          │
│   BEV 차량:  1대 팔면 → 3대로 계산 (Super Credit 3배)       │
│                                                             │
│   왜 이게 좋은가?                                            │
│   ─────────────────                                         │
│   BEV의 CO2 = 0 g/km (Tank-to-Wheel 기준)                   │
│   0 g/km 차량을 3대로 계산하면 → 평균 CO2가 확 낮아짐!      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 차 할인? 벌금 감면? 뭔가?

```
정답: 둘 다 아님! "CO2 계산 시 가중치"입니다!

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   Super Credit은:                                           │
│   ✗ 소비자에게 차 가격 할인해주는 것 아님                   │
│   ✗ 직접적으로 벌금 깎아주는 것 아님                        │
│   ✓ CO2 평균 계산할 때 유리하게 계산해주는 것!              │
│                                                             │
│   결과적으로:                                                │
│   → 평균 CO2가 낮아짐                                       │
│   → 기준 충족 가능성 높아짐                                  │
│   → 벌금 회피 가능 (간접 효과)                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 실제 계산 예시 (매우 중요!)

```
예시: A사가 1년간 총 100,000대 판매

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   판매 내역:                                                 │
│   ─────────────────────────────────────────────────         │
│   · 가솔린 SUV:    50,000대 × 150 g/km = 7,500,000 g        │
│   · 가솔린 세단:   30,000대 × 120 g/km = 3,600,000 g        │
│   · 하이브리드:    10,000대 × 70 g/km = 700,000 g           │
│   · BEV:          10,000대 × 0 g/km = 0 g                   │
│   ─────────────────────────────────────────────────         │
│   총 CO2:        11,800,000 g                               │
│                                                             │
│                                                             │
│   [Super Credit 없이 계산]                                   │
│   ─────────────────────────────────────────────────         │
│   총 대수: 100,000대                                        │
│   평균 CO2 = 11,800,000 / 100,000 = 118 g/km                │
│   → 기준 91.7 초과 → 불합격! 벌금!                          │
│                                                             │
│                                                             │
│   [Super Credit 적용 시]                                     │
│   ─────────────────────────────────────────────────         │
│   · 가솔린 SUV:    50,000대 (1배) = 50,000대                │
│   · 가솔린 세단:   30,000대 (1배) = 30,000대                │
│   · 하이브리드:    10,000대 (2배) = 20,000대 (가중)         │
│   · BEV:          10,000대 (3배) = 30,000대 (가중)          │
│   ─────────────────────────────────────────────────         │
│   가중 총 대수: 130,000대                                   │
│                                                             │
│   가중 평균 CO2 계산:                                        │
│   분자 = 7,500,000 + 3,600,000 + (700,000×2배 적용)         │
│          + (0×3배 적용)                                      │
│        = 7,500,000 + 3,600,000 + 1,400,000 + 0              │
│        = 12,500,000 g                                       │
│                                                             │
│   ※ 정확한 계산법: 각 차종의 CO2도 가중치 적용              │
│   평균 CO2 = 11,800,000 / 130,000 = 90.8 g/km               │
│   → 기준 91.7 이하 → 통과! 벌금 없음!                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘

핵심: BEV 1만대가 3만대로 계산되어 분모가 커짐
     → 평균 CO2가 118 → 90.8로 대폭 하락!
```

### HEV(하이브리드)의 기여도 이해

```
질문: 하이브리드가 70g/km인데, 2배로 계산하면 35g/km인가?

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   정답: 아니오! CO2 값 자체가 변하는 게 아닙니다!           │
│                                                             │
│   하이브리드 70g/km × 2배 크레딧 의미:                       │
│   ─────────────────────────────────────────────────         │
│   · 70g/km 차량이 35g/km로 되는 게 아님                     │
│   · 70g/km 차량 1대가 "2대분"으로 계산됨                    │
│                                                             │
│   효과:                                                     │
│   ─────────────────────────────────────────────────         │
│   · 평균 계산할 때 분모(총 대수)가 늘어남                   │
│   · 70g/km(낮은 값)가 평균에 2배 영향력                     │
│   · 결과적으로 전체 평균 CO2 낮아짐                          │
│                                                             │
│                                                             │
│   비유:                                                     │
│   ─────────────────────────────────────────────────         │
│   학급에 90점 학생이 있는데, 그 학생 점수를 2명분으로 계산   │
│   · 원래: (50+60+90)/3명 = 66.7점                          │
│   · 가중: (50+60+90+90)/4명분 = 72.5점                      │
│   → 90점 학생이 2배 기여해서 평균 상승!                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🔑 핵심 용어 상세

| 용어 | 영문 | 상세 설명 |
|------|------|----------|
| **CAFE** | Corporate Average Fuel Efficiency | 기업이 1년간 판매한 모든 차량의 평균 연비/CO2 |
| **BEE** | Bureau of Energy Efficiency | 인도 에너지부 산하 규제 기관, CAFE 주관 |
| **WLTP** | Worldwide Harmonised Light Vehicle Test Procedure | EU 기준 시험 방법, 실주행에 가까움 |
| **MIDC** | Modified Indian Driving Cycle | 인도 전용 시험 방법, 상대적 관대 |
| **Super Credit** | - | EV/HEV를 2~3배로 계산해주는 인센티브 |
| **CNF** | Carbon Neutrality Factor | 에탄올 등 바이오연료 CO2 감면 계수 |
| **공차중량** | Curb Weight | 연료만 가득 채우고 승객/짐 없는 차량 무게 |
| **Tank-to-Wheel** | - | 연료탱크→바퀴 구간 배출만 계산 (발전소 배출 미포함) |

## ⚠️ 909 kg 논쟁 (공차중량 특례)

### 공차중량(Curb Weight)이란?

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   공차중량 = 연료 가득 + 승객/짐 없음 상태의 차량 무게      │
│                                                             │
│   포함:                                                     │
│   ✓ 차체, 엔진, 변속기 등 모든 부품                        │
│   ✓ 윤활유, 냉각수 등 작동 유체                             │
│   ✓ 연료 (만땅 상태)                                       │
│   ✓ 공구, 스페어타이어 (옵션에 따라)                        │
│                                                             │
│   미포함:                                                   │
│   ✗ 운전자/승객                                             │
│   ✗ 화물/짐                                                 │
│                                                             │
│   비교:                                                     │
│   · 공차중량: 차량 자체 무게                               │
│   · 적재중량: 공차중량 + 승객 + 짐 (실제 주행 상태)        │
│   · 총중량: 최대 적재 상태 무게                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 💰 벌금 구조에서 "0.2L"이 뭔 말인가?

```
0.2L = 0.2 L/100km (연료소비량 기준)

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   CAFE 기준은 2가지로 표현 가능:                            │
│   ─────────────────────────────────────────────────         │
│   · CO2 배출량: 91.7 g/km                                   │
│   · 연료소비량: 3.73 L/100km (환산)                         │
│                                                             │
│   환산 공식 (가솔린 기준):                                   │
│   CO2 (g/km) ≈ 연료소비량 (L/100km) × 23.2                 │
│   91.7 g/km ≈ 3.95 L/100km                                  │
│                                                             │
│                                                             │
│   벌금 기준:                                                │
│   ─────────────────────────────────────────────────         │
│   │                                                         │
│   │  연료소비량 미달분이 기준                               │
│   │                                                         │
│   │  Case 1: 미달 < 0.2 L/100km                            │
│   │  ──────────────────────────                             │
│   │  목표: 3.73 L/100km                                    │
│   │  실적: 3.85 L/100km (0.12L 초과)                       │
│   │  → 0.2 미만이므로 "경미한 위반"                        │
│   │  → ₹25,000/대 (약 37,000원)                            │
│   │                                                         │
│   │  Case 2: 미달 ≥ 0.2 L/100km                            │
│   │  ──────────────────────────                             │
│   │  목표: 3.73 L/100km                                    │
│   │  실적: 4.10 L/100km (0.37L 초과)                       │
│   │  → 0.2 이상이므로 "중대한 위반"                        │
│   │  → ₹50,000/대 (약 75,000원)                            │
│   │                                                         │
└─────────────────────────────────────────────────────────────┘

왜 "L/100km"인가?
─────────────────────────────────────────────────
· CO2 (g/km)보다 직관적 (연료비 계산 쉬움)
· 글로벌 CAFE 규제 대부분 연료소비량 기준 병기
· 0.2L = 100km당 200ml 연료 = 연간 주행 시 상당량
```

## 🏭 왜 CAFE III가 중요한가?

### 자동차 업계 영향

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   CAFE III 미준수 시 벌금 예상:                             │
│                                                             │
│   · 미달 < 0.2 L/100km → ₹25,000/대 (약 37,000원)          │
│   · 미달 ≥ 0.2 L/100km → ₹50,000/대 (약 75,000원)          │

│   · 8개 OEM 합계: ₹7,300 crore (약 1.2조원)                │
│   · 현대 단독: ₹2,800 crore (약 4,500억원)                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
