# CORE CONCEPT: CAFE Phase 3 (인도)

> Corporate Average Fuel Efficiency Phase III - 인도의 2027년 연비/배출가스 규제

---

## 🎯 한 문장 정의

**CAFE Phase 3**는 2027년 4월부터 시행되는 인도의 기업 평균 연비 규제로, 자동차 제조사가 판매 차량의 평균 CO2 배출량을 **91.7 g/km** 이하로 유지해야 하는 법규입니다.

## ⚡ CAFE III의 3가지 핵심 변화

### 1. CO2 목표 강화
```
Phase II: 113 g/km → Phase III: 91.7 g/km (19% 감축)

연료소비량 환산:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   2027년: 3.73 L/100km                                     │
│         ↓                                                   │
│   2032년: 3.01 L/100km (19% 감소)                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. 시험 방법 변경
```
MIDC (Modified Indian Driving Cycle)
    ↓
WLTP (Worldwide Harmonised Light Vehicle Test Procedure)

차이점:
· MIDC: 인도 주행 패턴 기반, 상대적으로 관대
· WLTP: EU 표준, 더 실제 주행에 가까움
· 2026년 4월부터: 양쪽 모두 보고 의무
· 2027년 4월부터: WLTP 기준 적용
```

### 3. Super Credit 시스템
```
전기차/하이브리드 판매 시 가중치:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   BEV (배터리 전기차)     → 3배 가중치                      │
│   PHEV (플러그인 하이브리드) → 2.5배 가중치                  │
│   HEV (하이브리드)        → 2배 가중치                       │
│                                                             │
│   예시: BEV 1대 판매 = ICE 3대 효과 (CO2 계산 시)           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🏭 왜 CAFE III가 중요한가?

### 자동차 업계 영향

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   CAFE III 미준수 시:                                       │
│                                                             │
│   · 미달 < 0.2 L/100km → ₹25,000/대 벌금                   │
│   · 미달 ≥ 0.2 L/100km → ₹50,000/대 벌금                   │
│   · 기본 벌금: ₹10 lakh (약 1,500만원)                     │
│                                                             │
│   예상 벌금 규모:                                            │
│   · 8개 OEM 합계: ₹7,300 crore (약 1.2조원)                │
│   · 현대 단독: ₹2,800 crore (약 4,500억원)                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 내장재 설계 영향

```
경량화 압박:
· 10kg 경량화 → 약 1 g/km CO2 감소
· 내장재 = 차량 중량의 15-20%
· 경량화 목표: 부품당 5-15% 중량 감소

내장재 설계 변화:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   1. 소재 변경                                              │
│      · PP → 경량 PP 또는 천연섬유 복합재                    │
│      · 스틸 브라켓 → 플라스틱/알루미늄                      │
│                                                             │
│   2. 구조 최적화                                            │
│      · 리브 두께 감소                                       │
│      · 박육 설계 적용                                       │
│      · 일체화 설계 (부품 통합)                              │
│                                                             │
│   3. 신기술 적용                                            │
│      · 발포 사출 (MuCell)                                   │
│      · LWRT (경량 강화 열가소성)                            │
│      · 박막 필름 적용                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## ⚠️ 909 kg 논쟁 (Small Car Exemption)

```
논쟁의 핵심:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   BEE 초안:                                                 │
│   · 차량 중량 ≤ 909 kg                                     │
│   · 배기량 ≤ 1,200 cc                                      │
│   · 전장 ≤ 4,000 mm                                        │
│   → 위 조건 충족 시 3 g/km 추가 감면                        │
│                                                             │
│   찬성: Maruti Suzuki (95% 시장점유율 보유)                 │
│   반대: Tata, Hyundai, Mahindra, Kia, MG                   │
│         (19개 OEM 중 15개사 반대)                           │
│                                                             │
│   반대 이유:                                                 │
│   · "909 kg 이하 차량 중 B-NCAP 충족 차량 없음"            │
│   · "경량화 = 안전 저하" 우려                              │
│   · "특정 1개사만 혜택" (불공정)                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
