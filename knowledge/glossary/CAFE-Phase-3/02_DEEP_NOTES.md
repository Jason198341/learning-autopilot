## 🔬 WLTP 시험 방법 완전 해설

### WLTP란?

```
WLTP = Worldwide Harmonised Light Vehicle Test Procedure
       (세계 통합 경량차 시험 절차)

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   목적: 전 세계 공통 기준으로 연비/배출가스 측정            │
│   개발: UN ECE (유엔 유럽경제위원회)                        │
│   EU 도입: 2018년 9월 (NEDC 대체)                          │
│   인도 도입: 2027년 4월 (MIDC 대체 예정)                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### MIDC vs WLTP 상세 비교

```
Modified Indian Driving Cycle (MIDC)
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   특징:                                                     │
│   · 인도 주행 패턴 기반 (저속, 정체 많음)                  │
│   · 저속 주행 비중 높음                                     │
│   · 냉간 시동 미포함 (엔진 예열 후 측정)                   │
│   · 상대적으로 관대한 결과                                  │
│                                                             │
│   시험 조건:                                                │
│   · 최고속도: 90 km/h                                       │
│   · 평균속도: 약 32 km/h                                    │
│   · 총 거리: 약 8 km                                        │
│   · 소요시간: 약 15분                                       │
│   · 정지 비율: 25% 이상                                     │
│                                                             │
│   구성:                                                     │
│   · Part 1: 도심 주행 (4회 반복)                           │
│   · Part 2: 시외 주행 (1회)                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘

Worldwide Harmonised Light Vehicle Test Procedure (WLTP)
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   특징:                                                     │
│   · 글로벌 표준 (EU 2018년 도입)                          │
│   · 실제 주행에 더 가까움                                   │
│   · 냉간 시동 포함 (차가운 엔진 시작)                      │
│   · 더 엄격한 결과                                          │
│                                                             │
│   시험 조건:                                                │
│   · 최고속도: 131 km/h                                      │
│   · 평균속도: 약 47 km/h                                    │
│   · 총 거리: 약 23 km                                       │
│   · 소요시간: 약 30분                                       │
│   · 정지 비율: 12.6%                                       │
│                                                             │
│   구성:                                                     │
│   · Low: 저속 도심 (9.8분)                                 │
│   · Medium: 중속 시외 (7.2분)                              │
│   · High: 고속 간선 (7.6분)                                │
│   · Extra High: 고속도로 (5.4분)                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘

비교 결과:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   동일 차량 시험 시:                                        │
│                                                             │
│   MIDC 결과: 100 g/km                                       │
│   WLTP 결과: 115-120 g/km (+15-20%)                        │
│                                                             │
│   왜 WLTP가 더 높게 나오나?                                 │
│   ────────────────────────                                  │
│   1. 냉간 시동: 엔진 차가울 때 연료 더 많이 소비           │
│   2. 고속 주행: 131 km/h까지 측정 (공기저항↑)             │
│   3. 긴 시험: 23km vs 8km (지속적 부하)                   │
│   4. 가속 패턴: 급가속 구간 포함                           │
│                                                             │
│   → WLTP가 실제 주행에 더 가까움!                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🚗 Super Credit 상세

### 크레딧 배수 완전 해설

```
Super Credit = 평균 CO2 계산 시 "판매대수 뻥튀기"

┌──────────┬───────────┬─────────────┬────────────────────────┐
│ 차종     │ 크레딧    │ CO2        │ 실제 의미              │
│          │ 배수      │ (g/km)     │                        │
├──────────┼───────────┼─────────────┼────────────────────────┤
│ BEV      │ 3.0x      │ 0          │ 1대 팔면 3대로 계산    │
│ (배터리) │           │            │ 0g/km이 3대분 기여     │
├──────────┼───────────┼─────────────┼────────────────────────┤
│ FCEV     │ 3.0x      │ 0          │ 수소차도 BEV와 동일    │
│ (수소)   │           │            │                        │
├──────────┼───────────┼─────────────┼────────────────────────┤
│ PHEV     │ 2.5x      │ 30-50      │ 1대를 2.5대로 계산     │
│ (플러그인)│          │            │ 낮은 CO2가 2.5배 기여  │
├──────────┼───────────┼─────────────┼────────────────────────┤
│ SHEV     │ 2.0x      │ 50-80      │ 1대를 2대로 계산       │
│ (풀HEV)  │           │            │ 낮은 CO2가 2배 기여    │
├──────────┼───────────┼─────────────┼────────────────────────┤
│ MHEV     │ 1.0x      │ 90-110     │ 일반 계산 (별도 혜택)  │
│ (마일드) │           │            │                        │
├──────────┼───────────┼─────────────┼────────────────────────┤
│ ICE      │ 1.0x      │ 100-160+   │ 일반 계산              │
│ (내연기관)│          │            │                        │
└──────────┴───────────┴─────────────┴────────────────────────┘
```

### Super Credit 실전 계산

```
예시: B사 연간 판매 구성

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   실제 판매:                                                 │
│   ─────────────────────────────────────────────────         │
│   · 가솔린 SUV:   40,000대 × 145 g/km                      │
│   · 디젤 세단:    20,000대 × 125 g/km                      │
│   · SHEV (풀HEV): 15,000대 × 65 g/km                       │
│   · BEV:          5,000대 × 0 g/km                         │
│   ─────────────────────────────────────────────────         │
│   실제 총: 80,000대                                         │
│                                                             │
│                                                             │
│   [Super Credit 적용 계산]                                   │
│   ─────────────────────────────────────────────────         │
│                                                             │
│   가중 대수:                                                 │
│   · 가솔린 SUV:   40,000 × 1.0 = 40,000대                  │
│   · 디젤 세단:    20,000 × 1.0 = 20,000대                  │
│   · SHEV:        15,000 × 2.0 = 30,000대 (가중)            │
│   · BEV:          5,000 × 3.0 = 15,000대 (가중)            │
│   ─────────────────────────────────────────────────         │
│   가중 총: 105,000대                                        │
│                                                             │
│   CO2 총합:                                                 │
│   · 가솔린: 40,000 × 145 = 5,800,000 g                     │
│   · 디젤:  20,000 × 125 = 2,500,000 g                      │
│   · SHEV:  30,000 × 65 = 1,950,000 g (가중 대수 적용)      │
│   · BEV:   15,000 × 0 = 0 g                                │
│   ─────────────────────────────────────────────────         │
│   총: 10,250,000 g                                         │
│                                                             │
│   기업 평균 CO2 = 10,250,000 / 105,000 = 97.6 g/km         │
│                                                             │
│   ※ Super Credit 없었다면:                                  │
│      (5,800,000+2,500,000+975,000+0) / 80,000 = 116 g/km   │
│                                                             │
│   결론: 116 → 97.6 g/km로 개선! (아직 91.7 미달이지만)     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🔧 경량화 소재 완전 해설

### 1. PP-LGF30 (Long Glass Fiber Polypropylene 30%)

```
PP-LGF30 = 폴리프로필렌 + 장섬유 유리섬유 30%

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   정의:                                                     │
│   ─────────────────────────────────────────────────         │
│   · PP (폴리프로필렌) 기재에                                │
│   · LGF (Long Glass Fiber, 장섬유 유리섬유)를               │
│   · 30% 무게비로 보강한 복합소재                            │
│                                                             │
│   "장섬유"란?                                                │
│   ─────────────────────────────────────────────────         │
│   · 길이 10mm 이상 유리섬유 (보통 12-25mm)                  │
│   · vs 단섬유(SGF): 0.3-0.5mm                              │
│   · 장섬유가 더 우수한 물성 (강도, 내충격)                  │
│                                                             │
│                                                             │
│   물성 비교:                                                │
│   ─────────────────────────────────────────────────         │
│                                                             │
│   │ 항목         │ PP-TD20     │ PP-LGF30   │ 개선율     │
│   ├──────────────┼─────────────┼────────────┼────────────┤
│   │ 밀도 (g/cm³) │ 1.04        │ 1.14       │ +10%       │
│   │ 인장강도     │ 35 MPa      │ 80-100 MPa │ +130-185%  │
│   │ 굴곡탄성률   │ 2,500 MPa   │ 6,000 MPa  │ +140%      │
│   │ 충격강도     │ 8 kJ/m²     │ 25 kJ/m²   │ +212%      │
│   │ 열변형온도   │ 95°C        │ 150°C      │ +58%       │
│   └──────────────┴─────────────┴────────────┴────────────┘
│                                                             │
│                                                             │
│   원가/중량 분석:                                            │
│   ─────────────────────────────────────────────────         │
│   · 원료가: PP-TD20 대비 약 30-40% 높음                    │
│     - PP-TD20: ~$1.8/kg                                    │
│     - PP-LGF30: ~$2.5/kg                                   │
│                                                             │
│   · 중량 절감 효과:                                         │
│     - 동일 강도 기준 15-25% 경량화 가능                    │
│     - 벽두께 감소 가능 (3.0mm → 2.2mm)                     │
│     - 리브 간소화 가능                                      │
│                                                             │
│   · 실제 부품 원가:                                         │
│     - 소재비 증가: +20-30%                                 │
│     - 중량 감소: -15-20%                                   │
│     - 순 원가: ±0 ~ +10% (부품마다 다름)                   │
│                                                             │
│                                                             │
│   적용 부품:                                                │
│   ─────────────────────────────────────────────────         │
│   · IP (Instrument Panel) 캐리어                           │
│   · 도어 모듈 캐리어                                        │
│   · 시트 백보드                                             │
│   · 프론트엔드 모듈                                         │
│   · 범퍼 빔                                                 │
│                                                             │
│                                                             │
│   장점:                                                     │
```
