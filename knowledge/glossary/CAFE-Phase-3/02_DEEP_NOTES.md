## 🔬 시험 방법 상세

### MIDC vs WLTP 비교

```
Modified Indian Driving Cycle (MIDC)
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   특징:                                                     │
│   · 인도 주행 패턴 기반                                     │
│   · 저속 주행 비중 높음                                     │
│   · 냉간 시동 미포함                                        │
│   · 상대적으로 관대한 결과                                  │
│                                                             │
│   시험 조건:                                                │
│   · 최고속도: 90 km/h                                       │
│   · 평균속도: 약 32 km/h                                    │
│   · 총 거리: 약 8 km                                        │
│   · 소요시간: 약 15분                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘

Worldwide Harmonised Light Vehicle Test Procedure (WLTP)
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   특징:                                                     │
│   · 글로벌 표준 (EU, 2018년 도입)                          │
│   · 실제 주행에 더 가까움                                   │
│   · 냉간 시동 포함                                          │
│   · 더 엄격한 결과                                          │
│                                                             │
│   시험 조건:                                                │
│   · 최고속도: 131 km/h                                      │
│   · 평균속도: 약 47 km/h                                    │
│   · 총 거리: 약 23 km                                       │
│   · 소요시간: 약 30분                                       │
│                                                             │
│   4단계 구성:                                               │
│   · Low: 저속 도심 주행                                     │
│   · Medium: 중속 주행                                       │
│   · High: 고속 주행                                         │
│   · Extra High: 고속도로 주행                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘

비교 결과:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   동일 차량 시험 시:                                        │
│                                                             │
│   MIDC 결과: 100 g/km                                       │
│   WLTP 결과: 115-120 g/km (+15-20%)                        │
│                                                             │
│   → WLTP가 더 엄격 (실제 주행에 가까움)                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 💰 벌금 체계 상세

### 벌금 구조

```
Energy Conservation Act 2022 개정 벌금 규정:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   Case 1: 연료소비 미달 < 0.2 L/100km                       │
│   ─────────────────────────────────────                     │
│   · ₹25,000/대 (약 37,000원)                               │
│   · 예: 목표 3.73, 실적 3.85 → 차이 0.12                   │
│                                                             │
│   Case 2: 연료소비 미달 ≥ 0.2 L/100km                       │
│   ─────────────────────────────────────                     │
│   · ₹50,000/대 (약 75,000원)                               │
│   · 예: 목표 3.73, 실적 4.10 → 차이 0.37                   │
│                                                             │
│   기본 벌금: ₹10 lakh (약 1,500만원)                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘

벌금 계산 예시:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   A사: 연간 50만대 판매, 0.3 L/100km 초과                   │
│                                                             │
│   벌금 = 500,000대 × ₹50,000 + ₹10 lakh                    │
│        = ₹25,000,000,000 + ₹1,000,000                      │
│        = ₹2,500 crore (약 4,000억원)                       │
│                                                             │
│   실제 예상 (2025년 추정):                                   │
│   · Hyundai: ₹2,800 crore                                  │
│   · 8개 OEM 합계: ₹7,300 crore                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🔧 내장재 경량화 기술

### 소재별 경량화 잠재력

| 부품 | 기존 소재 | 경량화 소재 | 중량 감소 |
|------|----------|------------|----------|
| IP | PP-TD20 | PP-LGF30 | 15-20% |
| 도어 트림 | ABS | PP+NFRC | 10-15% |
| 헤드라이닝 | PU 폼 | 경량 PET | 20-25% |
| 시트 프레임 | 스틸 | CFRP/Mg | 30-40% |
| 카펫 | PET | 재생 LWRT | 15-20% |

### 설계 최적화 기법

```
1. 위상 최적화 (Topology Optimization)
   · 불필요한 부분 제거
   · 하중 경로 기반 설계
   · 10-30% 경량화 가능

2. 박육 설계
   · 벽두께 감소 (3.0 → 2.2mm)
   · 리브 최적화
   · 15-25% 경량화

3. 일체화 설계
   · 부품 통합 (브라켓 일체화)
   · 체결부 감소
   · 조립 비용 절감

4. 발포 사출 (MuCell)
   · 미세 기공 형성
   · 밀도 10-15% 감소
   · 싱크마크 개선
```
