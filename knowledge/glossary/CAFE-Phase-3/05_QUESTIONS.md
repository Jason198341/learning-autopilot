## 🔓 Open Questions (아직 답을 찾는 중)

### Question 1: 909 kg 기준의 최종 결정

**질문**: 909 kg 경차 특례는 최종 법규에 포함될 것인가?

```
현재 상황:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   BEE 초안 (2025.09):                                       │
│   · 909 kg 이하 차량에 3 g/km 공제                          │
│                                                             │
│   업계 반응:                                                 │
│   · 찬성: Maruti Suzuki, Renault (2개사)                   │
│   · 반대: Tata, Hyundai, Kia, Mahindra 등 (15개사)         │
│                                                             │
│   최신 동향 (2026.01):                                      │
│   · BEE: 공제폭 4.51~5.31 g/km로 확대 검토                 │
│   · 또는 기준 완전 삭제 검토 중                              │
│                                                             │
│   미해결 이슈:                                               │
│   · 안전 vs 환경 트레이드오프                               │
│   · 공정 경쟁 논란                                          │
│   · 정치적 요소 (선거, 로비)                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**왜 중요한가?**
- Maruti Suzuki (시장 1위)의 경쟁력에 직접 영향
- 경차 시장 존폐 결정
- 한국 OEM 전략에도 영향

**탐구 방향**:
- BEE 최종 고시 모니터링
- SIAM 공식 입장 추적
- 글로벌 선례 분석 (EU, 중국)

### Resolved 3: 벌금 체계

**질문**: CAFE 미준수 시 벌금은 얼마인가?

**답변**:
- 미달 < 0.2 L/100km: ₹25,000/대
- 미달 ≥ 0.2 L/100km: ₹50,000/대
- 기본 벌금: ₹10 lakh
