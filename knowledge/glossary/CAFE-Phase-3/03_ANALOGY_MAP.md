## 🎯 Primary Analogy: 학급 평균 성적 관리

```
[CAFE 규제] ≈ [학교 학급 평균 성적 관리]

학교 시스템                           CAFE 시스템
═══════════                          ═══════════

교육부                                BEE (에너지효율국)
"전국 학력 수준 관리"                  "전국 연비 수준 관리"
    │                                     │
    ▼                                     ▼
학교별 평균 성적 기준                   OEM별 평균 CO2 기준
"학급 평균 80점 이상"                   "기업 평균 91.7 g/km 이하"
    │                                     │
    ▼                                     ▼
학생별 성적 다양                        차종별 CO2 다양
(수학 50점, 영어 100점...)             (SUV 150g, 경차 70g...)
    │                                     │
    ▼                                     ▼
평균 미달 시 불이익                     평균 초과 시 벌금
(학급 페널티)                          (대당 ₹25,000~50,000)
```

### 비유 확장: 우등생과 열등생

```
학급에서의 전략:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   문제: "학급 평균 80점을 맞춰야 해요"                       │
│                                                             │
│   전략 1: 열등생 퇴출                                        │
│   → CAFE: 고배출 차종 단종                                  │
│                                                             │
│   전략 2: 우등생 영입                                        │
│   → CAFE: EV/HEV 판매 확대                                  │
│                                                             │
│   전략 3: 전체 수준 향상                                     │
│   → CAFE: 전 차종 연비 개선                                 │
│                                                             │
│   전략 4: 우등생 가산점                                      │
│   → CAFE: Super Credit (EV 3배 가중치)                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 💡 Metaphors

### Technical Metaphor

> "CAFE 규제는 **탄소 예산**과 같다. 각 OEM은 매년 정해진 탄소 예산 내에서 차를 팔아야 한다. EV를 팔면 예산이 남고, SUV를 팔면 예산이 빨리 소진된다. 예산을 초과하면 벌금을 내야 한다."

### Everyday Metaphor

> "CAFE는 **뷔페 칼로리 제한**과 같아요. 식당에서 '손님당 평균 2,000kcal 이하만 드세요'라고 하면, 많이 먹는 손님 때문에 적게 먹는 손님이 필요해요. SUV가 많이 먹는 손님이라면, EV는 샐러드만 먹는 손님이에요."

### Teaching Metaphor

> "909 kg 논쟁은 **시험 채점 특례**와 같아요. '작은 학생은 체력 시험에서 3점 가산점을 줄게요'라고 하면, 작은 학생이 많은 학교만 유리해지죠. Maruti Suzuki가 그 학교예요."

## 🧩 Where Analogies Break

### 비유의 한계

| 비유 | 한계점 |
|------|--------|
| **학급 평균** | CAFE는 판매대수 가중 평균 (단순 산술 평균 아님) |
| **탄소배출권** | CAFE 크레딧은 외부 거래 불가 (Pool 내부만) |
| **보험 등급** | CAFE는 사후 결과, 보험은 사전 예측 |
| **다이어트** | 차량 CO2는 쉽게 바뀌지 않음 (신차 개발 3-4년) |

### CAFE만의 특수성

```
CAFE 규제의 독특한 점:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   1. 판매 믹스 의존                                          │
│      · 같은 차종이라도 트림별 CO2 상이                       │
│      · 옵션(선루프 등)도 중량/CO2에 영향                    │
│                                                             │
│   2. 시장 예측 어려움                                        │
│      · 고객이 어떤 차를 살지 예측 불가                       │
│      · SUV 대세 → 갑자기 규제 미달                          │
│                                                             │
│   3. 신차 개발 리드타임                                      │
│      · 규제 대응에 3-4년 소요                               │
│      · 급격한 기준 변화 대응 어려움                          │
│                                                             │
│   4. Super Credit 효과                                       │
│      · EV 1대로 ICE 다수 상쇄 가능                          │
│      · 전략적 EV 출시 중요                                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🧠 Transfer Learning

### CAFE에서 배우는 범용 원리

```
1. 집단 평균 규제의 효과
   ─────────────────────
   · 개별 규제보다 유연성 제공
   · 기업 자율성 보장
   · 혁신 기술에 인센티브

2. 인센티브 설계의 중요성
   ─────────────────────
   · Super Credit = 목표 행동 유도
   · 벌금 = 비목표 행동 억제
   · 균형 잡힌 설계 필요

3. 이해관계자 갈등 관리
   ─────────────────────
   · 909 kg 논쟁 = 규제의 공정성 문제
   · 다수의 동의 필요
   · 투명한 기준 설정 중요
```
