## 🧠 Central Mental Model: 학급 평균 성적

```

     = 학교 학급의 "평균 성적"과 같은 개념

┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   [학급 평균 성적]              [기업 평균 연비]             │
│                                                             │
│   담임 선생님                   BEE (에너지효율국)           │
│        ↓                            ↓                       │
│   "평균 80점 이상               "평균 91.7 g/km 이하        │

│        ↓                            ↓                       │
│   학생별 성적 다양              차종별 CO2 다양              │
│   (50점~100점)                  (60~150 g/km)              │
│        ↓                            ↓                       │
│   평균 = 전체 합계 / 학생 수    평균 = 전체 CO2 / 판매대수   │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 📐 CAFE 계산 공식

### 기본 공식

```
                  Σ (각 차종 CO2 × 판매대수)
기업 평균 CO2 = ────────────────────────────
                      총 판매대수

예시: A사의 연간 판매
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   모델 A: 80 g/km × 10,000대 =   800,000                   │
│   모델 B: 100 g/km × 20,000대 = 2,000,000                   │
│   모델 C: 130 g/km × 5,000대 =   650,000                   │
│   ─────────────────────────────────────────                 │
│   합계:              35,000대   3,450,000                   │
│                                                             │
│   기업 평균 = 3,450,000 / 35,000 = 98.6 g/km               │
│                                                             │
│   CAFE III 기준: 91.7 g/km                                  │
│   결과: 6.9 g/km 초과 → 벌금 대상 ❌                        │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### Super Credit 적용 공식

```
전기차/하이브리드 가중치 적용:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   BEV 판매: 1,000대 × 3배 = 3,000대 효과 (CO2 = 0)         │
│   PHEV 판매: 500대 × 2.5배 = 1,250대 효과 (저 CO2)         │
│   HEV 판매: 2,000대 × 2배 = 4,000대 효과 (저 CO2)          │
│                                                             │
│   실제 판매: 38,500대                                       │
│   계산용 판매: 35,000 + 3,000 + 1,250 + 4,000 = 43,250대    │
│                                                             │
│   → 기업 평균 CO2 대폭 감소 효과                            │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🔄 CAFE 준수 전략 5가지

```
Strategy 1: 차량 경량화
─────────────────────────────────────────
· 10kg 감소 → 약 1 g/km CO2 감소
· 내장재, 차체, 파워트레인 경량화
· 투자비 대비 효과 검토 필요

Strategy 2: 파워트레인 효율화
─────────────────────────────────────────
· 터보 다운사이징
· 8단 이상 변속기
· 48V 마일드 하이브리드
· Start-Stop 시스템

Strategy 3: 전동화 확대
─────────────────────────────────────────
· BEV/PHEV/HEV 비중 확대
· Super Credit 활용
· 인도: EV 판매 8~10% 목표

Strategy 4: 포트폴리오 조정
─────────────────────────────────────────
· 고배출 차종 단종/축소
· 저배출 차종 확대
· CNG/LPG 차량 추가

Strategy 5: 크레딧 거래 (Pool)
─────────────────────────────────────────
· 제조사 간 크레딧 공유
· 잉여 크레딧 판매
· Pool Manager 지정
```
