# 옆산 4: 전장 통합의 소재 — 플라스틱이 전자 부품이 되는 시대

## 내장재가 바뀌고 있다

예전: PP 덩어리에 버튼을 꽂은 것
지금: 대시보드 전체가 디스플레이, 표면을 터치하면 반응

```
예전 질문: 이 플라스틱이 충분히 강한가? 안 녹는가?

새로운 질문: 빛을 통과시킬 수 있는가?
             전기가 흐를 수 있는가?
             터치를 감지할 수 있는가?
             이 모든 걸 하면서도 강하고 안 녹는가?
```

Day 1~10의 기본 요구 + 전기적/광학적 요구가 추가된 것.

## 1. 빛을 다루는 소재 — 숨은 조명(Hidden Lighting)

### 분자 수준에서 빛의 투과

빛이 고분자를 만나면 세 가지 중 하나:
- **투과(transmission)**: 빛이 통과 → 투명/반투명
- **흡수(absorption)**: 전자에 흡수 → 열로 변환
- **산란(scattering)**: 방향을 바꿈 → 불투명하게 보임

**PP가 불투명한 이유**: 결정 영역과 비정질 영역의 굴절률 차이 → 경계마다 산란

### 숨은 조명(Hidden-til-lit) 구현 방법

| 방법 | 원리 | 적용 |
|------|------|------|
| 얇은 벽 | 0.5~1.0mm 부분만 빛 투과 | 패턴/로고 |
| 2-shot 사출 | 불투명 PP + 투명 PC 일체화 | 정밀 디자인 |
| 레이저 에칭 | 표면/내부 부분 제거 | 고해상도 패턴 |
| 광도파 | 전반사로 빛 진행 → 노치에서 출광 | 라인 조명 |

### 광도파의 원리

```
투명 PC/PMMA 막대에 LED 빛을 넣으면
→ 전반사(total internal reflection)로 빛이 내부를 진행
→ 표면에 미세 패턴을 넣으면 그 지점에서 빛이 빠져나옴

PC 굴절률(~1.59) > 공기(1.0)
→ 임계각 이상으로 입사하면 빛이 빠져나가지 못함
→ Day 2의 빛-물질 상호작용
```

### 광학 소재 비교

| 소재 | 투과율 | HDT | 용도 |
|------|--------|-----|------|
| PMMA | 92% | 95°C | 렌즈, 도광판 |
| PC | 88% | 130°C | 구조+투명, 커버 |
| 투명 PP+핵제 | ~70% | ~100°C | 저비용 반투과 |

## 2. 전기를 다루는 소재 — 터치와 센서

### 정전용량 터치 원리

```
표면 아래 전극 패턴이 있음
→ 전극이 전기장을 형성
→ 손가락(전도체)이 접근하면 전기장이 변화
→ 위치를 감지
```

### 전도성 소재 — 플라스틱에 전기를 통하게

| 방법 | 면저항 | 투명성 | 특징 |
|------|--------|--------|------|
| 은 잉크 | ~0.1 Ω/sq | 불투명 | 저비용, 패턴 자유 |
| ITO | ~50~300 Ω/sq | 투명 | 스마트폰 표준, 깨지기 쉬움 |
| 전도성 고분자 (PEDOT:PSS) | ~100~1000 Ω/sq | 투명 가능 | 유연, 인쇄 가능 |
| 메탈 메쉬 | ~1~10 Ω/sq | 투명 | 대면적, 3D 성형 가능 |
| CNT/그래핀 | 다양 | 투명 가능 | 연구 단계 |

**전도성 고분자가 전기를 통하는 이유:**
- 사슬에 공액 이중결합(conjugated double bond)이 있음
- 전자가 이중결합을 따라 비편재화(delocalization)
- Day 2: 전자가 특정 원자에 묶이지 않고 사슬을 따라 이동

### 유전체(Dielectric) — 터치 감도 결정

```
정전용량 = ε × A / d
  ε: 유전율 (소재 고유값)
  A: 전극 면적
  d: 전극-손가락 사이 거리 (= 표피 두께)

소재별 유전율:
  PP:    2.2~2.6  ← 비극성, 터치 감도 낮음
  PC:    2.9~3.0  ← 적당
  PA:    3.4~4.5  ← 극성 높음
  PU:    6~8      ← 높음, 소프트 터치에 유리

PP 위에 터치를 구현하기 어려운 이유:
유전율이 낮아서 표피가 얇아야 하는데, 얇으면 강성/내구성 부족
→ PC나 PU 표피 사용이 유리
```

## 3. IME (In-Mold Electronics)

### 공정

```
① 평평한 PC/PET 필름에 전도성 잉크로 회로 인쇄
② 같은 필름에 LED, 저항 같은 부품을 실장(SMD)
③ 필름을 열성형 → 3D 형상으로 변형
④ 금형에 넣고 사출 → 수지가 뒤에 채워짐
⑤ 완성: 표면이 터치 센서이자 디스플레이이자 구조 부품
```

### IME의 분자적 도전

```
열성형 시 필름이 늘어남
→ 전도성 잉크도 함께 늘어남
→ 잉크 입자 간 거리가 벌어짐 → 저항↑
→ 너무 많이 늘어나면 회로 단선

사출 시 200°C+ 용융 수지가 필름 뒷면에 닿음
→ 필름/잉크/부품이 이 온도를 견뎌야
```

### IME가 바꾸는 것

```
기존:
  플라스틱 부품 + PCB + 스위치 + 하네스 + 조립
  → 부품 수 많음

IME:
  하나의 성형 부품 = 구조 + 회로 + 센서 + 조명
  → 부품 수 70~80% 감소
  → 중량 50% 감소
  → 두께 70% 감소
```

**적용 사례:** 공조 패널, 스티어링 휠 버튼, 도어 암레스트, 센터콘솔

## 4. 디스플레이 커버

### 요구사항

```
보호 (충격, 스크래치)
반사 방지 (눈부심)
터치 감지 (정전용량)
곡면 형상 (디자인)
```

### 소재 선택지

| 소재 | 장점 | 단점 |
|------|------|------|
| 유리 | 스크래치↓, 광학↑ | 무겁고 깨짐 |
| PC | 가볍고 안 깨짐 | 스크래치↑↑ |
| PMMA | 광학 최고 | 깨지기 쉬움 |

### PC의 스크래치 문제 해결

```
PC가 스크래치에 약한 분자적 이유:
  비정질 → 표면이 부드러움
  연필 경도: ~B (유리는 6H+)

해결: 하드코팅(Hard Coating)
  PC 표면에 실리콘계/아크릴계 하드코트 도포
  → 두께 5~15 μm
  → 경도 3H~9H로 향상
  → Si-O-Si 네트워크가 경도를 제공
```

### 반사 방지 (AR Coating)

```
반사 원리:
  빛이 공기(n=1.0) → PC(n=1.59) 경계를 지날 때
  → 굴절률 차이만큼 반사 (~5% per surface)

반사 방지 코팅:
  PC 위에 굴절률이 중간인 박막 (MgF₂, SiO₂)
  → 코팅 두께를 빛 파장의 1/4로 맞추면
  → 상면 반사파와 하면 반사파가 상쇄 간섭
  → 반사율 < 1%
```

## 5. 미래: 소재가 곧 기능이 되는 시대

| 기술 | 원리 | 적용 |
|------|------|------|
| **구조색** | 나노 구조 → 빛의 간섭 → 색상 | 안료 없이 색상 구현 |
| **자가치유** | 가역적 공유결합 또는 형상 기억 | 스크래치 자동 회복 |
| **감지 소재** | 압전 고분자(PVDF) 등 | 표면 전체가 센서 |
| **에너지 하베스팅** | 진동/열/빛 → 전기 | 선루프 태양전지 등 |

## 핵심 정리

```
전장 통합이 소재에 요구하는 새로운 물성:

광학:   투과율, 굴절률, 헤이즈 → 빛을 통과/제어
전기:   유전율, 면저항, 전도성 → 터치/센서/회로
열:     고온 안정성 유지하면서 위의 기능 공존
기계:   Day 1~10의 기존 요구 전부 유효

모든 새 물성의 뿌리:
  광학 → Day 2: 전자의 에너지 흡수와 빛의 상호작용
  전기 → Day 2: 전자 배치, 자유 전자의 유무
  유전율 → Day 2: 분자의 극성
  코팅 접착 → 옆산 1: 표면 에너지
```
