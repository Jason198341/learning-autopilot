# 옆산 1: 표면 처리의 과학 — 왜 PP에는 도장이 안 붙는가

## 모든 건 표면 에너지로 시작한다

페인트든 접착제든 코팅이든, 뭔가를 표면에 붙이려면 그 액체가 표면 위에 **퍼져야(wetting)** 해.

```
유리 위의 물:        납작하게 퍼짐 (접촉각 < 30°)
PP 위의 물:          동그란 방울 (접촉각 > 90°)
테플론 위의 물:      거의 완벽한 구 (접촉각 > 110°)
```

이 차이가 **표면 에너지(surface energy)**야.

---

## 표면 에너지란

분자 수준에서 생각해. 표면의 분자는 한쪽이 비어 있어. 이 불균형이 표면 에너지야.

```
내부 분자:   ← → 사방에서 당김 = 균형
             ↑ ↓

표면 분자:   ← → 옆에서 당김
             ↓   아래서 당김
             ↑   위에는 아무것도 없음 = 불균형 = 표면 에너지
```

- **표면 에너지가 높은 소재**: "나한테 와서 붙어" → 액체가 잘 퍼짐
- **표면 에너지가 낮은 소재**: "나 혼자 괜찮아" → 방울이 맺힘

---

## 소재별 표면 에너지

| 소재 | 표면 에너지 (mJ/m²) | 도장 가능? |
|------|---------------------|-----------|
| 알루미늄 | ~500 | 바로 가능 |
| 유리 | ~300 | 바로 가능 |
| PA (나일론) | ~46 | 가능 |
| ABS | ~42 | 가능 ← 도장에 적합 |
| PC | ~40 | 가능 |
| **PP** | **~30** | **불가** ← 여기가 문제 |
| PE | ~31 | 불가 |
| PTFE (테플론) | ~18 | 거의 불가능 |

도장/접착이 잘 되려면 보통 **38 mJ/m² 이상**이 필요. PP는 30.

---

## 왜 PP의 표면 에너지가 낮은가

PP는 탄소와 수소로만 이루어진 **비극성** 고분자. 사슬 사이 힘이 약한 런던 분산력뿐.

- ABS: 니트릴기(C≡N) → 극성기 있음 → 표면 에너지↑
- PA: 아미드기(N-H, C=O) → 극성기 있음 → 표면 에너지↑
- PP: 극성기 없음 → 표면 에너지↓

---

## PP에 도장을 붙이는 방법

### 방법 1: 화염 처리 (Flame Treatment)

```
프로판/부탄 화염을 PP 표면에 0.5~1초 스침
→ 화염 속의 산소 라디칼이 PP 표면의 C-H를 공격
→ C-H → C=O, C-OH, COOH (극성기 생성)
→ 표면 에너지: 30 → 50+ mJ/m²

분자 수준:
화염 전:  —CH₂—CH(CH₃)—CH₂—  (비극성)
화염 후:  —CH₂—C(=O)—CH₂—     (극성기 도입)
```

**자동차 범퍼 도장 라인에서 가장 많이 쓰는 방법**

### 방법 2: 플라즈마 처리 (Plasma Treatment)

```
저압 또는 대기압 플라즈마를 표면에 조사
→ 플라즈마 속 활성종(O*, OH*, O₃)이 표면을 공격
→ 화염 처리와 같은 원리로 극성기 생성
→ 표면 에너지: 30 → 55+ mJ/m²
```

화염보다 더 균일한 처리 가능, 열에 민감한 부품에 적합

### 방법 3: 프라이머 (Primer / Adhesion Promoter)

```
염소화 폴리올레핀(CPO) 프라이머를 뿌림
→ CPO의 올레핀 부분이 PP 표면과 분산력으로 결합
→ CPO의 염소 부분이 극성을 제공 → 도료와 결합
→ PP와 도료 사이의 '통역사' 역할

PP 표면 ←(분산력)→ CPO의 올레핀 부분
                    CPO의 Cl 부분 ←(극성 상호작용)→ 도료
```

### 방법 4: 코로나 처리 (Corona Treatment)

- 고전압 방전으로 공기를 이온화
- 필름/시트 같은 얇은 소재에 주로 사용
- 3D 사출 부품에는 균일성 문제로 적용 어려움

## 도장 말고 다른 표면 처리

### 증착 (Metallization)

```
진공 챔버에서 알루미늄을 기화
→ Al 원자가 플라스틱 표면에 내려앉음
→ 수십 nm 두께의 금속 박막 형성
→ 거울 같은 광택 = 크롬 도금 외관

3층 구조: 베이스코트 / Al / 탑코트
```

적용: 라디에이터 그릴, 도어 핸들, 내장 가니쉬의 크롬 룩

### IMD (In-Mold Decoration)

```
인쇄된 필름을 금형에 넣고 사출
→ 수지가 필름 뒤에 채워지면서 일체화
→ 별도 도장 공정 불필요

용융 수지(200°C+)가 필름 뒷면에 접촉
→ 필름의 고분자와 수지의 사슬이 서로 침투
→ Day 7 용착과 같은 원리
```

---

## 접착의 4가지 메커니즘

| 메커니즘 | 설명 | 적용 |
|----------|------|------|
| **기계적 결합** | 접착제가 표면 요철에 침투 → 앵커 역할 | 샌딩 후 접착 |
| **분자간 힘** | 반데르발스력 + 극성 상호작용 | 표면 에너지가 핵심 |
| **확산** | 같은 고분자끼리 사슬이 침투 | ABS 아세톤 접착 |
| **화학 결합** | 실제 공유결합 형성 | 실란 커플링제 |

## 실무 의사결정: 표면 처리 선택

```
소재가 뭐야?
    │
    ├─ PP/PE (표면 에너지 < 35)
    │   ├─ 도장? → 화염 처리 + CPO 프라이머 + 도장
    │   ├─ 접착? → 화염/플라즈마 + 전용 접착제
    │   └─ 무도장? → IMD, 또는 착색 컴파운드로 해결
    │
    ├─ ABS / PC/ABS (표면 에너지 > 38)
    │   ├─ 도장? → 직접 도장 가능
    │   └─ 증착? → 베이스코트 + 증착 + 탑코트
    │
    └─ PA / PBT (극성 소재)
        └─ 도장/접착? → 잘 붙지만 흡습에 의한 장기 열화 주의
```
