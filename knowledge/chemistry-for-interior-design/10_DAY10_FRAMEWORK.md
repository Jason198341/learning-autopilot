# DAY 10: 설계 의사결정 프레임워크 — 모든 Day가 하나로 연결되는 순간

## 새 프로젝트가 시작됐다

신차 프로젝트가 떨어졌어. 대시보드를 새로 설계해야 해.

수십 가지 결정을 내려야 하는데, 그 결정들이 **전부 Day 1~9에서 배운 분자 과학에 근거해**.

---

## Phase 1: 요구사항 정의 — 분자에게 뭘 시킬 건가

### 환경 조건 매핑

```
온도 범위:
  사용 환경: -30°C (러시아 겨울) ~ +80°C (인도 여름)
  단기 피크: +110°C (직사광선 대시보드 표면)

  → Tg가 -30°C보다 낮아야 함 (안 깨지려면)
  → HDT가 +110°C보다 높아야 함 (안 처지려면)
  → Day 3: Tg, Tm의 의미 / Day 8: HDT 시험

UV 노출:
  대시보드 상면: 직사광선 직격
  도어트림 상단: 간접광
  도어트림 하단: 거의 없음

  → 부위별 UV 안정제 차등 적용
  → Day 6: UV 열화 / Day 6+: HALS + UVA 조합

기계적 하중:
  정적: 물건 올려놓음, 팔꿈치 지지
  동적: 에어백 전개, 무릎 충격 (ECE R21)
  반복: 글로브박스 개폐 10,000회

  → Day 8: 인장/충격/굴곡 시험 / Day 7: 스냅핏 허용 변형률

화학적 노출:
  핸드크림, 선크림, 세정제, 음료

  → Day 9: ESC 저항성 / 소재별 내화학성 필수

규제:
  연소: FMVSS 302 (전 부품)
  VOC: 현대차 MS300-57
  FOG: 현대차 MS300-55
  내광성: SAE J2412
```

## Phase 2: 소재 선정 — 분자 구조를 고르는 것

### 의사결정 트리

```
이 부품의 주요 기능은?
    │
    ├─ 구조/강성 (기재, 브라켓)
    │   ├─ 경량화 우선? → PP+LGF30
    │   ├─ 비용 우선? → PP+TD20+EPDM15 (자동차 표준)
    │   ├─ 고온/고강성? → PA6+GF30 또는 PBT+GF30
    │   └─ 금속 대체? → PP+LGF40 또는 PA66+GF50
    │
    ├─ 외관/촉감 (표피, 가니쉬)
    │   ├─ 소프트 터치? → TPO, PVC, TPU
    │   ├─ 고광택? → ABS, PC/ABS, PMMA
    │   ├─ 도장? → ABS, PC/ABS
    │   └─ 무도장 고외관? → PP+핵제, 저광택 컴파운드
    │
    ├─ 쿠션/흡음 (폼, 패드)
    │   ├─ 촉감 쿠션? → PU 폼 (반경질)
    │   ├─ 에너지 흡수? → EPP, EPS
    │   └─ 흡음? → PU 폼 (연질) 또는 펠트
    │
    └─ 투명/반투명 (렌즈, 디스플레이 커버)
        ├─ 고투명+내충격? → PC
        ├─ 고투명+내스크래치? → PMMA
        └─ 저비용 반투명? → PP+핵제(Millad)
```

### 소재 선정 체크리스트

```
□ Tg < 최저 사용 온도? (저온 충격)
□ HDT > 최고 사용 온도? (고온 변형)
□ CTE가 인접 부품과 호환? (열팽창 단차)
□ UV 등급이 노출 조건에 적합? (내후성)
□ 연소 등급 충족? (FMVSS 302)
□ VOC/FOG 규격 충족? (실내 공기질)
□ ESC 저항성이 접촉 화학물질에 적합? (크랙 방지)
□ 재활용 가능? (EU ELV 지침: 차량 85% 재활용)
□ 공급사 2곳 이상 확보? (공급 리스크)
□ 기존 금형/설비와 호환? (투자비)
```

## Phase 3: 부품 설계 — 분자가 견딜 수 있는 형상

### 벽 두께 설계

```
기본 벽 두께: PP 내장재 2.0 ~ 3.0mm가 표준

< 2.0mm: 유동 거리 부족 → 숏샷 위험, 강성 부족
> 3.0mm: 냉각 시간↑ → 비용↑, 싱크마크 위험, 수축량↑

핵심: 균일 벽 두께
두께 변화 → 수축률 차이 → 싱크/워피지
불가피한 변화는 점진적으로 (3:1 이하 테이퍼)
```

### 리브(Rib) 설계

```
리브 두께 = 벽 두께 × 0.5~0.7
  → 리브가 두꺼우면 교차부 열집중 → 싱크마크

리브 높이 ≤ 벽 두께 × 3
  → 너무 높으면 충전 불량, 이형 시 파손

리브 뿌리 R값 ≥ 0.25 × 벽 두께
  → 날카로운 코너 = 응력 집중 → 크랙 시작점
```

### 보스(Boss) 설계

```
보스 외경 = 나사 외경 × 2.0~2.5
보스 벽 두께 = 벽 두께 × 0.5~0.75

→ 너무 두꺼우면 싱크마크
→ 너무 얇으면 나사 조임 시 크랙
→ 반복 체결 필요하면 금속 인서트
```

### 스냅핏 설계

```
허용 변형률:
  PP+TD20: ~2.5%
  ABS: ~3.5%
  PC/ABS: ~4.0%
  PA+GF30: ~1.5%

훅 설계:
  최대 변형량 = 훅 길이 × 허용 변형률
  예: PP+TD20, 훅 길이 10mm
  → 최대 변형 = 10 × 0.025 = 0.25mm

해제 각도:
  영구 체결: 90°
  분리 가능: 30~45°
```

## Phase 4: 금형/가공 조건

### 게이트 위치 결정

```
① 두꺼운 곳에서 얇은 곳으로 흐르게
② 웰드라인이 비가시면/비구조면에 오게
③ 유동 거리가 최소화되게
④ 에어백 도어 근처 피하기

시뮬레이션: Moldflow, Moldex3D
```

### 가공 조건 최적화

```
수지 온도 (PP: 200~240°C):
  → Tm(165°C) 이상으로 사슬을 충분히 풀어야
  → 너무 높으면: 열분해 → 은줄, 변색
  → 너무 낮으면: 유동 부족 → 숏샷

금형 온도 (PP: 30~60°C):
  → 높으면: 표면 외관↑, 웰드라인↓, 사이클↑
  → 낮으면: 사이클↓, 표면↓, 잔류 응력↑

사출 속도:
  → 빠르면: 유동성↑, 플래시/젯팅 위험
  → 느리면: 숏샷, 웰드라인 심화

보압 (Packing Pressure):
  → 높으면: 싱크↓, 잔류 응력↑, 플래시 위험
  → 낮으면: 싱크↑, 수축↑
```

## Phase 5: 검증 — 분자에게 증명하라고 요구

### 시험 단계

```
Level 1: 소재 시험 (시편 단위)
  → 인장, 충격, 굴곡, HDT, CLTE
  → 소재 선정이 맞았는지 확인

Level 2: 부품 시험 (단품 단위)
  → 내후성, 내열노화, VOC, FOG
  → 연소시험 (FMVSS 302)
  → 부품 형상에서의 강도/강성

Level 3: 모듈 시험 (조립체 단위)
  → 에어백 전개 시험
  → BSR 시험 (진동 환경)
  → 단차/간극 측정 (열사이클 후)

Level 4: 차량 시험 (실차)
  → 내구 주행 시험
  → 환경 챔버 (고온/저온/고습)
  → 필드 테스트 (인도, 러시아, 중동)
```

### 불합격 시 의사결정

```
시험 불합격
    │
    ├─ 소재 문제?
    │   ├─ 물성 부족 → 첨가제 조정
    │   ├─ 내후성 부족 → UV 안정제↑
    │   ├─ VOC/FOG 초과 → 저방출 등급
    │   └─ 근본 한계 → 소재 변경 (Phase 2로)
    │
    ├─ 설계 문제?
    │   ├─ 강성 부족 → 리브 추가
    │   ├─ 응력 집중 → R값 확보
    │   ├─ 열팽창 단차 → 체결점 재배치
    │   └─ BSR → 간극/접촉면 재설계
    │
    └─ 가공 문제?
        ├─ 웰드라인 → 게이트 변경
        ├─ 싱크/워피지 → 조건 + 설계 조정
        └─ 외관 불량 → 금형 수정
```

## Phase 6: 양산/원가

### 원가 구조

```
소재비 (40~60%):
  PP 수지: ~$1.2/kg
  탈크 MB: ~$0.8/kg (탈크↑ = 원가↓)
  EPDM: ~$2.0/kg (고가)
  UV 안정제: ~$15/kg
  GF: ~$1.5/kg

가공비 (20~30%):
  사이클 타임 × 기계 시간당 비용
  → 벽 두께↑ = 냉각↑ = 비용↑
  → 핵제 → 결정화↑ → 사이클↓

금형비 (10~20%):
  대시보드 금형: 5~10억 원
  → 설계 단순화 = 금형비↓

후가공비:
  도장, 패드인쇄, 레이저 각인
  → 무도장 전환 = 후가공비 제거 + VOC↓
```

### 원가절감(VA/VE) 의사결정

| 방법 | 절감 원리 | 리스크 |
|------|----------|--------|
| 탈크 함량↑ | 저가 충전제 증량 | 충격↓, 외관↓ |
| 벽 두께↓ | 소재량↓, 사이클↓ | 강성↓, 숏샷 |
| 부품 통합 | 금형비↓, 조립공수↓ | 금형 복잡도↑ |
| 무도장 전환 | 도장 공정 제거 | 외관 관리 어려움 |
| 재활용 소재(PCR) | 원료비↓, ESG | 물성 산포↑, 색상 제한 |

## 전체 흐름 요약

```
Day 1-2: 원자 → 전자 → 결합 → 분자간 힘
    ↓
Day 3: 결합 → 고분자 사슬 → Tg, Tm, 결정성
    ↓
Day 4: 사슬 + 첨가제 → 컴파운드 (자동차 등급 소재)
    ↓
Day 5: 컴파운드 → 사출 성형 → 부품
    ↓
Day 6: 표면 물성 → 촉감, 열전도, 변색의 분자 원리
    ↓
Day 6+: 첨가제 심화 → 12종 첨가제의 메커니즘과 튜닝
    ↓
Day 7: 부품 → 조립 → 체결/열팽창/BSR의 분자 원리
    ↓
Day 8: 시험 = 분자에게 질문 → 규격의 분자적 의미
    ↓
Day 9: 불량 = 분자의 신호 → 근본 원인 진단
    ↓
Day 10: 전체 통합 → 프로젝트 의사결정 프레임워크
```

## 마무리: 네가 얻은 것

이 10일 과정을 통과하면, 소재 데이터시트의 숫자가 **분자의 이야기**로 읽혀.

"인장강도 25MPa" → "사슬 간 반데르발스력의 총합이 25메가파스칼의 응력을 버틴다"

불량이 생기면 **"어떤 결합이 어떤 조건에서 어떻게 깨졌는가"**로 진단할 수 있어.

이건 단순히 지식이 아니라 **엔지니어링 판단력의 근거**야.

소재 업체가 새 등급을 제안할 때, 원가절감 아이디어가 나올 때, 필드 클레임이 들어올 때 —

**"그래서 분자 수준에서 뭐가 달라지는데?"**라고 물을 수 있는 힘.
