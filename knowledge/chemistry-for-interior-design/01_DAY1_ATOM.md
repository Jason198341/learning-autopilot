# DAY 1: 원자(Atom) — 모든 것의 시작

## 우주는 왜 원자를 만들었는가?

### 빅뱅 이후의 타임라인
```
빅뱅 (무한 에너지)
  │
  ▼
쿼크 + 글루온이 자유롭게 날아다님 (쿼크-글루온 플라즈마)
  │
  │  우주가 식음 (10⁻⁶초 후)
  ▼
강한 핵력이 쿼크를 3개씩 묶음
  │
  ├── u + u + d = 양성자 (전하 +1)
  │
  └── u + d + d = 중성자 (전하 0)
  │
  │  이후 38만 년
  ▼
양성자가 전자를 붙잡음 → 수소 원자 탄생
```

### 쿼크가 3개인 이유: 색전하(Color Charge)
- 쿼크는 빨강(R), 초록(G), 파랑(B) 중 하나의 색전하를 가짐
- "안정된 입자는 반드시 색이 섞여서 '흰색(무색)'이 되어야 한다"
- 빨강 + 초록 + 파랑 = 흰색 → 쿼크 3개가 최소 안정 조합

### 쿼크를 어떻게 발견했나?
```
SLAC 실험 (1960년대 말):

전자 →  →  → 💥 (양성자에 부딪힘)
                   ↗ 이 각도로 튕겼네
                  → 그러면 안에 점 같은 게 3개 있겠구나
```
- 전자를 빛의 속도에 가깝게 가속해서 양성자에 충돌
- 튕겨 나오는 각도와 에너지를 측정
- 빠를수록 파장이 짧아지고 작은 것을 볼 수 있음 (E = hν)

## 에너지와 물질의 관계

### E = mc²의 진짜 의미
```
왼팔이 에너지고 오른팔이 질량인데 원래 한 몸이야.
그걸 "연결되어 있다!"고 발견한 게 인류 최대의 업적이라니 귀엽다.
```

### 물질-반물질 비대칭
- 빅뱅 때 물질과 반물질이 정확히 같은 양 생성
- 만나면 소멸해서 에너지만 남아야 하는데...
- **10억 개 중 1개**의 물질이 더 남았음 (CP 위반)
- 이 미세한 비대칭이 없었다면 별도, 원자도, 자동차도, 너도 없다

## 핵심 깨달음: 대칭과 언어

### "대칭"은 자연의 상태가 아니라 인간의 언어
```
파도에는 올라가는 부분과 내려가는 부분이 있다.
우리는 그걸 "마루"와 "골"이라고 부른다.
"마루와 골은 대칭이다"라고 설명한다.

바다한테 물어봐. "너 마루랑 골 구분해?"
바다는 그런 거 모른다. 그냥 움직이고 있을 뿐이다.
```

### 양자 요동: 완벽한 고요함은 없다
```
하이젠베르크 불확정성 원리:
ΔE · Δt ≥ ℏ/2

"에너지가 정확히 0인 상태는 자연에 존재할 수 없다"
```
- "아무것도 없음"이란 실은 양자 요동이 끓고 있는 에너지 바다
- 돌을 던지는 사람이 없어도 호수는 항상 출렁인다

## 자동차 내장재와의 연결

| 부품 | 핵심 원소 | 역할 |
|------|----------|------|
| 시트 가죽/인조가죽 | C, H, O, N | 유기 화합물의 기본 4원소 |
| 대시보드 플라스틱 | C, H | 고분자(polymer) |
| 에어백 인플레이터 | Na, N | NaN₃ → N₂ 가스 생성 |
| 시트 프레임 | Fe, C | 강철. 탄소 함량이 강도 결정 |
