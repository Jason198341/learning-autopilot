# DAY 7: 조립의 과학 — 왜 그렇게 체결하는가

## 부품 하나가 아니라 시스템이다

대시보드를 봐. 하나의 덩어리처럼 보이지만 실제로는 10~20개 이상의 개별 부품이 조립된 거야.

### 왜 하나로 안 만들까? 세 가지 이유

**①소재가 다르다**
- 기재는 PP+TD20(강성)
- 표피는 PVC/TPO(촉감)
- 폼은 PU(쿠션)
- Day 3~4에서 배웠듯이 각 기능에 최적화된 고분자가 다름

**②성형 한계**
- 사출 금형은 언더컷(undercut)이 있으면 빼낼 수가 없어
- 복잡한 형상을 하나의 금형으로 만들면 금형 비용이 천문학적
- 분할해서 각각 성형한 다음 조립하는 게 경제적

**③정비성**
- 에어백, 공조 덕트, 전장 부품은 교체/수리가 필요
- 일체형이면 대시보드 전체를 버려야 함
- 모듈화하면 해당 부분만 분리 가능

## 체결 방식 3가지와 분자 수준의 원리

### 1. 스냅핏 (Snap-fit) — 탄성 변형

```
훅이 밀려들어가면서 휘었다가 → 걸림턱을 넘으면 원래로 복귀
이게 Day 3의 고분자 사슬 거동이야.

힘을 가함 → 사슬이 펴짐(탄성 변형)
→ 힘을 제거 → 사슬이 다시 엉킴(복귀)
→ 체결 유지
```

**핵심 물성**: 굴곡탄성률 + 파단신율
- 너무 딱딱하면(탈크 과다) → 휘기 전에 부러짐
- 너무 유연하면(EPDM 과다) → 체결력 부족, 헐거움

**허용 변형률(allowable strain)**
| 소재 | 허용 변형률 |
|------|------------|
| PP+TD20 | ~2.5% |
| ABS | ~3.5% |
| PC/ABS | ~4.0% |
| PA+GF30 | ~1.5% |

탈크를 30%로 올리면 허용 변형률이 1.5%로 떨어져서 기존 스냅핏이 깨질 수 있어.

### 2. 나사/볼트 체결 — 크리프와의 싸움

```
나사를 조이면 → 플라스틱에 압축 응력이 걸림
→ 처음에는 괜찮음
→ 시간이 지나면 고분자 사슬이 천천히 미끄러짐
→ 응력이 풀림 → 체결력 저하

이것이 크리프(creep)와 응력이완(stress relaxation)
```

Day 3의 열가소성 본질이야. 사슬이 화학결합이 아니라 분자간 힘(반데르발스)으로 붙어 있으니까, 장시간 하중 하에서 서서히 흘러.

**해결책:**
- 금속 인서트(insert) 삽입
- 보스 주변 리브(rib) 보강
- 토크 관리 (과토크 → 크랙, 저토크 → 진동에 풀림)

### 3. 용착/접착 — 분자 수준의 결합

```
진동 용착(vibration welding):
두 부품을 맞대고 진동시킴 → 마찰열 발생
→ 접합면의 고분자가 용융(Tm 이상)
→ 사슬이 서로 침투(interdiffusion)
→ 냉각 → 하나의 고분자 네트워크 형성
→ 사실상 하나의 부품이 됨
```

Day 5의 사출 성형과 같은 원리: 용융 → 유동 → 냉각 → 고화

**접착제(adhesive)**: 고분자 사슬이 양쪽 표면의 미세 요철에 침투하고, 분자간 힘(반데르발스, 수소결합)으로 붙음 (Day 2)

## 조립에서 생기는 문제

### 열팽창 차이 (CTE mismatch)

```
대시보드 기재: PP+TD20 (CTE ≈ 40~60 μm/m·°C)
크래시패드 표피: PVC (CTE ≈ 70~80 μm/m·°C)
금속 브라켓: Steel (CTE ≈ 12 μm/m·°C)

여름 80°C → 겨울 -30°C = 110°C 온도차
1m 길이 부품이면:
  PP: 110 × 50 × 10⁻⁶ = 5.5mm 변화
  Steel: 110 × 12 × 10⁻⁶ = 1.3mm 변화
  차이: 4.2mm
```

4.2mm면 눈에 보여. 단차(gap & flush) 불량의 주범.

**해결**: 슬롯 홀(slot hole)로 열팽창 흡수, 체결점 간격 계산 배치

**CTE가 왜 다를까?** (Day 3)
- 고분자 사슬: 열 → 진동 커짐 → 사슬 간 거리 벌어짐 → 크게 팽창
- 금속: 결정 격자가 단단 → 덜 팽창
- 탈크/GF: 무기물이 팽창 억제 → CTE 낮아짐

### 소음 (BSR: Buzz, Squeak, Rattle)

```
두 부품이 접촉한 상태에서:
진동 → 미세 슬라이딩 → 마찰음(squeak)
간극 → 부품이 부딪힘 → 딸각음(rattle)
공명 → 특정 주파수에서 증폭 → 윙윙(buzz)
```

BSR은 고객 불만 1위. 분자 수준에서 두 고분자 표면 접촉 시 마찰계수가 핵심.

**결정 요소**: 표면 에너지(Day 2), 활제 함량(Day 6+), 표면 거칠기

**해결**: 펠트 테이프, 실리콘 그리스, 저마찰 코팅

## 실무 연결: 부품 간 관계 맵

```
[크래시패드 표피 PVC/TPO]
    │ 접착 or 진공성형
    ▼
[PU 폼 — 쿠션/촉감]
    │ 폼인플레이스(foam-in-place)
    ▼
[PP+TD20 기재 — 구조]
    │ 스냅핏 + 나사
    ├──→ [에어백 도어] — 티어시임(tear seam) 설계
    ├──→ [공조 덕트 연결부] — 클립 체결
    ├──→ [계기판 하우징] — 나사 + 인서트
    │ 볼트 체결
    ▼
[CCAB 크로스멤버 — Steel/Mg]
    │ 볼트
    ▼
[차체(BIW)]
```

## Day 7 정리

```
스냅핏   → 탄성 변형 → 고분자 사슬의 가역적 변형 (Day 3)
나사체결 → 크리프    → 사슬의 비가역적 미끄러짐 (Day 3)
용착     → 재용융    → Tm 이상 가열 → 사슬 침투 (Day 5)
접착     → 분자간 힘 → 반데르발스/수소결합 (Day 2)
열팽창   → CTE 차이  → 사슬 진동 vs 결정 격자 (Day 3)
BSR      → 마찰      → 표면 에너지 + 활제 (Day 2, 6+)
```
