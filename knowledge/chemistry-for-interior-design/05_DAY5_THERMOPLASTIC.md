# DAY 5: 열가소성 vs 열경화성 — 녹는 놈과 안 녹는 놈

## 핵심 비유: 초콜릿 vs 계란

```
초콜릿 = 열가소성 (Thermoplastic)
  → 열을 가하면 녹음
  → 식으면 굳음
  → 다시 열을 가하면 또 녹음
  → 몇 번이고 반복 가능

계란 = 열경화성 (Thermoset)
  → 열을 가하면 익음
  → 식어도 날계란으로 안 돌아감
  → 다시 열을 가하면 그냥 탐
```

## 분자 구조의 차이

### 열가소성 — 사슬이 따로 놀아
```
가열 전:
  ~~~~~~~~
  ~~~~~~~~    사슬끼리 엉켜있지만
  ~~~~~~~~    연결은 안 되어있음

가열하면:
  ~~~  ~~~
    ~~  ~~    사슬이 풀려서 자유롭게 움직임
  ~~  ~~~     → 녹음. 흐름.

식히면:
  ~~~~~~~~
  ~~~~~~~~    다시 엉킴
  ~~~~~~~~    → 굳음. 원래대로.
```

### 열경화성 — 사슬이 연결되어 있어
```
가열 전 (경화 전):
  ~~~~~~~~
  ~~~~~~~~    아직 가교 안 됨
  ~~~~~~~~

가열하면 (경화):
  ──╫──╫──
  ──╫──╫──    가교(╫)가 형성됨
  ──╫──╫──    사슬끼리 화학결합으로 연결

다시 가열하면:
  ──╫──╫──
  ──╫──╫──    가교가 안 끊어짐
  ──╫──╫──    → 안 녹음. 그냥 타버림.
```

## 비교 정리

|  | 열가소성 | 열경화성 |
|--|---------|---------|
| 분자 구조 | 사슬이 따로 | 사슬이 가교로 연결 |
| 비유 | 국수, 초콜릿 | 그물, 계란 |
| 열을 가하면 | 녹음 | 안 녹음 (타버림) |
| 재성형 가능? | 가능 (여러 번) | 불가능 (1회성) |
| 재활용? | 가능 | 어려움 |
| 강도 | 상대적으로 낮음 | 높음 |
| 내열성 | 상대적으로 낮음 | 높음 |

## 제조 공정의 차이

### 열가소성 공정 — 사출 성형 (Injection Molding)
```
1. 플라스틱 알갱이 (펠릿)를 호퍼에 넣음
2. 가열해서 녹임 (200~300°C)
3. 스크류로 밀어서 금형(틀)에 고압으로 쏴넣음
4. 금형 안에서 식힘
5. 금형 열고 부품 꺼냄
6. 끝. 1개 만드는 데 30초~2분.

[펠릿] → [가열] → [사출] → [냉각] → [완성]

✓ 물리적 변화 (녹였다 굳힘)
✓ 복잡한 모양 대량생산 가능
✓ 재활용 가능
```

### 열경화성 공정 — 반응 사출 성형 (RIM)
```
1. A액과 B액을 준비
2. 두 액체를 고압으로 혼합하며 금형에 주입
3. 금형 안에서 화학반응 시작 (가교 형성)
4. 반응 완료될 때까지 대기 (수 분)
5. 금형 열고 부품 꺼냄

[A액] + [B액] → [혼합 주입] → [반응 경화] → [완성]

✓ 화학적 변화 (새로운 결합 생성)
✓ 되돌릴 수 없음
✓ 내열성, 내구성 우수
```

## 자동차 내장재에서의 적용

### 열가소성 부품들
| 부품 | 소재 | 공정 | 이유 |
|------|------|------|------|
| 대시보드 | PP | 사출 성형 | 복잡한 형상, 대량생산 |
| 센터콘솔 | ABS | 사출 성형 | 강도 + 광택 |
| 도어트림 기재 | PP | 사출 성형 | 경량화 |
| 시트 표면 | PU/PVC | 캘린더링/진공 | 촉감, 외관 |
| 에어백 하우징 | PA(나일론) | 사출 성형 | 내충격성 |

### 열경화성 부품들
| 부품 | 소재 | 공정 | 이유 |
|------|------|------|------|
| 시트 쿠션 | PU 폼 | 반응 사출 | 내열성, 복원력 |
| 구조 접착제 | 에폭시 | 경화 접착 | 내구성 |
| 선루프 씰 | EPDM 고무 | 가황 성형 | 내열, 내후성 |
| 헤드라이너 접착 | 에폭시 | 경화 접착 | 내구성 |

## 왜 시트 쿠션은 열경화성인가?

```
시트에 앉으면 체온 37°C
여름 차 안 온도: 80°C까지 상승

만약 시트 쿠션이 열가소성이면?
→ 여름에 쿠션이 녹아서 주저앉음

열경화성이라 가교가 유지됨
→ 열에도 모양을 유지
→ 앉았다 일어나면 복원
```

## 팀장의 판단 기준

```
이 부품은 열이 많이 가해지나?
  → YES: 열경화성 고려

복잡한 모양을 대량생산해야 하나?
  → YES: 열가소성 유리

재활용이 중요한가?
  → YES: 열가소성

둘 다 필요하면?
  → 복합 구조 (기재는 열가소성 + 코팅은 열경화성)
```

## 제조 비용 비교

| 항목 | 열가소성 | 열경화성 |
|------|---------|---------|
| 금형 비용 | 높음 (고압 필요) | 낮음 |
| 사이클 타임 | 빠름 (30초~2분) | 느림 (수 분) |
| 재료 비용 | 낮음~중간 | 중간~높음 |
| 재작업 가능 | 가능 | 불가능 |
| 대량생산 적합도 | 높음 | 중간 |
