# 옆산 2: 발포의 과학 — 공기를 가두는 기술

## 발포란 뭐냐

고분자 안에 **기포(cell)**를 만드는 것. 그게 전부야.

```
일반 사출 부품:   [████████████]  ← 속이 꽉 찬 고체
발포 부품:        [█ ○ █ ○ █ ○ █]  ← 고분자 벽 + 기포
```

---

## 기포가 하는 일 — 분자 수준에서

### 경량화
```
고분자 비중: ~0.9~1.4
공기 비중: ~0.001

발포 배율 2배 = 부피의 50%가 공기
→ 무게가 거의 절반
```

### 쿠션 (탄성)
```
손가락으로 폼을 누르면
→ 기포 벽(고분자)이 휘어짐 (탄성 변형)
→ 기포 안 공기가 압축됨
→ 손을 떼면 공기가 팽창 + 벽이 복귀
→ 푹신한 촉감
```

### 흡음
```
소리가 발포체에 들어오면
→ 기포 벽 사이의 좁은 통로를 지나면서
→ 공기와 벽 사이 마찰 → 진동 에너지 → 열 에너지로 변환
→ 소리가 줄어듦

핵심: 기포가 열려 있어야(연통 구조) 흡음
      기포가 닫혀 있으면(독립 구조) 차음
```

### 단열
```
열 전달 경로가 끊어짐

고체: 분자 → 분자 → 분자 (연속 열전도)
발포: 분자 → 공기 → 분자 → 공기 (공기가 끊어줌)

공기의 열전도율: 0.026 W/m·K
PP의 열전도율:  0.22 W/m·K (8배 차이)
```

---

## 기포는 어떻게 만드는가

### 방법 1: 화학 발포 (Chemical Foaming)

```
발포제를 수지에 섞음
→ 가열하면 발포제가 분해되면서 가스 발생
→ 가스가 용융 수지 안에서 기포를 형성
→ 냉각하면 기포가 고정

대표적 발포제:
  아조디카본아미드(ADCA): 분해 시 N₂, CO 발생 (분해 온도 ~200°C)
  탄산수소나트륨: 분해 시 CO₂ 발생 (분해 온도 ~150°C)
```

장점: 기존 사출기에 발포제 마스터배치만 추가
단점: 기포 크기 불균일, VOC/냄새 가능

### 방법 2: 물리 발포 (Physical Foaming)

```
초임계 유체(SCF)를 사출기 배럴에 직접 주입
→ 초임계 CO₂ 또는 N₂가 용융 수지에 용해
→ 금형에 사출되면 압력 강하
→ 급격한 압력 강하로 가스가 과포화 → 핵 생성 → 기포

초임계 유체란?
  CO₂: 31°C, 73기압 이상이면 액체도 기체도 아닌 상태
  → 액체처럼 밀도가 높아서 수지에 잘 녹고
  → 기체처럼 확산이 빨라서 균일하게 퍼짐

대표 기술: MuCell® (Trexel, MIT 발)
```

장점: 기포가 매우 작고 균일(마이크로셀, 5~50μm), VOC 없음
단점: 전용 설비 투자 필요

## 기포 형성의 과학

### 핵 생성 (Nucleation)

```
용융 수지 안에 가스가 녹아 있음
→ 압력이 떨어지면 가스가 과포화 상태
→ 기포가 되려면 '핵'이 필요

에너지 장벽을 낮추는 방법:
  ① 탈크 입자 표면에서 핵 생성 (이종 핵 생성)
  ② 압력 강하를 크고 빠르게
  ③ 가스 농도를 높게

핵이 많이 생기면 → 기포가 작고 많아짐 (물성에 유리)
핵이 적게 생기면 → 기포가 크고 적어짐
```

### 기포 성장 (Growth)

```
핵이 생긴 후 가스가 핵으로 확산 → 기포가 커짐

성장 속도 결정 요소:
  ① 가스 확산 속도 — 온도↑ → 확산↑
  ② 용융 점도 — 점도↑ → 성장 억제
  ③ 냉각 속도 — 빨리 굳으면 성장 멈춤
```

## 기포 구조의 두 가지 유형

### 독립 기포 (Closed Cell)

```
[벽|공기|벽|공기|벽]
 각 기포가 독립된 방

특성:
  - 압축 시 공기가 스프링 역할
  - 물/습기 침투 불가 → 방수
  - 흡음 나쁨, 차음에는 유리

용도: EPP 에너지 흡수체, 범퍼, 선바이저 코어
```

### 연통 기포 (Open Cell)

```
[벽  공기  벽  공기  벽]
      ↕        ↕
 기포끼리 통로로 연결

특성:
  - 압축 시 공기가 빠져나감 → 부드러운 쿠션
  - 물/습기 침투 가능
  - 흡음 우수

용도: PU 시트 폼, 헤드레스트, 흡음재
```

## 자동차 내장재의 주요 발포 적용

### 1. PU 폼 — 시트 쿠션

```
이소시아네이트(MDI) + 폴리올 → 반응 → PU 생성
동시에 물 + 이소시아네이트 → CO₂ 발생 → 발포

열경화성이라 여름에 녹지 않음

밀도:
  시트 쿠션: 40~60 kg/m³
  헤드레스트: 50~70 kg/m³
  암레스트: 80~120 kg/m³ (반경질)
```

### 2. 크래시패드 발포

```
대시보드 단면:
[표피 PVC/TPO]  0.5~1.0mm
[PU 폼]         5~10mm ← 이게 발포
[PP 기재]       2.5~3.0mm

PU 폼의 역할:
  ① 소프트 촉감
  ② 에너지 흡수 (HIC 값 저감)
  ③ 표피-기재 사이 완충
```

### 3. 마이크로셀 발포 사출 (MuCell®)

```
PP+TD20 대시보드 기재를 MuCell로 사출하면:
  기포 크기: 5~50 μm
  중량 감소: 10~15%
  사이클 타임: 10~20% 단축
  싱크마크: 거의 제거

trade-off:
  표면 외관: swirl mark 가능 → 기재(표피로 덮이는 부품)에 적합
  충격강도: 소폭 저하
```

### 4. EPP — 에너지 흡수

```
PP 비드를 스팀으로 발포 → 금형에서 성형

특성:
  비중: 0.02~0.20
  반복 충격 흡수: 충격 후 80%+ 형상 복귀
  (EPS는 한 번 깨지면 끝)

적용: 범퍼 에너지 흡수체, 도어 충격 패드, 트렁크 정리함
```

## 발포와 물성의 관계

```
발포 배율↑ (기포 비율 증가) 하면:

경량화     ↑↑↑  ← 이게 목적
쿠션감     ↑↑   (연통 구조일 때)
흡음       ↑↑   (연통 구조일 때)
단열       ↑↑   (독립 구조일 때)

강성       ↓↓   ← 고분자 벽이 얇아지니까
인장강도   ↓↓
충격강도   ↓
표면 외관  ↓↓
치수 정밀도 ↓
```

**핵심**: 필요한 만큼만 발포. 10~15% 경량화가 물성 저하 없는 실용적 한계.

---

## 기포 크기가 왜 중요한가

```
같은 발포 배율이라도:

큰 기포 (>200 μm):
  → 벽 두께 불균일 → 응력 집중 → 강도↓
  → 눈에 보임 → 외관↓

작은 기포 (<50 μm, 마이크로셀):
  → 균일한 벽 두께 → 응력 분산 → 강도 유지
  → 눈에 안 보임 → 외관 유지
```

**핵 생성 많이 → 기포 작고 많이 → 물성 유리**
