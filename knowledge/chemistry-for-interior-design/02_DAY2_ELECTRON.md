# DAY 2: 전자 배치 — 게임의 아이템 조합법

## 전자는 왜 핵에 붙어버리지 않냐

### 두 가지 에너지의 균형
```
에너지

  │╲                          ╱
  │  ╲ 운동 에너지            ╱ 위치 에너지
  │    ╲ (안으로 갈수록    ╱   (밖으로 갈수록
  │      ╲  폭발)       ╱      올라감)
  │        ╲          ╱
  │          ╲      ╱
  │            ╲  ╱
  │              V ← 1층. 균형점. 전체 에너지 최소.
  └──────────────────────────────
         핵    1층           바깥
```

### 불확정성 원리와 운동 에너지
```
Δx · Δp ≥ ℏ/2

전자를 핵 안으로 밀어 넣으려고 함
  │
  │  위치가 정확해짐 (Δx 감소)
  ▼
운동량이 미쳐 날뜀 (Δp 증가)
  │
  │  E = p²/2m
  ▼
운동 에너지가 폭발적으로 증가 → 핵 안에 머물 수 없음
```

## 전자의 자리 — 에너지 계단

### 층(n) 구조
```
       ┌──────── n=4  (4층)
       │
       ┌──────── n=3  (3층)
       │
       ┌──────── n=2  (2층)
       │
       ┌──────── n=1  (1층) ← 핵에 가장 가까움, 에너지 최저
       │
      ⚛ 핵
```
- 1.5층은 없다. 정수만 가능. 이것이 "양자"
- 각 층의 에너지: 1층(-13.6 eV) < 2층(-3.4 eV) < 3층(-1.51 eV) < 4층(-0.85 eV)

### 방의 종류: s, p, d, f
```
1층: s만                    → 방 1개  → 전자 최대 2개
2층: s + p(3개)             → 방 4개  → 전자 최대 8개
3층: s + p(3개) + d(5개)    → 방 9개  → 전자 최대 18개
4층: s + p + d + f(7개)     → 방 16개 → 전자 최대 32개
```

### 오비탈 모양
- **s 오비탈**: 공 모양 (구형)
- **p 오비탈**: 아령 모양, 3방향 (px, py, pz)
- **d 오비탈**: 복잡한 꽃잎 모양, 5개
- **f 오비탈**: 더 복잡한 모양, 7개

## 탄소의 전자 배치

```
C: 1s² 2s² 2p²

1층:  1s  [↑↓]           ← 꽉 참

2층:  2s  [↑↓]           ← 꽉 참
      2px [↑ ]           ← 반만 참
      2py [↑ ]           ← 반만 참
      2pz [  ]           ← 빈방

전자 총 6개: 2 + 2 + 1 + 1 + 0 = 6 ✓
빈자리 4개: 이게 결합의 손 4개
```

### 훈트의 규칙 (Hund's Rule)
- 빈방이 있으면 각자 다른 방에 1개씩 들어감
- 전자끼리 같은 방에 있으면 반발 → 에너지 높음
- 따로 앉아서 편안함 = 에너지 낮음 = 안정

## 전자의 층 이동과 빛

### 흡수와 방출
```
전자가 층을 이동할 때:

  올라감 → 에너지를 흡수 → 빛을 먹음
  내려감 → 에너지를 방출 → 빛을 뱉음

방출/흡수하는 에너지 = 두 층의 차이 = E = hν = 빛의 색깔
```

### 실제 응용
- 네온사인: 네온 원자의 전자가 층 이동하며 빨간 빛 방출
- 불꽃놀이: Na→노랑, Cu→초록, Sr→빨강
- LED: 반도체의 전자 층 이동 → 특정 색 빛

### 자동차 내장재 색상
```
빛이 내장재에 닿음
  → 특정 에너지의 빛을 전자가 흡수 (전자가 올라감)
  → 나머지 빛이 반사됨
  → 반사된 빛의 색 = 네 눈에 보이는 내장재 색상
```

## 에너지 관계 심화

### 비리얼 정리 (Virial Theorem)
```
운동 에너지 = -1/2 × 위치 에너지

1층:
  위치 에너지 = -27.2 eV
  운동 에너지 = +13.6 eV
  전체        = -13.6 eV

2층:
  위치 에너지 = -6.8 eV
  운동 에너지 = +3.4 eV
  전체        = -3.4 eV
```
- 바깥층 전자는 느긋하지만 높은 곳에 있어서 에너지가 높다
- 1층 전자는 빠르지만 깊은 곳에 있어서 에너지가 낮다
