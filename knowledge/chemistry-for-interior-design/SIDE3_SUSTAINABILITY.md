# 옆산 3: 지속가능성의 과학 — "친환경"의 분자적 진실

## 먼저 솔직해지자

"친환경 플라스틱"이라는 말은 마케팅이야. 분자 수준에서 보면 훨씬 복잡해.

```
플라스틱의 일생:

원료 → 중합 → 컴파운딩 → 성형 → 사용 → 폐기
 ↑                                           ↓
 어디서 왔나?                           어디로 가나?
 (석유? 식물? 재활용?)            (매립? 소각? 재활용? 분해?)
```

이 두 질문이 지속가능성의 전부야.

---

## Part 1: 어디서 왔나 — 원료의 기원

### 석유 기반 (Fossil-based)
```
석유 → 나프타 → 에틸렌/프로필렌 → 중합 → PE/PP

땅속에 묻혀 있던 탄소를 꺼내서 쓰는 것
→ 새로운 탄소를 탄소 순환에 추가 = 탄소 배출
```

### 바이오 기반 (Bio-based)
```
식물 → 당/전분/셀룰로스 → 화학 변환 → 모노머 → 중합

예시:
  사탕수수 → 에탄올 → 에틸렌 → 바이오 PE
  옥수수 전분 → 젖산 → PLA (폴리유산)
  피마자유 → 세바스산 → 바이오 PA

식물이 광합성으로 CO₂를 흡수한 탄소 사용
→ 이론적으로 탄소 중립(carbon neutral)
```

### 분자 수준에서 차이가 있나?

**바이오 PE와 석유 PE는 분자 구조가 완전히 동일**. 같은 [-CH₂-CH₂-]ₙ 사슬.

```
구분이 중요해:

바이오 기반(bio-based) = 원료가 식물에서 옴
생분해성(biodegradable) = 미생물이 분해할 수 있음

이 두 개는 별개야!

바이오 PE:     바이오 기반 O, 생분해 X
PLA:           바이오 기반 O, 생분해 O (조건부)
PBAT:          석유 기반 O, 생분해 O
기존 PP:       석유 기반 O, 생분해 X
```

## Part 2: 어디로 가나 — 폐기의 과학

### 매립 (Landfill)

```
PP를 땅에 묻으면?
→ 수백~수천 년 동안 거의 변하지 않음

왜?
  미생물 효소가 C-C 주쇄를 끊을 수 없음
  → 생분해 불가

효소가 끊을 수 있는 결합:
  에스테르 결합(-COO-): ★★★ → PLA 분해 가능
  아미드 결합(-CONH-): ★☆☆ → 매우 느림
  C-C 주쇄(PP/PE):      ☆☆☆ → 사실상 불가
```

### 소각 (Incineration)

```
PP를 태우면?
→ C + O₂ → CO₂
→ H + O₂ → H₂O
→ 열 에너지 방출 (PP: ~46 MJ/kg)

장점: 에너지 회수 가능
단점: CO₂ 배출, 소재 가치 소멸
```

### 기계적 재활용 (Mechanical Recycling)

```
폐플라스틱 수거 → 분류 → 분쇄 → 세척 → 재용융 → 펠릿화

분자 수준에서 무슨 일이?

① 분자량 저하
   재가공(재용융) 시 열 + 산소 + 전단
   → 사슬 절단 → 분자량↓ → 물성↓

② 오염
   다른 수지 혼입 (PP 속 PE 0.5%만 혼입돼도 충격강도 30%↓)

③ 첨가제 누적/변질
   여러 출처의 다른 첨가제가 혼합 → 예측 불가능
```

**자동차 적용의 현실:**
- PCR PP + 버진 PP 블렌딩 (20~30%)
- 블랙/다크 색상 위주
- 비가시 부품부터 적용

### 화학적 재활용 (Chemical Recycling)

```
열분해(Pyrolysis):
  폐플라스틱 → 400~600°C 무산소 가열
  → 고분자 사슬이 랜덤하게 끊어짐
  → 짧은 사슬 = 오일/왁스/가스
  → 이 오일로 새 모노머 생산 → 새 폴리머

분자 수준:
  [-CH₂-CH(CH₃)-]₁₀₀₀₀  (긴 PP 사슬)
      ↓ 400~600°C
  CH₂=CH-CH₃ + CH₃-CH₂-CH₃ + ... (모노머급)
      ↓ 정제 + 재중합
  [-CH₂-CH(CH₃)-]₁₀₀₀₀  (새 PP, 버진과 동일)

핵심: 사슬을 모노머까지 되돌려서 처음부터 다시 만듦
→ 버진과 동일한 품질
→ 무한 반복 가능 (이론적으로)
```

### 생분해 (Biodegradation)

```
미생물 효소가 에스테르 결합을 가수분해
→ 사슬을 잘게 자름 → 올리고머 → 모노머
→ 모노머를 미생물이 대사
→ CO₂ + H₂O + 바이오매스

핵심: 효소가 인식할 수 있는 결합이 있어야 함
```

## Part 3: 주요 "친환경" 소재의 분자적 실체

### PLA (Polylactic Acid, 폴리유산)

```
옥수수 전분 → 발효 → 젖산 → 중합 → PLA

구조: [-O-CH(CH₃)-CO-]ₙ
  에스테르 결합이 주쇄에 있음 → 가수분해 가능

물성:
  인장강도: ~60 MPa (PP보다 높음)
  HDT: ~55°C ← 치명적 약점 (여름 대시보드 불가)
  충격강도: 낮음 (취성)

생분해 조건:
  산업용 퇴비화 시설: 58°C + 습도 + 미생물 → 3~6개월
  일반 토양/해양: 수년~수십 년 → 사실상 안 분해됨
```

### PHA (Polyhydroxyalkanoate)

```
미생물이 직접 만드는 폴리에스테르

장점:
  진짜 해양 생분해 가능 (30°C 해수에서 수개월)
  바이오 기반 + 생분해 둘 다 충족

단점:
  가격: PP의 5~10배
  물성이 PP 대비 전반적으로 낮음

현재 자동차에는 거의 미적용
```

### 바이오 PA (Bio-based Polyamide)

```
피마자유 → 세바스산 → 바이오 PA610, PA11

분자 구조: 기존 나일론과 동일한 아미드 결합
물성: 석유 기반 PA와 동등
바이오 함량: 40~60%

자동차 적용: 연료 라인, 냉각 호스, 커넥터
```

### 재활용 PET → 자동차 내장재

```
페트병(PET) → 분쇄 → 세척 → 섬유화 → 부직포

자동차에서 가장 성공적인 재활용 사례

적용: 천장재, 트렁크 트림, 바닥 카펫
현대차: 차량 1대당 PET 병 약 60~80개 분량
```

## Part 4: 진짜 지속가능한 의사결정

### LCA (Life Cycle Assessment) 관점

```
PLA vs PP 비교 예시:

원료 단계:
  PLA: 옥수수 재배 (물, 비료, 농약, 토지)
  PP:  석유 채굴/정제

폐기 단계:
  PLA: 산업 퇴비화 시설 필요 (없으면 매립 = PP와 같음)
  PP:  기계적 재활용 가능 (인프라 존재)

결론이 단순하지 않아:
  PLA가 항상 PP보다 친환경이라고 할 수 없음
  재활용 인프라, 사용 지역, 폐기 방법에 따라 달라짐
```

### Design for Recycling — 분자를 위한 설계

```
재활용을 어렵게 만드는 것들:

① 다중 소재 구조
   표피(PVC) + 폼(PU) + 기재(PP) = 분리 불가
   → 해결: 단일 소재 구조 (all-PP, all-TPO)

② 도장
   도막이 PP와 다른 수지 → 재활용 시 이물질
   → 해결: 무도장 (착색 컴파운드, IMD)

③ 금속 인서트
   → 해결: 분리 용이 설계

④ 접착제
   열경화성 접착제는 재용융 불가
   → 해결: 기계적 체결(스냅핏) 우선
```

## 핵심 정리

```
"친환경"을 분자 수준으로 번역하면:

바이오 기반 = 탄소의 출처가 식물 (분자 구조는 같을 수 있음)
생분해성   = 효소가 인식할 수 있는 결합이 주쇄에 있음
재활용     = 사슬을 다시 녹이거나 모노머로 되돌리거나

PP/PE가 안 썩는 이유: C-C 주쇄를 끊는 효소가 없음
PLA가 썩는 이유: 에스테르 결합을 끊는 효소가 있음
재활용 물성이 떨어지는 이유: 재가공 시 사슬이 짧아짐
화학적 재활용이 답인 이유: 모노머까지 되돌리면 사슬 길이 리셋
```

---

*"당장은 PCR PP + rPET 섬유가 현실적 답. 설계 단계에서 재활용을 고려하는 게 가장 효과적."*
