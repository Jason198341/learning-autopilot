# DAY 8: 시험과 규격 — 그 숫자가 분자 수준에서 뭘 의미하는가

## 시험은 분자에게 질문하는 것이다

시험 규격서를 보면 숫자가 나와. 인장강도 25MPa, HDT 110°C, FOG 값 2mg.

이 숫자들이 뭘 뜻하는지 분자 수준에서 이해하면, 시험 불합격이 났을 때 원인을 바로 짚을 수 있어. "수치가 낮다"가 아니라 **"어떤 결합이 어떤 조건에서 깨졌다"**로 읽히게 되는 거야.

---

## 1. 기계적 시험 — 결합을 끊어보는 것

### 인장시험 (Tensile Test) — ISO 527 / ASTM D638

```
시편 양쪽을 잡고 당김
→ 처음에는 사슬이 펴짐 (탄성 영역)
→ 더 당기면 사슬이 미끄러지기 시작 (항복점)
→ 더 당기면 사슬이 배향(정렬)됨 (연신)
→ 최종적으로 사슬 간 결합이 끊어짐 (파단)
```

| 측정값 | 의미 | 분자 수준 |
|--------|------|----------|
| 인장강도 | 파단 시점의 최대 응력 | 사슬 간 반데르발스력의 총합 |
| 신율 | 얼마나 늘어나다 끊어지는가 | EPDM이 에너지 흡수하며 늘어남 |
| 탄성률 | 초기 기울기 (저항력) | 탈크/GF가 하중 분담 |

**PP 신율 변화**: 순수 PP(200~400%) → PP+TD20(20~50%). 탈크 입자가 사슬 이동을 물리적으로 막음.

### 충격시험 (Impact Test) — ISO 179/180

```
노치(V홈)가 있는 시편에 해머를 내리침
→ 노치 끝에 응력 집중
→ 크랙이 시작됨 → 전파됨 → 깨짐

측정하는 것: 깨지는 데 소모된 에너지 (kJ/m²)
```

**크랙 전파** = 사슬이 끊어지거나 빠져나오는 과정

EPDM 고무 입자 → 크랙 끝에서 변형되며 에너지 흡수 → 충격강도↑

**저온 충격이 중요한 이유**: Tg 근처에서 사슬 움직임이 얼어붙음 → 에너지 흡수 불가 → 유리처럼 깨짐(brittle fracture)

### 굴곡시험 (Flexural Test) — ISO 178

```
시편 양 끝을 받치고 가운데를 누름 (3점 굴곡)
→ 윗면: 압축 응력
→ 아랫면: 인장 응력
```

- **굴곡탄성률**: 대시보드가 처지지 않는 능력
- **굴곡강도**: 부러지는 시점

PP+TD20: ~2,500~3,000 MPa vs 순수 PP: ~1,300 MPa (탈크가 2배 상승)

## 2. 열적 시험 — 분자의 움직임 한계를 측정

### HDT (Heat Deflection Temperature) — ISO 75

```
시편에 일정 하중을 걸고 온도를 올림
→ 어느 온도에서 0.25mm 변형이 생기는가
→ 그 온도가 HDT
```

**HDT = "이 온도 넘으면 부품이 처지기 시작한다"**

분자 수준: 온도↑ → 사슬 열운동↑ → 하중을 못 버팀 → 흐름

자동차 내장재 HDT 요구: 90~120°C (여름 대시보드 표면 80~100°C)

### CLTE (열팽창계수) — ISO 11359

```
온도를 올리면서 길이 변화를 측정
→ 단위: μm/m·°C
```

| 소재 | CLTE |
|------|------|
| PP 순수 | ~100 |
| PP+TD20 | ~50 |
| Steel | ~12 |

Day 7 열팽창 문제의 근거 데이터. 체결점 간격과 슬롯 홀 크기 계산에 사용.

## 3. 내후성 시험 — 분자의 수명을 측정

### 촉진내후성 (Xenon Arc) — ISO 4892 / SAE J2412

```
제논 램프로 태양광 스펙트럼을 재현
→ UV + 열 + 습도를 동시에 가함
→ 수백~수천 시간 조사
→ 색차(△E), 광택 변화, 크랙 발생 관찰
```

**SAE J2412**: 자동차 실내용 표준
- 2,500 kJ/m² ≈ 플로리다 5년 실내 노출 상당

**△E (색차)**: 변색 정도 수치화
- △E < 1.0: 사람 눈으로 구분 불가
- 자동차 기준: △E < 3.0 (3,000시간 후)

### 내열노화 (Heat Aging) — ISO 188

```
고온 오븐에서 장시간 방치 (예: 120°C × 500시간)
→ 인장강도, 신율의 유지율 측정
```

**분자 수준**: 열에 의한 산화 열화 → 산화방지제 소진 시점 측정

**OIT(산화유도시간)**: 산화방지제 잔량의 직접 지표

## 4. 방출 시험 — 분자가 탈출하는 것을 측정

### FOG (Fogging) — ISO 6452 / SAE J1756

```
시편을 100°C로 가열
→ 휘발되는 물질이 위의 차가운 유리판에 응축
→ 응축물의 무게 또는 유리 흐림도 측정
```

**FOG 값 = 대시보드에서 날아가서 앞유리에 붙는 양**

분자 수준: 가소제, 활제, 대전방지제 같은 **저분자량 첨가제**가 탈출

분자량이 작을수록 빨리 탈출:
- TOTM (분자량 ~547): FOG 낮음
- DOP (분자량 ~391): FOG 높음

### VOC (Volatile Organic Compounds) — VDA 278

```
시편을 90°C에서 30분 가열하며 방출가스 분석 (GC-MS)
→ VOC: 끓는점 < 250°C인 물질 총량
→ FOG: 끓는점 > 250°C인 물질 총량
```

**새 차 냄새의 정체 = VOC**

원인: 잔류 모노머, 용제, 저분자 첨가제

## 5. 연소 시험 — 분자가 타는 속도를 측정

### FMVSS 302 (미국 자동차 내장재 연소 규격)

```
시편을 수평으로 놓고 한쪽 끝에 15초간 불을 붙임
→ 불을 끔
→ 불꽃이 퍼지는 속도를 측정

합격 기준: 연소속도 ≤ 102 mm/min
또는 불이 자기소화(self-extinguishing)되면 합격
```

**분자 수준**: 연소 = 열분해 → 가연성 가스 방출 → 가스 연소 → 열 발생 연쇄

난연제가 연쇄를 끊는 방식:
- 할로겐계: 가스상 라디칼 포집
- 금속수산화물: 흡열 분해 + 수증기
- 팽창형: 탄화층으로 산소 차단

## 6. 외관 시험 — 분자 배열이 빛을 어떻게 다루는가

### 광택 (Gloss) — ISO 2813

```
60° 각도로 빛을 쏘고 반사되는 빛의 양을 측정

고광택: 표면이 매끄러움 → 정반사 많음
저광택: 표면이 거칠음 → 난반사 많음
```

자동차 내장재는 저광택(matte) 선호 → 햇빛 반사 눈부심 방지

### 색차 (Color Matching) — △E (CIE Lab)

```
L* = 밝기 (0 검정 ~ 100 흰색)
a* = 빨강(+) ~ 초록(-)
b* = 노랑(+) ~ 파랑(-)

△E = √(△L² + △a² + △b²)
```

| △E | 판정 |
|----|------|
| < 1.0 | 구분 불가 → 합격 |
| 1.0~2.0 | 주의 깊게 보면 보임 |
| > 3.0 | 누구나 보임 → 불합격 |
