## 핵심 깨달음
- **도구의 한계를 안다**: 물리학 공식은 자연 그 자체가 아니라 "달을 가리키는 손가락"이다
- **불완전한 도구를 능숙하게 쓴다**: E=mc², 전자 배치, 결합 에너지 - 이 도구들로 자동차를 설계한다
- **나만의 도구를 만들 준비를 한다**: 기존 도구를 마스터한 후 혁신한다

## 철학적 기반

### 자연과 언어의 관계
```
자연의 실제 모습:
  → 하나의 거대한 움직임
  → 대칭도 비대칭도 없음
  → 그냥 있는 그대로

인간이 하는 일:
  → 그 움직임을 나눔 (물질/반물질, 음/양)
  → "대칭이다"라고 이름 붙임
  → 그 도구로 계산하고 예측하고 설계함
```

### 최소 단위의 존재 이유
```
변화가 있으려면
  → 이전과 이후가 달라야 하고
  → 다르려면 최소한의 차이가 있어야 하고
  → 그 최소한의 차이가 "양자(quantum)"

"뭔가가 있다" = "최소 단위가 있다"
이 둘은 같은 말이다.
```

## 자동차 내장재와의 연결

```
전자 배치
  → 최외각 전자의 개수와 배치
    → 결합 방식
      → 분자 구조
        → 재료의 물성 (강도, 유연성, 색상, 내열성)
          → 자동차 시트의 촉감
```
