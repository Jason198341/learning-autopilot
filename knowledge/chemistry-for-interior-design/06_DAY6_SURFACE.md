# DAY 6: 표면 처리와 촉감 — 고급차와 저가차를 나누는 차이

## 핵심 질문
같은 PP 대시보드인데 왜 어떤 차는 고급스럽고 어떤 차는 싸구려 느낌이냐?

**답은 표면이야.** 사람은 내장재의 분자 구조를 못 느껴. 눈으로 보고 손으로 만지는 **표면 0.1mm**가 전부야.

## 1. 인간이 촉감을 느끼는 원리

손가락 끝에 **기계수용체(mechanoreceptor)**라는 센서가 있어. 1cm²당 약 250개.

### 센서가 감지하는 4가지
| 요소 | 설명 |
|------|------|
| 거칠기 | 표면의 요철 크기 |
| 마찰 | 미끄러운가 걸리는가 |
| 온감 | 차가운가 따뜻한가 |
| 경도 | 딱딱한가 말랑한가 |

### 고급 vs 싸구려 느낌
```
고급 느낌:
  거칠기 = 미세하고 균일한 요철 (가죽 느낌)
  마찰 = 적당히 걸림 (손에 착 감기는 느낌)
  온감 = 따뜻함 (열전도율 낮음)
  경도 = 약간 눌림 (소프트 터치)

싸구려 느낌:
  거칠기 = 매끄럽거나 불균일 (플라스틱 느낌)
  마찰 = 미끄러움 (손에서 미끄러짐)
  온감 = 차가움 (열전도율 높음)
  경도 = 딱딱함 (하드 플라스틱)
```

## 2. 왜 차갑게 느껴지면 싸구려냐

### 열전도의 원리
- 손가락 온도: 약 33°C
- 대시보드 표면: 실내 온도 약 23°C
- 손이 만지면 → 열이 손에서 대시보드로 이동

```
열이 빨리 빠져나가면 → "차갑다" → 금속/딱딱한 플라스틱 → 싸구려
열이 천천히 빠져나가면 → "따뜻하다" → 가죽/우드 느낌 → 고급
```

### 분자 수준의 원리 (Day 3 연결)
- **금속결합**: 자유 전자가 열을 빠르게 전달 → 차가움
- **공유결합(PP)**: 자유 전자 없음 → 열 전달 느림
- **폼 층 추가**: 공기가 갇힘 → 열전도율 극도로 낮음 → 따뜻함

### 열전도율 비교 (W/m·K)
| 재료 | 열전도율 | 느낌 |
|------|---------|------|
| 알루미늄 | 237 | 매우 차가움 |
| 강철 | 50 | 차가움 |
| 딱딱한 PP | 0.22 | 약간 차가움 |
| 가죽 | 0.14 | 적당히 따뜻함 |
| PU 폼 | 0.03 | 따뜻함 |
| 공기 | 0.025 | 가장 따뜻함 |

## 3. 열 이동의 3가지 방식

### 열전도 (Conduction) — 접촉
```
손가락이 대시보드에 닿음
→ 분자끼리 부딪히면서 열이 전달됨
→ 직접 접촉해야 일어남

뜨거운 쪽 분자가 빠르게 진동
→ 옆 분자를 때림
→ 그 분자도 빠르게 진동 시작
→ 열이 전달됨
```
- 금속결합의 자유 전자가 이걸 빠르게 함
- 공유결합은 자유 전자 없어서 느림

### 대류 (Convection) — 흐름
```
히터를 틀면 따뜻한 공기가 위로 올라감
→ 찬 공기가 아래로 내려옴
→ 공기가 순환하면서 열이 퍼짐
```
- 자동차 에어컨/히터의 원리

### 복사 (Radiation) — 빛
```
태양열이 우주 진공을 지나서 지구에 옴
→ 접촉도 없고 공기도 없는데 열이 전달됨
→ 전자기파(적외선)로 에너지가 날아옴
```
- 여름 주차 차량 대시보드가 뜨거운 이유
- Day 2의 전자 에너지 흡수 원리

## 4. 표면 처리 기술

### 그레인(Grain) — 표면 무늬 찍기
```
금형 표면:
  ┌──╱╲──╱╲──╱╲──╱╲──┐
  │  미세한 가죽 무늬   │
  └──────────────────┘
         ↓ 사출 성형
플라스틱 표면:
  ╲╱──╲╱──╲╱──╲╱──╲╱
  가죽처럼 보이는 표면
```

| 무늬 깊이 | 느낌 |
|----------|------|
| 0.05mm | 매끄러움 → 모던/심플 |
| 0.15mm | 적당한 요철 → 가죽 느낌 |
| 0.30mm | 깊은 요철 → 러스틱/빈티지 |

### 소프트 페인트 — 말랑한 코팅
```
구조:
  ┌─ 소프트 페인트 (0.02mm) ← 만지는 표면
  ├─ 프라이머 (0.01mm)      ← 접착층
  └─ PP 기재               ← 본체

손가락이 누르면 페인트층이 미세하게 변형
→ "소프트 터치" 느낌
```

### 인몰드 데코레이션(IMD) — 필름 일체 성형
```
  ┌─ 투명 보호층
  ├─ 인쇄층 (나무결, 금속 느낌 등)
  ├─ 접착층
  └─ PP/ABS 기재

겉보기: 진짜 나무처럼 보임
실제: 플라스틱 + 필름
```
- 고급차 우드 트림의 실체

## 5. 소프트 터치의 구조 — 차급별 차이

### 저가차 대시보드 — 하드 타입
```
  ┌──────────────┐
  │   PP 사출물    │  ← 이게 전부. 딱딱함.
  └──────────────┘

느낌: 딱딱하고 약간 차갑고 톡톡 소리남
원가: 낮음
```

### 중급차 대시보드 — 소프트 페인트
```
  ┌─ 소프트 페인트  ← 0.02mm 말랑한 코팅
  └─ PP 사출물

느낌: 약간 부드러움. 겉만 소프트.
원가: 중간
```

### 고급차 대시보드 — 발포 일체
```
  ┌─ 스킨 (PVC/TPO)     ← 0.5~1mm 표면층
  ├─ PU 폼              ← 3~5mm 쿠션층 (핵심)
  └─ PP 기재            ← 구조물

느낌: 눌리면서 천천히 복원. 따뜻함. 고급스러움.
원가: 높음
```

**핵심**: 3~5mm PU 폼 안의 미세한 공기 방울
- 누르면 공기가 압축 → 쿠션감
- 공기가 열 전달 차단 → 따뜻함
- 천천히 복원 → 고급감
- Day 5의 열경화성 PU 폼 (여름 80°C에도 형태 유지)

## 6. 색상과 광택

### 색상 — 전자의 에너지 흡수 (Day 2 연결)
```
검정/짙은 회색:
  → 거의 모든 파장을 흡수
  → 반사광 최소 → 눈에 피로 적음
  → 대부분의 차에 사용

베이지/탄색:
  → 짧은 파장(파란빛) 흡수, 긴 파장 반사
  → 따뜻하고 밝은 느낌
  → 고급차 옵션

투톤 (상부 어두움 + 하부 밝음):
  → 상부 어두움: 유리 반사 줄임 (안전)
  → 하부 밝음: 실내가 넓어 보임
  → 프리미엄 전략
```

### 광택 — 표면의 미세 구조
```
매끄러운 표면:
  ──────────  → 거울처럼 반사
  ↗ ↗ ↗ ↗ ↗
  빛이 한 방향으로 반사 → 고광택

거친 표면:
  ╱╲╱╲╱╲╱╲  → 빛이 산란
  ↗ ↑ ↙ ↗ ↑
  빛이 여러 방향으로 흩어짐 → 무광택(매트)
```

### 광택도 기준 (Gloss Unit, 60°)
| 광택도 | 용도 |
|--------|------|
| 80 이상 (고광택) | 피아노 블랙 트림, 일부 장식 |
| 30~50 (반광택) | 도어 핸들, 스위치류 |
| 5~15 (저광택) | 대시보드, 도어트림 |
| 2 이하 (무광택) | 최상단 대시보드 (반사 방지) |

**중요**: 대시보드는 거의 저광택/무광택 → 햇빛 반사 시 운전 시야 방해 방지 (안전 설계)

## 7. 내장 설계 팀장의 판단 프로세스

```
1단계: 기재 선택
  → PP (경량, 원가) vs ABS (강도, 광택)
  → 열가소성이니까 사출 성형

2단계: 소프트 여부
  → 저가차: 하드 타입 (PP만)
  → 중급차: 소프트 페인트 또는 부분 소프트
  → 고급차: 스킨 + 폼 + 기재 (발포 일체)

3단계: 그레인 설계
  → 무늬 패턴: 가죽? 기하학? 심플?
  → 무늬 깊이: 0.05~0.30mm
  → 차급과 브랜드 아이덴티티에 맞춤

4단계: 색상 + 광택
  → 색상: 검정 / 그레이 / 베이지 / 투톤
  → 광택: 대시보드 상단은 반드시 저광택
  → CMF 디자이너와 협업

5단계: 내구성 검증
  → 내열: 80°C 500시간 → 변형/변색 없어야 함
  → 내광: 자외선 조사 → 색 바래지면 안 됨
  → 내마모: 손톱으로 긁힘 테스트
  → 내약품: 자외선 차단제, 손소독제에 견뎌야 함
```

## Day 6 정리

```
표면이 결정하는 것:
  촉감 → 거칠기 + 마찰 + 온감 + 경도
  시각 → 색상 + 광택 + 무늬

고급차 vs 저가차:
  고급: 스킨+폼+기재, 저광택, 소프트터치, 따뜻함
  저가: PP 사출만, 딱딱함, 약간 차가움

핵심 원리:
  따뜻함 = 열전도율 낮음 = 공기층(폼)
  소프트 = PU 폼의 공기 방울
  무광택 = 표면 요철이 빛을 산란
  색상 = 전자의 에너지 흡수/반사
```

*"사람은 분자 구조를 못 느낀다. 표면 0.1mm가 전부다. 그 0.1mm를 어떻게 설계하느냐가 고급차와 저가차를 나눈다."*
