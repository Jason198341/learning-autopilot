# DAY 9: 불량의 과학 — 모든 불량은 분자가 보내는 신호다

## 불량을 두 종류로 나눠

- **가공 불량**: 사출 성형 과정에서 생김. 금형에서 부품이 나오는 순간 이미 문제.
- **필드 불량**: 차가 고객에게 간 뒤 시간이 지나면서 생김. 변색, 크랙, 소음.

둘 다 원인은 분자 거동이야.

---

## Part 1: 사출 성형 불량

### 웰드라인 (Weld Line)

```
용융 수지가 금형 안에서 흐르다가
→ 장애물(구멍, 코어핀)을 만나면 둘로 갈라짐
→ 다시 만남
→ 만나는 지점에 선이 생김 = 웰드라인
```

**분자 수준**: 두 흐름이 만날 때 사슬들이 서로 충분히 침투(interdiffusion)해야 강한 결합. 만나는 시점에 온도가 떨어지면 사슬 움직임이 둔해져 침투 불충분 → 약한 경계면.

**강도 문제**: 웰드라인 부위 인장강도 = 원래의 50~80%
- GF 소재: 섬유가 웰드라인에서 흐름에 직각으로 놓여서 보강 효과 사라짐

**해결**:
1. 수지 온도↑ → 만나는 시점에도 사슬이 움직임
2. 금형 온도↑ → 냉각 속도↓
3. 사출 속도↑ → 합류 시점 온도 유지
4. 게이트 위치 변경 → 웰드라인을 비가시 영역으로
5. 오버플로우 웰 → 차가운 수지를 밖으로

### 싱크마크 (Sink Mark)

```
두꺼운 부위가 냉각될 때
→ 표면이 먼저 굳음 (스킨층)
→ 내부가 나중에 굳으면서 수축
→ 내부가 표면을 안쪽으로 끌어당김
→ 표면에 움푹 파인 자국
```

**분자 수준**: 용융→고화 시 사슬 간 거리 줄어듦(수축). PP 수축률 1.5~2.0%는 결정화 때문.

**해결**:
1. 보압(packing pressure)↑
2. 보압 시간↑
3. 냉각 시간↑
4. 벽 두께 균일화 (리브 두께 = 벽 두께 × 60~70%)
5. 탈크/GF 추가 → 수축률↓ (PP 1.5% → PP+TD20 0.8%)

### 플래시 (Flash / Burr)

```
금형 파팅라인(PL)의 틈새로 수지가 새어나옴
→ 얇은 막 형태의 돌출
```

**분자 수준**: 용융 수지 점도가 핵심. MFI 높으면 작은 틈으로도 침투. 고압에서 전단담화(shear thinning)로 점도 더 낮아짐.

**해결**:
1. 사출압/보압↓
2. 형체력(clamping force)↑
3. 금형 PL 정비
4. 수지 온도↓ (미충전 주의)

### 미성형 / 숏샷 (Short Shot)

```
수지가 금형 끝까지 못 채움
→ 부품 일부가 빠져 있음
```

**분자 수준**: 벽면에서 냉각되며 고화층(frozen layer) 형성 → 유로 좁아짐 → 막힘

**해결**:
1. 수지 온도↑
2. 사출 속도↑
3. 금형 온도↑
4. 벽 두께↑ (설계 변경)
5. 게이트 크기↑ 또는 추가
6. 벤트(vent) 확인

### 휨/뒤틀림 (Warpage)

```
부품이 금형에서 나온 후 휘어짐
```

**원인**:
1. **불균일 냉각**: 코어/캐비티 쪽 냉각 속도 차이
2. **섬유 배향**: GF 소재의 수축 이방성 (흐름 방향 0.3% vs 직각 0.8%)
3. **잔류 응력**: 사슬이 강제 배향된 채 굳음

**해결**:
1. 냉각 균일화 (금형 수로 최적화)
2. 게이트 위치/개수 최적화
3. 보압 프로파일 최적화
4. 탈크 사용 (GF 대신, 이방성 적음)
5. 지그 냉각

### 은줄 (Silver Streaks)

```
부품 표면에 은색 줄무늬
→ 수지 흐름 방향으로 나타남
```

**분자 수준**: 수지 안 수분/가스가 고온에서 기화 → 기포가 늘어지며 표면 아래 얇은 기포층 형성 → 빛 산란 → 은색

**주된 원인**: 원료 건조 불량

**해결**:
1. 원료 충분 건조 (PA: 80°C/4h, PC: 120°C/4h)
2. 수지 온도 확인 (과열 → 열분해 가스)
3. 스크류 배압↑
4. 사출 속도↓

### 젯팅 (Jetting)

```
게이트에서 수지가 뱀처럼 구불구불하게 들어감
→ 표면에 꼬불꼬불한 자국
```

**분자 수준**: 게이트 작고 사출 속도 빠르면 수지가 제트처럼 분사 → 불규칙하게 접힘 → 자국

**해결**:
1. 게이트 크기↑
2. 사출 속도 초기↓
3. 게이트 방향 변경 (벽면에 부딪히도록)

## Part 2: 필드 불량 — 시간이 만드는 불량

### 변색 (Discoloration)

**원인 3가지**:
1. **UV 열화**: 결합 절단 → 산화 → 카르보닐기 → 황변
2. **열 산화**: 산화방지제 소진 후 급격 열화
3. **오염물 산화**: 표면 기름/먼지가 열에 의해 탄화

**진단법**:
- 변색 부위가 직사광선 패턴과 일치? → UV 열화
- 전체적으로 균일한 황변? → 열 산화
- 국부적 갈색 자국? → 오염물
- FTIR 분석에서 카르보닐 피크(1720 cm⁻¹) 확인

### 크랙 (Stress Cracking)

**환경응력균열(ESC)**: 눈에 보이는 응력 없이 크랙 발생

**분자 수준**: 잔류 응력 + 화학 물질(세정제, 방향제, 핸드크림) 접촉 → 화학 물질이 사슬 사이 침투 → 분자간 힘 약화 → 크랙

- PC(폴리카보네이트): 알코올 세정제에 취약
- ABS: 응력 상태에서 특정 용제에 크레이징

**해결**:
1. 잔류 응력 최소화 (어닐링, 가공 조건 최적화)
2. ESC 저항성 높은 소재 등급
3. OEM 승인 세정제만 사용
4. 설계에서 응력 집중부 제거 (R값 확보)

### 백화 (Whitening / Stress Whitening)

```
부품을 구부리거나 충격을 받은 부위가 하얗게 변함
→ 특히 PP에서 흔함
```

**분자 수준**: 응력 부위에 미세한 크레이즈/보이드 형성 → 빛 산란 → 하얗게 보임

EPDM 함량↑ → 크레이즈 형성 억제 → 백화↓

### 블루밍/블리딩 (Blooming/Bleeding)

```
표면에 하얀 가루나 끈적한 막이 형성
→ 시간이 지날수록 심해짐
```

**분자 수준**: 저분자량 첨가제(활제, 대전방지제, UV 안정제)가 표면으로 확산
- 블루밍: 표면에서 결정화 (가루)
- 블리딩: 액상으로 남음 (끈적)

**해결**:
1. 첨가제 함량 최적화 (과량 금지)
2. 고분자량 첨가제로 전환
3. 첨가제-수지 상용성 확인

### BSR (Buzz, Squeak, Rattle)

**Squeak(찍찍)**:
```
두 표면이 미끄러질 때 stick-slip 현상
→ 붙었다 미끄러졌다 반복 → 진동
→ 가청 주파수면 찍찍 소리
```

**분자 수준 원인**:
- 두 고분자 표면 에너지가 비슷하면 접착력↑ → stick 강화 → 소음↑
- 온도 변화로 부품 팽창/수축 → 미끄러짐
- 습도 변화로 PA 부품 팽윤 → 간극 변화

**해결**:
1. 재질 조합 변경
2. 표면 텍스처 변경 (접촉 면적↓)
3. 그리스/펠트 테이프 적용
4. 설계 공차 최적화

## 트러블슈팅 플로우차트

```
불량 발생
    │
    ├─ 성형 직후 발견?
    │   ├─ YES → 가공 불량
    │   │   ├─ 표면에 선이 있다 → 웰드라인 or 젯팅
    │   │   ├─ 표면이 움푹 파였다 → 싱크마크
    │   │   ├─ 가장자리에 막이 있다 → 플래시
    │   │   ├─ 일부가 안 채워졌다 → 숏샷
    │   │   ├─ 은색 줄무늬 → 은줄 (수분/가스)
    │   │   └─ 전체가 휘었다 → 워피지
    │   │
    │   └─ 1차 점검: 수지온도, 금형온도, 사출속도, 보압
    │
    └─ 시간 지나서 발견?
        ├─ YES → 필드 불량
        │   ├─ 색이 변했다 → 변색 (UV/열산화)
        │   ├─ 갈라졌다 → 크랙 (ESC/크리프)
        │   ├─ 하얗게 됐다 → 백화 (크레이즈)
        │   ├─ 표면에 가루/끈적 → 블루밍/블리딩
        │   └─ 소리가 난다 → BSR
        │
        └─ 1차 점검: 노출 환경, 첨가제 처방, 설계(응력집중)
```

## 핵심 원칙

```
가공 불량의 80%는 4가지로 해결된다:
→ 수지 온도, 금형 온도, 사출 속도, 보압

필드 불량의 80%는 3가지로 예방된다:
→ 첨가제 최적화, 잔류 응력 최소화, 소재-환경 적합성
```

모든 불량은 분자가 **"나 여기서 이렇게 못 있어"**라고 보내는 신호야.

---
