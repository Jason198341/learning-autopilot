# DAY 4: 고분자(Polymer) — 자동차 내장재의 진짜 정체

## 고분자란?

**poly = 많은, mer = 단위**

작은 분자(단량체, monomer)가 반복적으로 연결된 긴 사슬.

```
단량체 (클립 1개):
  [─]

고분자 (클립 수천 개):
  [─][─][─][─][─][─][─][─][─][─][─][─]...

  이걸 수천~수만 번 반복
```

자동차 내장재의 80%가 고분자.

## 결정성 (Crystallinity) — 같은 성분, 다른 물성

### 국수 비유
```
배열 1: 국수를 아무렇게나 던져놓음 (비정질)

  ~~~~  ~~~
    ~~~~  ~~
  ~~~  ~~~~

  → 엉켜있지만 잡아당기면 풀려
  → 부드럽고 유연함


배열 2: 국수를 가지런히 정렬 (결정질)

  ─────────────
  ─────────────
  ─────────────

  → 서로 밀착되어 단단함
  → 뻣뻣하고 강함
```

| 결정성 | 특성 | 예시 |
|--------|------|------|
| 낮음 (비정질) | 부드럽고 투명함 | 랩 필름 |
| 높음 (결정질) | 단단하고 불투명함 | 우유통 |

## 폴리에틸렌(PE) — 가장 단순한 고분자

```
  H H H H H H H H
  │ │ │ │ │ │ │ │
─ C─C─C─C─C─C─C─C ─
  │ │ │ │ │ │ │ │
  H H H H H H H H
```

### LDPE (저밀도 폴리에틸렌)
```
주 사슬에서 가지(branch)가 많이 나옴:

─── C─C─C─C─C─C─C ───
        │       │
        C─C─C   C─C

가지 때문에 사슬이 정렬을 못 함 → 부드럽고 투명
```
- 용도: 비닐봉지, 랩 필름

### HDPE (고밀도 폴리에틸렌)
```
가지가 거의 없는 직선 사슬:

─── C─C─C─C─C─C─C─C─C─C ───
─── C─C─C─C─C─C─C─C─C─C ───

직선이라 사슬끼리 밀착 정렬됨 → 단단하고 불투명
```
- 용도: 세제통, 자동차 연료탱크

**화학 성분이 완전히 같은데 가지 유무 하나로 비닐봉지 vs 연료탱크**

## 폴리프로필렌(PP) — 대시보드의 재료

```
  H  CH₃ H  CH₃ H  CH₃
  │  │   │  │   │  │
─ C──C── C──C── C──C ─
  │  │   │  │   │  │
  H  H   H  H   H  H

PE에서 수소 하나를 CH₃로 바꾼 것뿐.
CH₃가 어느 쪽에 붙느냐에 따라 3종류.
```

### Isotactic (같은 쪽에 붙음)
```
  CH₃  CH₃  CH₃  CH₃
   │    │    │    │
── C────C────C────C──
   │    │    │    │
   H    H    H    H

→ 정렬 잘 됨 → 결정성 높음 → 단단함
→ 자동차 대시보드에 사용
```

### Syndiotactic (번갈아 붙음)
```
  CH₃   H   CH₃   H
   │    │    │    │
── C────C────C────C──
   │    │    │    │
   H   CH₃   H   CH₃

→ 정렬 됨 → 중간 강도
```

### Atactic (랜덤으로 붙음)
```
  CH₃   H    H   CH₃
   │    │    │    │
── C────C────C────C──
   │    │    │    │
   H   CH₃  CH₃   H

→ 정렬 안 됨 → 부드러움 → 접착제, 씰링재
```

## ABS — 자동차 내장재의 왕

**A**crylonitrile + **B**utadiene + **S**tyrene

```
A = 아크릴로니트릴 → 내열성 + 내화학성 (C≡N 삼중결합)
B = 부타디엔       → 충격 흡수 (C=C 이중결합, 고무 성분)
S = 스티렌         → 강도 + 광택 + 가공성 (벤젠 고리)
```

### ABS 구조
```
  S─S─S─S─S─S─S─S─S    ← 스티렌 사슬 (뼈대, 강도)
       │          │
       A─A─A─A   A─A    ← 아크릴로니트릴 (내열성)
       │
    (B─B─B─B─B)          ← 부타디엔 고무 입자
                            (충격 흡수 쿠션)
```

개별로는 부족:
- A만: 단단한데 깨지기 쉬움
- B만: 탄성 있는데 흐물흐물
- S만: 광택 나는데 충격에 약함

세 개를 합치면 각각의 약점을 보완 → 단단하면서 충격에 강함

## 고분자의 물성을 결정하는 4가지

| 요소 | 영향 | 예시 |
|------|------|------|
| 1. 사슬의 성분 | PE, PP, ABS 등의 차이 | 단량체 종류 |
| 2. 사슬의 배열 | 결정성 (정렬 vs 엉킴) | LDPE vs HDPE |
| 3. 가지의 유무 | 가지 많으면 부드러움 | LDPE vs HDPE |
| 4. 가교의 유무 | 가교 있으면 탄성, 내열성↑ | 타이어 가황 |

## 자동차 내장재별 고분자

| 부품 | 소재 | 특징 |
|------|------|------|
| 대시보드 | PP (isotactic) | 정렬됨, 단단함 |
| 시트 쿠션 | PU 폼 | 가스 방울 포함, 푹신함 |
| 센터콘솔 | ABS | 3가지 혼합, 강하고 안 깨짐 |
| 시트벨트 | 나일론 | 사슬 정렬, 인장 강도 높음 |
| 카펫 | PET 섬유 | 내마모성 |
| 연료탱크 | HDPE | 직선 사슬, 내화학성 |
