## 1. 빛의 단위 완전 정복

### 1.1 왜 빛의 단위가 여러 개인가요?

**핵심 개념: 빛을 측정하는 관점이 다르기 때문!**

```
┌─────────────────────────────────────────────────────────────────────┐
│                     빛의 4가지 측정 관점                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [1] 광원 자체가 얼마나 밝은가?                                      │
│      → 광속 (Luminous Flux) = 루멘 (lm)                             │
│      → "전구가 총 얼마나 빛을 내는가"                                │
│                                                                     │
│  [2] 특정 방향으로 얼마나 밝은가?                                    │
│      → 광도 (Luminous Intensity) = 칸델라 (cd)                      │
│      → "특정 방향으로 얼마나 강하게 비추는가"                        │
│                                                                     │
│  [3] 표면이 얼마나 밝게 보이는가?                                    │
│      → 휘도 (Luminance) = cd/m²                                     │
│      → "눈에 보이는 밝기" (무드램프의 핵심!)                         │
│                                                                     │
│  [4] 표면에 얼마나 많은 빛이 닿는가?                                 │
│      → 조도 (Illuminance) = 럭스 (lux)                              │
│      → "책상 위에 얼마나 밝은가"                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 1.2 단위별 상세 설명

**① 루멘 (lm) - 광속**

```
┌─────────────────────────────────────────────────────────────────────┐
│                          루멘 (lm)                                   │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  정의: 광원이 1초 동안 내보내는 총 빛의 양                            │
│                                                                     │
│  비유: 수도꼭지에서 나오는 "총 물의 양"                               │
│        꼭지를 틀면 물이 사방으로 퍼지는데, 그 전체 양                  │
│                                                                     │
│  [일상 예시]                                                         │
│  ┌────────────────────────────────────────────────────────┐        │
│  │ 광원               │ 루멘 (lm)  │ 비고                  │        │
│  ├────────────────────────────────────────────────────────┤        │
│  │ 촛불 1개            │ ~12 lm    │ 은은한 분위기         │        │
│  │ 스마트폰 플래시      │ ~50 lm    │ 손전등 대용           │        │
│  │ 40W 백열등 (옛날)   │ ~450 lm   │ 침실 조명             │        │
│  │ 60W LED 전구        │ ~800 lm   │ 거실 스탠드           │        │
│  │ 100W 백열등 대체    │ ~1600 lm  │ 밝은 실내             │        │
│  │ 자동차 헤드라이트    │ ~3000 lm  │ 전방 조명             │        │
│  │ 무드램프 LED 1개    │ 1~20 lm   │ 은은하게!             │        │
│  └────────────────────────────────────────────────────────┘        │
│                                                                     │
│  💡 무드램프는 "밝게 비추는 게 목적이 아님"                           │
│     → 그래서 루멘이 낮아도 OK (1~20 lm 정도)                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**② 칸델라 (cd) - 광도**

```
┌─────────────────────────────────────────────────────────────────────┐
│                         칸델라 (cd)                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  정의: 특정 방향으로 방출되는 빛의 세기                               │
│                                                                     │
│  비유: 호스로 물 뿌릴 때 "특정 방향의 수압"                           │
│        넓게 뿌리면 약하고, 한 점에 집중하면 강함                      │
│                                                                     │
│  [어원] Candela = 라틴어로 "양초"                                    │
│         양초 1개의 특정 방향 밝기를 1 cd로 정의 (역사적 유래)         │
│                                                                     │
│  ┌────────────────────────────────────────────────────────┐        │
│  │         같은 루멘이라도 칸델라는 다를 수 있음!           │        │
│  │                                                        │        │
│  │   [넓게 퍼지는 전구]        [집중하는 스포트라이트]      │        │
│  │         ( )                       │                   │        │
│  │        /│\                        │                   │        │
│  │       / │ \                       │                   │        │
│  │      ↙  ↓  ↘                      ↓                   │        │
│  │    낮은 cd                     높은 cd                 │        │
│  │    (넓게 분산)               (한 방향 집중)             │        │
│  │                                                        │        │
│  │    무드램프: 넓게 퍼지는 타입 선호 (눈부심 방지)         │        │
│  └────────────────────────────────────────────────────────┘        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**③ cd/m² (칸델라 퍼 제곱미터) - 휘도** ⭐ 무드램프 핵심!

```
┌─────────────────────────────────────────────────────────────────────┐
│                    휘도 (cd/m²) - 가장 중요!                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  정의: 단위 면적당 눈에 보이는 밝기                                   │
│        = "그 표면이 얼마나 밝게 보이는가"                             │
│                                                                     │
│  비유: TV 화면, 스마트폰 화면 밝기                                    │
│        "이 화면이 눈에 얼마나 밝게 느껴지는가"                        │
│                                                                     │
│  [왜 무드램프에서 휘도가 핵심인가?]                                   │
│  ─────────────────────────────────────                              │
│  무드램프 = 도광체(라이트 가이드)가 "발광하는 표면"                    │
│  → 그 표면이 눈에 얼마나 밝게 보이는지가 중요!                        │
│  → 너무 밝으면 눈부심, 너무 어두우면 안 보임                          │
│                                                                     │
│  [휘도 기준값 비교]                                                   │
│  ┌────────────────────────────────────────────────────────┐        │
│  │ 대상                   │ 휘도 (cd/m²)   │ 체감        │        │
│  ├────────────────────────────────────────────────────────┤        │
│  │ 맑은 하늘              │ ~10,000       │ 눈부심!     │        │
│  │ 맑은 날 흰 종이        │ ~25,000       │ 매우 밝음   │        │
│  │ 형광등 표면            │ ~7,000        │ 직시 힘듦   │        │
│  │ PC 모니터 (최대)       │ 300~500       │ 밝음       │        │
│  │ 스마트폰 화면          │ 400~1000      │ 밝음       │        │
│  │ 촛불 불꽃              │ ~7,000        │ 밝지만 작음 │        │
│  │ ─────────────────────────────────────────────────────  │        │
│  │ ★ 자동차 무드램프       │ 10~50        │ 은은함!     │        │
│  │ ─────────────────────────────────────────────────────  │        │
│  │ 달 표면 (보름달)        │ ~2,500       │ 야간 가시   │        │
│  │ 별빛                   │ ~0.001       │ 거의 안 보임 │        │
│  └────────────────────────────────────────────────────────┘        │
│                                                                     │
│  ★ 무드램프 설계 기준: 10~50 cd/m²                                   │
│                                                                     │
│  ┌────────────────────────────────────────────────────────┐        │
│  │  왜 10~50 cd/m² 인가?                                   │        │
│  ├────────────────────────────────────────────────────────┤        │
│  │                                                        │        │
│  │  < 10 cd/m²  : 야간에도 거의 안 보임 → 무드등 의미 없음 │        │
│  │  10~30 cd/m² : 은은한 분위기 조명 → 일반 무드램프       │        │
│  │  30~50 cd/m² : 약간 밝은 조명 → 스위치 위치 확인용     │        │
│  │  > 50 cd/m²  : 야간 운전 시 눈부심 유발 → 사용 안 함   │        │
│  │                                                        │        │
│  │  💡 야간 운전 시 눈의 순응 상태 고려                     │        │
│  │     (어둠에 적응된 눈은 밝은 빛에 민감)                  │        │
│  └────────────────────────────────────────────────────────┘        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

**④ 럭스 (lux) - 조도**

```
┌─────────────────────────────────────────────────────────────────────┐
│                          럭스 (lux)                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  정의: 1㎡ 면적에 1루멘의 빛이 도달할 때 = 1 lux                      │
│        = "표면에 얼마나 많은 빛이 닿는가"                             │
│                                                                     │
│  공식: lux = lm / m²                                                 │
│                                                                     │
│  비유: 정원에 스프링클러로 물 뿌릴 때                                 │
│        "땅 1㎡에 얼마나 많은 물이 닿는가"                             │
│                                                                     │
│  [일상 조도 예시]                                                     │
│  ┌────────────────────────────────────────────────────────┐        │
│  │ 환경                     │ 조도 (lux)     │ 비고       │        │
│  ├────────────────────────────────────────────────────────┤        │
│  │ 맑은 날 한낮 야외         │ 100,000       │ 매우 밝음   │        │
│  │ 흐린 날 야외              │ 10,000        │ 밝음       │        │
│  │ 사무실 조명               │ 300~500       │ 업무 적합   │        │
│  │ 거실 조명                 │ 150~300       │ 편안함      │        │
│  │ 복도/화장실               │ 100~200       │ 기본 시야   │        │
│  │ 영화관                    │ 1~5           │ 어두움      │        │
│  │ 보름달 야외               │ 0.1~0.3       │ 야간       │        │
│  │ 별빛만 있는 야외          │ 0.0001        │ 매우 어두움 │        │
│  └────────────────────────────────────────────────────────┘        │
│                                                                     │
│  ❓ 무드램프에서 조도(lux)는 왜 잘 안 쓰나요?                         │
│  ─────────────────────────────────────────                          │
│  무드램프는 "비추는 조명"이 아니라 "보이는 조명"                       │
│  → 표면에 빛이 얼마나 닿는지(조도)보다                                │
│  → 눈에 얼마나 밝게 보이는지(휘도)가 더 중요!                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 2. 휘도 (Luminance) - 밝기의 핵심

### 2.1 휘도 설계의 중요성

```
┌─────────────────────────────────────────────────────────────────────┐
│                  무드램프 휘도 설계 기준                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [위치별 권장 휘도]                                                   │
│                                                                     │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ 위치               │ 권장 휘도    │ 이유                     │    │
│  ├────────────────────────────────────────────────────────────┤    │
│  │ IP 상단 라인        │ 15~30 cd/m² │ 간접 조명, 주시각 밖     │    │
│  │ 도어 트림 라인      │ 20~40 cd/m² │ 도어 포켓 위치 확인      │    │
│  │ 센터 콘솔 라인      │ 10~25 cd/m² │ 운전 방해 방지          │    │
│  │ 풋웰 (발밑)         │ 5~15 cd/m²  │ 승하차 시만 필요        │    │
│  │ 도어 핸들 조명      │ 30~50 cd/m² │ 핸들 위치 빠른 파악     │    │
│  │ 글로브 박스 내부    │ 20~40 cd/m² │ 물건 찾기 용이          │    │
│  └────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  [설계 고려 사항]                                                     │
│                                                                     │
│  1. 주간 vs 야간                                                     │
│     ───────────────                                                 │
│     주간: 휘도 높아도 상관없음 (주변이 더 밝으니까)                    │
│     야간: 휘도 낮아야 함 (어둠 적응된 눈이 민감)                       │
│     → 자동으로 밝기 조절하는 오토 디밍 기능 적용                       │
│                                                                     │
│  2. 직접 시야 vs 주변 시야                                           │
│     ───────────────────────                                         │
│     직접 시야(운전자가 바로 보는 곳): 휘도 낮게                        │
│     주변 시야(곁눈질로 보는 곳): 휘도 약간 높아도 OK                   │
│                                                                     │
│  3. 색상별 휘도 체감 차이                                             │
│     ───────────────────────                                         │
│     같은 cd/m²라도 색상에 따라 다르게 느껴짐                          │
│     - 흰색/노란색: 밝게 느껴짐                                        │
│     - 파란색: 어둡게 느껴짐 (눈의 감도 특성)                          │
│     → 파란색은 휘도를 약간 높여야 동등하게 보임                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 3. 광균일도 (Light Uniformity)

### 3.1 광균일도란?

```
┌─────────────────────────────────────────────────────────────────────┐
│                     광균일도 완전 이해                                │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  정의: 조명 표면의 밝기가 얼마나 고르게 분포하는가                     │
│                                                                     │
│  공식: 균일도 = (최소 휘도 / 최대 휘도) × 100%                        │
│                                                                     │
│  [시각적 이해]                                                        │
│                                                                     │
│    균일도 낮음 (나쁨)           균일도 높음 (좋음)                    │
│    ═══════○═══════             ═════════════════                    │
│        밝음↑어두움                  고르게 밝음                       │
│                                                                     │
│    LED 바로 앞은 밝고           전체가 비슷하게 밝음                   │
│    멀어지면 어두움                                                   │
│                                                                     │
│                                                                     │
│  [균일도 계산 예시]                                                   │
│  ─────────────────                                                  │
│  라이트 가이드 5개 지점 휘도 측정:                                    │
│                                                                     │
│    지점 A: 35 cd/m²                                                  │
│    지점 B: 32 cd/m²                                                  │
│    지점 C: 28 cd/m²  ← 최소                                          │
│    지점 D: 33 cd/m²                                                  │
│    지점 E: 38 cd/m²  ← 최대                                          │
│                                                                     │
│    균일도 = (28 / 38) × 100% = 73.7%                                 │
│                                                                     │
│    ❌ 규격 미달 (최소 80% 이상 필요)                                  │
│                                                                     │
│                                                                     │
│  [균일도 등급]                                                        │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ 균일도        │ 등급   │ 평가                              │    │
│  ├────────────────────────────────────────────────────────────┤    │
│  │ 90% 이상      │ 우수   │ 프리미엄 차량, 눈에 띄는 차이 없음 │    │
│  │ 80~90%       │ 양호   │ 일반 양산 기준, 미세 차이 인지     │    │
│  │ 70~80%       │ 보통   │ 밝고 어두운 부분 눈에 띔           │    │
│  │ 70% 미만      │ 불량   │ 명확한 밝기 차이, 품질 불량        │    │
│  └────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  ★ 자동차 무드램프 기준: 80% 이상 (프리미엄은 90% 이상)               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 3.2 균일도가 나빠지는 원인과 해결책

```
┌─────────────────────────────────────────────────────────────────────┐
│                   광균일도 저하 원인과 해결책                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [원인 1] LED 광원 위치에서만 밝음 (Hot Spot)                         │
│  ─────────────────────────────────────                              │
│                                                                     │
│     LED ━━━━━━━━━━━━━━━━━━━━━━━━━━━                                 │
│      ↓                                                              │
│     🔴 밝음    ───────────────────  어두움                          │
│                                                                     │
│  해결책:                                                             │
│  • LED 입광부에 확산 처리 (디퓨저)                                    │
│  • 초기 광 믹싱 영역 확보 (5~10mm)                                    │
│  • 입광부 반사판 설계로 빛 분산                                       │
│                                                                     │
│                                                                     │
│  [원인 2] 끝으로 갈수록 어두움                                        │
│  ─────────────────────────────                                      │
│                                                                     │
│     LED ━━━━━━━━━━━━━━━━━━━━━━━━━━━                                 │
│      밝음 ───────────────────▶ 어두움                                │
│                (빛이 점점 빠져나가서)                                 │
│                                                                     │
│  해결책:                                                             │
│  • 끝쪽 추출 패턴 밀도 높이기                                         │
│  • 반대쪽에 반사 미러 추가                                            │
│  • 양쪽 입광 방식 적용                                                │
│                                                                     │
│                                                                     │
│  [원인 3] 추출 패턴이 직접 보임                                       │
│  ─────────────────────────────                                      │
│                                                                     │
│     ┌────────────────────────────┐                                  │
│     │  ·  ·  ·  ·  ·  ·  ·  · │ ← 도트 패턴이 눈에 보임             │
│     └────────────────────────────┘                                  │
│                                                                     │
│  해결책:                                                             │
│  • 패턴 크기 줄이기 (0.3mm 이하)                                      │
│  • 불규칙 패턴 배열 (무아레 방지)                                     │
│  • 표면 확산 처리 (샌드블라스트, 매트 코팅)                            │
│                                                                     │
│                                                                     │
│  [원인 4] 온도 변화로 광 특성 변화                                    │
│  ───────────────────────────────                                    │
│                                                                     │
│  온도 상승 → 재료 굴절률 변화 → 광 추출 변화 → 균일도 저하            │
│                                                                     │
│  해결책:                                                             │
│  • 온도 안정성 높은 재료 선정 (광학 등급 PMMA)                         │
│  • LED 방열 설계 강화                                                 │
│  • -40~85°C 범위 광학 시뮬레이션                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

```
│  방법 1: 전압 낮추기 (아날로그 디밍)                                  │
│  ────────────────────────────────────                               │
│  3.2V → 2.0V로 낮추면 어두워짐                                        │
│  BUT! 문제점:                                                        │
│  • 전압 낮으면 LED 색상이 변함 (빨간색 쪽으로)                         │
│  • 에너지 효율 낮음 (저항에서 열 발생)                                 │
│  • 정밀한 제어 어려움                                                 │
│                                                                     │
│  방법 2: PWM (디지털 디밍) ⭐ 채택!                                   │
│  ────────────────────────────────                                   │
│  전압은 그대로 3.2V, 대신 빠르게 ON/OFF 반복                          │
│  장점:                                                               │
│  • 색상 변화 없음 (항상 정격 전압)                                     │
│  • 에너지 효율 좋음 (스위칭만)                                         │
│  • 정밀한 디지털 제어 가능 (256단계, 1024단계...)                      │
│                                                                     │
│                                                                     │
│  [PWM 동작 원리 - 시각화]                                             │
│                                                                     │
│     50% 밝기 (Duty 50%)                                              │
│     ┌──┐  ┌──┐  ┌──┐  ┌──┐                                         │
│     │  │  │  │  │  │  │  │                                         │
│  ───┘  └──┘  └──┘  └──┘  └───                                       │
│     ON OFF ON OFF ON OFF ON OFF                                      │
│     └─1주기─┘                                                        │
│                                                                     │
│     25% 밝기 (Duty 25%)                                              │
│     ┌┐    ┌┐    ┌┐    ┌┐                                            │
│     ││    ││    ││    ││                                            │
│  ───┘└────┘└────┘└────┘└────                                        │
│     ON  OFF                                                         │
│                                                                     │
│     75% 밝기 (Duty 75%)                                              │
│     ┌────┐┌────┐┌────┐┌────┐                                        │
│     │    ││    ││    ││    │                                        │
│  ───┘    └┘    └┘    └┘    └                                        │
│       ON   OFF                                                       │
│                                                                     │
│                                                                     │
│  Duty Cycle = ON 시간 / 전체 주기 × 100%                              │
│                                                                     │
│  Duty 0% = 완전 OFF (안 보임)                                         │
│  Duty 100% = 완전 ON (최대 밝기)                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 6.2 PWM 주파수가 중요한 이유

```
┌─────────────────────────────────────────────────────────────────────┐
│                   PWM 주파수와 깜빡임 문제                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [문제: 사람 눈은 깜빡임을 인지함!]                                    │
│  ─────────────────────────────────                                  │
│                                                                     │
│  PWM = 빠르게 ON/OFF 반복                                            │
│  → 주파수가 낮으면 "깜빡거림"이 보임!                                  │
│  → 눈 피로, 두통, 불쾌감 유발                                         │
│                                                                     │
│                                                                     │
│  [주파수별 깜빡임 인지]                                               │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ PWM 주파수    │ 깜빡임 인지        │ 평가                   │    │
│  ├────────────────────────────────────────────────────────────┤    │
│  │ 30Hz 이하    │ 명확하게 보임       │ ❌ 사용 불가           │    │
│  │ 30~60Hz     │ 주변 시야에서 보임   │ ❌ 불편함              │    │
│  │ 60~100Hz    │ 빠른 움직임 시 보임  │ ⚠️ 개선 필요           │    │
│  │ 100~200Hz   │ 민감한 사람만 인지   │ △ 최소 기준           │    │
│  │ 200~500Hz   │ 거의 인지 못함      │ ✅ 권장                │    │
│  │ 500Hz~1kHz  │ 인지 불가          │ ✅ 우수                │    │
│  │ 1kHz 이상   │ 인지 불가          │ ✅ 최상 (과도할 수 있음) │    │
│  └────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  ★ 자동차 무드램프 권장: 200Hz ~ 1kHz                                 │
│                                                                     │
│                                                                     │
│  [왜 200Hz 이상이어야 하나요?]                                        │
│  ──────────────────────────                                         │
│                                                                     │
│  1. 인간 눈의 잔상 효과 (Persistence of Vision)                       │
│     ────────────────────────────────────────                        │
│     눈이 본 이미지는 약 1/60초(16ms) 동안 유지됨                       │
│     → 60Hz 이상이면 연속으로 보임                                     │
│     BUT 주변 시야, 움직임 감지 등에서는 더 민감                        │
│     → 안전하게 200Hz 이상 권장                                        │
│                                                                     │
│  2. 자동차 특수 환경                                                  │
│     ────────────────────                                            │
│     • 야간 운전: 눈이 어둠에 적응 → 밝기 변화에 민감                   │
│     • 주변 시야: 무드램프는 직접 안 보지만 곁눈에 들어옴               │
│     • 진동: 차량 진동 + 저주파 PWM = 깜빡임 심화                       │
│                                                                     │
│                                                                     │
│  [왜 너무 높은 주파수는 안 쓰나요?]                                    │
│  ─────────────────────────────────                                  │
│                                                                     │
│  주파수 높이면 단점:                                                  │
│  • MCU 부하 증가 (빠른 스위칭 처리)                                    │
│  • EMI 노이즈 증가 (고주파 전자기 간섭)                                │
│  • 스위칭 손실 증가 (MOSFET 효율 저하)                                 │
│                                                                     │
│  → 200Hz~1kHz가 최적 구간!                                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 6.3 PWM 해상도 (비트 수)

```
┌─────────────────────────────────────────────────────────────────────┐
│                      PWM 해상도 (비트 수)                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [해상도란?]                                                         │
│                                                                     │
│  밝기를 몇 단계로 나눌 수 있는가                                      │
│                                                                     │
│  8bit = 2^8 = 256단계 (0~255)                                        │
│  10bit = 2^10 = 1024단계                                             │
│  12bit = 2^12 = 4096단계                                             │
│                                                                     │
│                                                                     │
│  [해상도별 밝기 단계]                                                 │
│                                                                     │
│     8bit (256단계)                                                   │
│     ├─┼─┼─┼─┼─┼─┼─┼─┤                                              │
│     0  32 64 96 ... 255                                              │
│     OFF           MAX                                                │
│                                                                     │
│     → 단계가 크면 밝기 변화 시 "툭툭" 끊기는 느낌                      │
│                                                                     │
│     12bit (4096단계)                                                 │
│     ├┼┼┼┼┼┼┼┼┼┼┼┼┼┼┼┼┼┼┼┼┼┤                                        │
│     0                  4095                                          │
│     OFF                MAX                                           │
│                                                                     │
│     → 매우 부드러운 디밍 (페이드 인/아웃)                              │
│                                                                     │
│                                                                     │
│  [무드램프 권장 해상도]                                               │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ 해상도    │ 단계 수   │ 용도                               │    │
│  ├────────────────────────────────────────────────────────────┤    │
│  │ 8bit     │ 256      │ 기본 무드램프, ON/OFF 위주          │    │
│  │ 10bit    │ 1024     │ 부드러운 디밍 필요시                │    │
│  │ 12bit    │ 4096     │ 고급 앰비언트 라이팅, 페이드 효과   │    │
│  └────────────────────────────────────────────────────────────┘    │
│                                                                     │
│  ★ RGB LED 3채널 = 각각 8bit × 3 = 16,777,216가지 색상 표현 가능!    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 7. LED 전기적 특성

### 7.1 순방향 전압 (Vf) - 왜 3.2V인가?

```
┌─────────────────────────────────────────────────────────────────────┐
│                순방향 전압 (Forward Voltage, Vf)                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  정의: LED가 켜지기 위해 필요한 최소 전압                              │
│        = LED 양단에 걸리는 전압                                       │
│                                                                     │
│                                                                     │
│  [왜 "순방향"인가요?]                                                 │
│  ─────────────────────                                              │
│  LED = 다이오드의 일종                                                │
│  다이오드는 한 방향으로만 전류가 흐름                                  │
│                                                                     │
│     ▶ 순방향: (+) → LED → (-) = 불 켜짐 💡                          │
│     ▷ 역방향: (-) → LED → (+) = 안 켜짐 ⬛                          │
│                                                                     │
│  순방향 전압 = 순방향으로 연결했을 때 LED 양단 전압                    │
│                                                                     │
│                                                                     │
│  [색상별 순방향 전압 - 왜 다른가?]                                     │
│  ─────────────────────────────────                                  │
│                                                                     │
│  핵심 원리: 빛의 에너지 = 전자의 에너지                                │
│                                                                     │
│  E = h × f  (플랑크 상수 × 주파수)                                   │
│                                                                     │
│  → 파란색 빛 = 높은 주파수 = 높은 에너지 = 높은 전압 필요              │
│  → 빨간색 빛 = 낮은 주파수 = 낮은 에너지 = 낮은 전압 OK                │
│                                                                     │
│                                                                     │
│  [색상별 순방향 전압]                                                  │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ LED 색상     │ 파장 (nm)    │ 순방향 전압 (Vf) │ 반도체 재료│    │
│  ├────────────────────────────────────────────────────────────┤    │
│  │ 적외선 (IR)  │ >760        │ 1.2~1.5V        │ GaAs      │    │
│  │ 빨강 (Red)   │ 620~760     │ 1.8~2.2V        │ AlGaAs    │    │
│  │ 주황 (Orange)│ 590~620     │ 2.0~2.4V        │ GaAsP     │    │
│  │ 노랑 (Yellow)│ 570~590     │ 2.0~2.4V        │ GaAsP     │    │
│  │ 초록 (Green) │ 495~570     │ 2.0~3.5V        │ GaP/InGaN │    │
│  │ 파랑 (Blue)  │ 450~495     │ 3.0~3.5V        │ InGaN     │    │
│  │ 흰색 (White) │ (혼합)      │ 3.0~3.5V        │ InGaN+형광│    │
│  │ 보라 (UV)    │ <400        │ 3.5~4.0V        │ AlGaN     │    │
│  └────────────────────────────────────────────────────────────┘    │
│                                                                     │
│                                                                     │
│  [★ 무드램프에서 "Vf = 3.2V"가 많은 이유]                             │
│  ─────────────────────────────────────────                          │
│                                                                     │
│  무드램프 RGB LED 구성:                                               │
│  • Red: ~2.0V                                                       │
│  • Green: ~3.0V                                                     │
│  • Blue: ~3.2V ← 가장 높음!                                          │
│                                                                     │
│  시스템 설계 시 가장 높은 전압(Blue)에 맞춰야 함                       │
│  → 그래서 대표값으로 "3.2V" 또는 "3.0~3.5V"를 언급                    │
│                                                                     │
│  ❓ 근데 차량은 12V인데 3.2V LED를 어떻게?                            │
│  → 저항 또는 정전류 드라이버로 전압 조절                              │
│  → 또는 LED 여러 개 직렬 연결 (3.2V × 3개 = 9.6V)                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

```
│                    순방향 전류 (If)와 밝기                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  정의: LED에 흐르는 전류량                                            │
│                                                                     │
```

### 7.3 와트수 (전력) 계산

```
┌─────────────────────────────────────────────────────────────────────┐
│                      LED 전력 (와트수) 계산                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [기본 공식]                                                         │
│                                                                     │
│     P = V × I                                                        │
│                                                                     │
│     P: 전력 (W, 와트)                                                │
│     V: 전압 (V, 볼트) = 순방향 전압 Vf                                │
│     I: 전류 (A, 암페어) = 순방향 전류 If                              │
│                                                                     │
│                                                                     │
│  [계산 예시 1] 일반 RGB LED 1개                                       │
│  ─────────────────────────────                                      │
│  Blue LED: Vf=3.2V, If=20mA=0.02A                                   │
│                                                                     │
│  P = 3.2V × 0.02A = 0.064W = 64mW                                   │
│                                                                     │
│  RGB 3색 모두 켜면:                                                   │
│  R: 2.0V × 0.02A = 0.04W                                            │
│  G: 3.0V × 0.02A = 0.06W                                            │
│  B: 3.2V × 0.02A = 0.064W                                           │
│  합계: 약 0.16W = 160mW                                              │
│                                                                     │
│                                                                     │
│  [계산 예시 2] 무드램프 1개 라인 (LED 10개)                            │
│  ───────────────────────────────────────                            │
│  LED당 0.1W × 10개 = 1W                                              │
│                                                                     │
│                                                                     │
│  [계산 예시 3] 차량 전체 무드램프                                       │
│  ──────────────────────────────                                     │
│  - IP 라인: 2W                                                       │
│  - 도어 트림 4개: 1W × 4 = 4W                                         │
│  - 센터 콘솔: 1W                                                     │
│  - 풋웰 4개: 0.5W × 4 = 2W                                           │
│  ──────────────────────────                                         │
│  합계: 약 9W                                                         │
│                                                                     │
│  → 차량 배터리 관점에서 부담 없는 수준                                 │
│     (헤드라이트 1개가 55W 이상)                                       │
│                                                                     │
│                                                                     │
│  [와트수가 정해진 이유가 있나요?]                                      │
│  ─────────────────────────────                                      │
│                                                                     │
│  LED 자체에 "정해진 와트수"는 없음!                                    │
│  제조사가 권장하는 "정격" 스펙이 있을 뿐                               │
│                                                                     │
│  정격 와트수 = Vf(정격) × If(정격)                                    │
│                                                                     │
│  예) "1W LED" = 3.2V × 350mA = 1.12W ≈ 1W                           │
│                                                                     │
│  더 낮은 전류로 구동하면 와트수도 낮아짐                               │
│  → 밝기↓, 발열↓, 수명↑                                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 8.1 저항 계산 (정전류 구동)

```
┌─────────────────────────────────────────────────────────────────────┐

│                                                                     │
│  [왜 저항이 필요한가?]                                                │

│                                                                     │
│                                                                     │
│  [저항 계산 공식]                                                     │
│                                                                     │
│     R = (Vsupply - Vf) / If                                          │
│                                                                     │
│     R: 저항 (Ω, 옴)                                                  │
│     Vsupply: 공급 전압 (예: 12V)                                     │
│     Vf: LED 순방향 전압 (예: 3.2V)                                   │
│     If: 원하는 전류 (예: 20mA = 0.02A)                                │
│                                                                     │
│                                                                     │
│  [계산 예시]                                                         │
│  ─────────────                                                      │
│  조건: 12V 전원, Blue LED (Vf=3.2V), 20mA 전류                       │
│                                                                     │
│     R = (12V - 3.2V) / 0.02A                                         │
│       = 8.8V / 0.02A                                                 │
│       = 440Ω                                                         │
│                                                                     │
│  → 440Ω 저항 사용 (실제로는 430Ω 또는 470Ω 표준값 사용)               │
│                                                                     │
│                                                                     │
│  [저항 발열량 계산]                                                   │
│  ─────────────────                                                  │
│  저항에서 소비되는 전력 = 열로 변환                                    │
│                                                                     │
│     P = (Vsupply - Vf) × If                                          │
│       = 8.8V × 0.02A                                                 │
│       = 0.176W                                                       │
│                                                                     │
│  → 1/4W(0.25W) 저항 사용 OK (여유 있게 1/2W 권장)                     │
│                                                                     │
│                                                                     │
│  [회로도]                                                            │
│                                                                     │
│     +12V ────┬──[ R ]──┬──▷|──┬── GND                               │
│              │         │  LED │                                     │
│              │        저항    │                                     │
│              │        440Ω   │                                      │
│              │               │                                      │
│              └───────────────┘                                      │
│                                                                     │
│  💡 실제 무드램프는 저항 대신 "정전류 드라이버 IC" 많이 사용            │
│     (전압 변동에도 일정한 전류 유지)                                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 8.2 휘도 목표 달성 계산

```
┌─────────────────────────────────────────────────────────────────────┐
│                    휘도 설계 역산 프로세스                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [목표]                                                              │
│  도어 트림 라인 휘도: 30 cd/m²                                        │
│  라인 길이: 50cm                                                     │
│  발광폭: 5mm                                                         │
│                                                                     │
│                                                                     │
│  [Step 1] 발광 면적 계산                                              │
│  ─────────────────────                                              │
│  면적 = 500mm × 5mm = 2,500 mm² = 0.0025 m²                         │
│                                                                     │
│                                                                     │
│  [Step 2] 필요 광도 계산                                              │
│  ─────────────────────                                              │
│  휘도 = 광도 / 면적                                                   │
│  광도 = 휘도 × 면적 = 30 cd/m² × 0.0025 m² = 0.075 cd                │
│                                                                     │
│                                                                     │
│  [Step 3] 필요 광속 추정                                              │
│  ─────────────────────                                              │
│  시야각 120°의 램버시안 발광 가정                                     │
│  광속 ≈ 광도 × π ≈ 0.075 cd × 3.14 = 0.24 lm                         │
│                                                                     │
│  BUT! 라이트 가이드 효율 고려 (보통 30~50%)                            │
│  필요 LED 광속 = 0.24 lm / 0.4 = 0.6 lm                               │
│                                                                     │
│                                                                     │
│  [Step 4] LED 선정                                                    │
│  ─────────────────                                                  │
│  일반적인 SMD LED 광속: 5~20 lm @ 20mA                                │
│                                                                     │
│  0.6 lm 필요 → 5 lm LED를 저전류로 구동                               │
│  또는 밝기 조절을 위해 PWM Duty 12%로 설정                            │
│                                                                     │
│                                                                     │
│  [실제 설계 시 고려사항]                                               │
│  ───────────────────────                                            │
│  1. 라이트 가이드 효율은 형상/재질마다 다름 → 시뮬레이션 필요          │
│  2. 온도에 따른 LED 효율 변화 고려                                    │
│  3. 노화에 따른 밝기 저하 마진 (보통 +20~30%)                          │
│  4. 로트별 LED 편차 고려                                              │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 📋 핵심 정리 체크리스트

```
┌─────────────────────────────────────────────────────────────────────┐
│                    핵심 파라미터 정리표                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [빛의 단위]                                                         │
│  □ 루멘(lm): 총 빛의 양 - 무드램프 LED는 1~20lm 정도                  │
│  □ 칸델라(cd): 특정 방향 밝기                                         │
│  □ 휘도(cd/m²): 눈에 보이는 밝기 - 무드램프 10~50cd/m² ⭐             │
│  □ 럭스(lux): 표면에 닿는 빛                                          │
│                                                                     │
│  [광학 파라미터]                                                      │
│  □ 광균일도: 80% 이상 (프리미엄 90% 이상)                             │
│  □ 발광폭: 3~10mm (위치별 상이)                                       │
│  □ 시야각: 넓을수록 좋음 (120° 이상)                                  │
│  □ 색온도: RGB로 자유롭게 표현                                        │
│                                                                     │
│  [PWM 디밍]                                                          │
│  □ 주파수: 200Hz~1kHz (깜빡임 방지)                                   │
│  □ 해상도: 8bit(256단계) ~ 12bit(4096단계)                            │
│  □ Duty: 0%(OFF) ~ 100%(MAX)                                         │
│                                                                     │
│  [LED 전기 특성]                                                      │
│  □ 순방향 전압(Vf): 색상에 따라 1.8~3.5V (Blue 기준 ~3.2V)            │
│  □ 순방향 전류(If): 정격 20mA (타입별 상이)                            │
│  □ 전력(W) = Vf × If                                                 │
│                                                                     │
│  [설계 핵심]                                                          │
│  □ 저항 계산: R = (Vsupply - Vf) / If                                 │
│  □ 휘도 목표 → LED 선정 → PWM 설정                                   │
│  □ 라이트 가이드 효율 30~50% 고려                                     │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
