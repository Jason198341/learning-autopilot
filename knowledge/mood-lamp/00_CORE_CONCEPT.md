## 2. 트랜시버 (Transceiver) 완전 이해

### 2.1 트랜시버란?

```
┌─────────────────────────────────────────────────────────────────────┐
│                    트랜시버 (Transceiver) 정의                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  트랜시버 = Transmitter(송신) + Receiver(수신)의 합성어              │
│                                                                     │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                              │   │
│  │   무드램프에서 트랜시버의 역할:                               │   │
│  │                                                              │   │
│  │   1. [수신] BCM으로부터 명령 수신 (LIN/CAN 통신)              │   │
│  │      └─ "파란색, 밝기 70%로 설정해라"                        │   │
│  │                                                              │   │
│  │   2. [처리] 명령 해석 및 LED 구동 신호 생성                   │   │
│  │      └─ RGB 각 채널 PWM 값 계산                              │   │
│  │                                                              │   │
│  │   3. [송신] LED에 PWM 신호 출력                               │   │
│  │      └─ R: 0%, G: 50%, B: 100% (파란색)                      │   │
│  │                                                              │   │
│  │   4. [송신] BCM에 상태 정보 회신 (진단용)                     │   │
│  │      └─ "LED 정상 동작 중, 온도 45°C"                        │   │
│  │                                                              │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  [비유로 이해하기]                                                   │
│                                                                     │
│      트랜시버는 "통역사 + 실행자"                                    │
│                                                                     │
│      BCM(사장님)                    트랜시버(팀장)      LED(직원)    │
│         │                              │                  │         │
│         │  "파란색으로!"              │                  │         │
│         │ ─────────────────────────► │                  │         │
│         │                              │  "R=0, G=128,   │         │
│         │                              │   B=255!"       │         │
│         │                              │ ───────────────►│         │
│         │                              │                  │ 💡파란빛│
│         │                              │                  │         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 2.4 트랜시버 통신 방식

```
┌─────────────────────────────────────────────────────────────────────┐
│                   트랜시버 통신 프로토콜                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  [LIN (Local Interconnect Network)]                                 │
│                                                                     │
│  무드램프에서 가장 많이 사용하는 통신 방식                            │
│                                                                     │
│      ┌───────────┐                                                 │
│      │   BCM     │  Master                                         │
│      │ (마스터)  │                                                 │
│      └─────┬─────┘                                                 │
│            │ LIN Bus (Single Wire)                                 │
│      ══════╪═══════════════════════════════════════════            │
│            │           │           │           │                   │
│       ┌────┴────┐ ┌────┴────┐ ┌────┴────┐ ┌────┴────┐             │
│       │Slave #1 │ │Slave #2 │ │Slave #3 │ │Slave #4 │             │
│       │트랜시버 │ │트랜시버 │ │트랜시버 │ │트랜시버 │             │
│       │(도어L)  │ │(도어R)  │ │(IP)     │ │(콘솔)   │             │
│       └─────────┘ └─────────┘ └─────────┘ └─────────┘             │
│                                                                     │
│  LIN 특징:                                                          │
│  - 단선식 (Single Wire) → 비용 절감                                 │
│  - 저속 (20kbps) → 조명 제어에 충분                                 │
│  - Master-Slave 구조 → BCM이 마스터                                 │
│  - 저비용 → CAN 대비 1/3 가격                                       │
│                                                                     │
│  [LIN 프레임 구조]                                                   │
│                                                                     │
│  ┌──────┬──────┬──────────────────────┬──────┐                     │
│  │Break │ Sync │       Identifier     │ Data │                     │
│  │ 13b  │ 0x55 │    (ID + Parity)     │0~8B  │                     │
│  └──────┴──────┴──────────────────────┴──────┘                     │
│                                                                     │
│  예시 명령: "도어 좌측 무드램프를 파란색 70%로"                       │
│                                                                     │
│  ID: 0x21 (도어좌측 트랜시버 주소)                                   │
│  Data: [0x00, 0x46, 0xFF, 0xB3]                                    │
│         │     │     │     │                                        │
│         R=0   G=70  B=255 밝기=70%                                  │
│                                                                     │
│  [RGB-W 제어 데이터 포맷]                                            │
│  ┌────────────────────────────────────────────────────────────┐    │
│  │ Byte │  내용      │  범위      │  설명                     │    │
│  ├────────────────────────────────────────────────────────────┤    │
│  │  0   │  Red       │  0~255     │  빨간색 PWM 값            │    │
│  │  1   │  Green     │  0~255     │  초록색 PWM 값            │    │
│  │  2   │  Blue      │  0~255     │  파란색 PWM 값            │    │
│  │  3   │  Intensity │  0~100%    │  전체 밝기 (디밍)         │    │
│  │  4   │  Effect    │  0~10      │  효과 (정적/호흡/웨이브)  │    │
```
