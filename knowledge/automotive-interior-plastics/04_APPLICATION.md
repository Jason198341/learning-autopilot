## Real-World Use Cases

### Use Case 1: 인스트루먼트 패널 소재 선정

**Context**: 신차 개발 프로젝트에서 IP 기재 소재 선정

**Requirements**:
- 내열: IP 상단 110°C 노출
- 강성: 에어백 전개 반력 지지
- 비용: 양산 원가 목표 충족
- 경량: 연비 목표 달성

**How it's applied**:
```
Step 1: 후보 재질 선정
├── PP-TD30: 내열 부족 (HDT 110°C)
├── PP-LGF30: 내열 OK (HDT 150°C) ★
├── PP-GF30: 표면 품질 이슈
└── PC/ABS: 비용 초과

Step 2: PP-LGF30 상세 검토
├── 물성: 인장 80MPa, 굴곡탄성률 5.5GPa ✓
├── 내열: HDT 150°C > 요구 110°C ✓
├── 비중: 1.12 (ABS 대비 경량) ✓
└── 비용: ABS 대비 90% ✓

Step 3: 표면 처리 결정
├── 스킨 일체 성형: TPO 슬러시 스킨
└── 무도장 그레인 패턴
```

**Result**: PP-LGF30 기재 + TPO 스킨 일체 성형

### Use Case 2: 도어 트림 구성

**Context**: 도어 트림 소재/구조 설계

**Structure**:
```
┌─────────────────────────────────────────┐
│           도어 트림 단면                 │
├─────────────────────────────────────────┤
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 표피 (TPO 또는 PVC/ABS 시트)     │   │
│  ├─────────────────────────────────┤   │
│  │ 폼층 (PP 폼 또는 PU 폼)         │   │ ← 소프트 터치
│  ├─────────────────────────────────┤   │
│  │ 기재 (PP-TD20)                  │   │ ← 구조 지지
│  └─────────────────────────────────┘   │
│                                         │
│  ★ 암레스트: 소프트 터치 필수 (폼 두께↑) │
│  ★ 맵포켓: 강성 필요 (폼 없이 기재만)    │
│                                         │
└─────────────────────────────────────────┘
```

**Material Selection**:
| 부위 | 재질 | 이유 |
|------|------|------|
| 기재 | PP-TD20 | 저비용, 충분한 강성 |
| 표피 | TPO | 소프트터치, 재활용 용이 |
| 폼 | PP 폼 | 올레핀 호환 재활용 |
| 스위치 베젤 | PC/ABS | 도금, 외관 품질 |
| 스피커 그릴 | ABS | 세밀 패턴, 음향 설계 |

### Use Case 3: 소프트 터치 구현 방법 비교

**Context**: 센터 콘솔에 소프트 터치 감성 구현

**Options**:

| 방법 | 구조 | 장점 | 단점 | 비용 |
|------|------|------|------|------|
| **TPO 스킨** | 기재+TPO | 재활용↑ | 디테일↓ | ★★☆ |
| **PU 스킨** | 기재+PU스프레이 | 감성↑ | 재활용↓ | ★★★ |
| **PVC 시트** | 기재+PVC+폼 | 다양한 패턴 | 환경이슈 | ★★☆ |
| **소프트 도료** | 기재+도료 | 공정 단순 | 박리 위험 | ★★★ |
| **2K 사출** | Hard+Soft 일체 | 공정↓ | 금형비↑ | ★★☆ |

**Recommendation Flow**:
```
고급 감성 필요?
├── Yes → PU 스킨 또는 가죽
└── No → 재활용 중요?
          ├── Yes → TPO 스킨
          └── No → PVC 시트 (저비용)
```

## Practice Problems

### Level 1: Basic (기초)

**Problem 1.1**: 재질 식별
> 다음 부품에 가장 적합한 재질은?
> - 글로브박스 내부

<details>
<summary>Solution</summary>

```
정답: PP 또는 PP-TD10

이유:
- 비노출 부위 → 외관 품질 낮아도 됨
- 내열 낮아도 됨 (IP 하단)
- 저비용 우선
```

</details>

**Problem 1.2**: 물성 해석
> PP-TD20의 HDT 100°C는 무엇을 의미하는가?

<details>
<summary>Solution</summary>

```
HDT (Heat Deflection Temperature):
- 1.8 MPa 하중에서 변형 시작 온도
- 100°C = 100°C 이상 환경에서 변형 가능성

실무 적용:
- IP 상단 (110°C 노출) → PP-TD20 부적합
- 도어 트림 (85°C 노출) → PP-TD20 적합
```

</details>

### Level 2: Intermediate (중급)

**Problem 2.1**: 소재 선정
> 클러스터 렌즈(계기판 투명 커버) 소재로 PMMA와 PC 중 어떤 것이 적합한가?

<details>
<summary>Solution</summary>

```
비교:
┌─────────────┬───────────┬───────────┐
│ 특성        │ PMMA      │ PC        │
├─────────────┼───────────┼───────────┤
│ 투명도      │ 92%       │ 89%       │
│ 내충격      │ 낮음      │ 매우 높음  │
│ 내스크래치  │ 좋음      │ 나쁨      │
│ 내열        │ 85°C      │ 135°C     │
│ 가격        │ 중간      │ 높음      │
└─────────────┴───────────┴───────────┘

정답: PC (폴리카보네이트)

이유:
1. 클러스터는 충격 시 파손 안전성 필수 (PC 내충격↑)
2. 클러스터 위치는 내열 필요 (PC 135°C)
3. 스크래치는 하드코팅으로 보완 가능
```

</details>

**Problem 2.2**: 문제 해결
> 도장한 ABS 부품에서 도막 박리가 발생했다. 원인과 대책은?

<details>
<summary>Solution</summary>

```
가능한 원인:
1. 전처리 불량 (탈지/활성화)
2. 프라이머 미적용 또는 부적합
3. 사출 시 이형제 과다
4. 재생재 혼입
5. 건조 불충분

대책:
1. IPA 세정 → 화염처리 또는 플라즈마 처리
2. ABS 전용 프라이머 적용
3. 이형제 사용 최소화
4. 버진재 사용 또는 재생재 비율 제한
5. 충분한 플래시 오프 타임 확보
```

</details>

### Level 3: Advanced (고급)

**Problem 3.1**: 원가 분석
> PP-TD20 (3,000원/kg) vs PC/ABS (5,500원/kg)
> 부품 중량 500g일 때 재료비 차이는?
> 도장 비용 PC/ABS 600원, PP-TD20 도장 불가 시 어떻게 판단?

<details>
<summary>Solution</summary>

```
재료비:
- PP-TD20: 0.5kg × 3,000원 = 1,500원
- PC/ABS: 0.5kg × 5,500원 = 2,750원
- 차이: 1,250원 (PC/ABS가 비쌈)

도장 포함 총비용:
- PP-TD20: 도장 불가 → 무도장 (성형색/시보)
- PC/ABS + 도장: 2,750 + 600 = 3,350원

판단:
1. 외관 품질이 중요하면? → PC/ABS + 도장
2. 비용 최우선이면? → PP-TD20 무도장
3. 절충안: PP 기재 + IMD 필름 (2,200원 수준)
```

</details>

**Problem 3.2**: 설계 변경
> 기존 PA6-GF30 브라켓을 PP-LGF40으로 대체 검토. 고려사항은?

<details>
<summary>Solution</summary>

```
물성 비교:
┌─────────────┬───────────┬───────────┐
│ 특성        │ PA6-GF30  │ PP-LGF40  │
├─────────────┼───────────┼───────────┤
│ 인장강도    │ 180 MPa   │ 100 MPa   │
│ 굴곡탄성률  │ 9.5 GPa   │ 7.0 GPa   │
│ HDT         │ 210°C     │ 160°C     │
│ 비중        │ 1.35      │ 1.18      │
│ 흡습        │ 있음      │ 없음      │
│ 가격        │ 높음      │ 중간      │
└─────────────┴───────────┴───────────┘

고려사항:
1. 강도 저하 → 단면 보강 (리브 추가) 필요
2. 내열 저하 → 적용 위치 확인 (엔진룸 근접 ✗)
3. 치수 안정성 → PP가 오히려 유리 (흡습 없음)
4. 원가 절감 → 20~30% 절감 가능
5. 경량화 → 비중 저하로 추가 이점

결론:
- 고온 노출 없고, 설계 변경 가능하면 PP-LGF 검토 가치 있음
- 구조 해석 필수
```

</details>

## 재질 선정 체크리스트

```
□ 1. 기능 요구사항
   □ 필요 강도/강성 (인장, 굴곡, 충격)
   □ 사용 온도 범위 (최고/최저)
   □ 내약품성 (세정제, 핸드크림 등)
   □ 치수 안정성 요구 수준

□ 2. 외관 요구사항
   □ 노출/비노출 부품 여부
   □ 표면 처리 방법 (무도장/도장/IMD/도금)
   □ 색상/광택 요구사항
   □ 그레인/텍스처 요구사항

□ 3. 품질 요구사항
   □ VOC/Fogging 기준
   □ 내광성/내열노화
   □ 스크래치/마모 저항
   □ 난연 등급 (해당 시)

□ 4. 가공 요구사항
   □ 부품 크기/복잡도
   □ 성형 방법 (사출/압출/RIM 등)
   □ 2차 가공 (도장/조립/용접)

□ 5. 비용 요구사항
   □ 재료비 목표
   □ 가공비 포함 총원가
   □ 금형비/투자비

□ 6. 기타 요구사항
   □ 재활용 요구 (올레핀계 선호?)
   □ 중량 목표
   □ 공급사 제약
```

## Common Mistakes

| Mistake | Why It's Wrong | Correct Approach |
|---------|----------------|------------------|
| PP에 바로 도장 | 밀착 불량 (무극성) | PP 도장 지양 또는 화염처리+프라이머 |
| PA 부품 설계 시 Dry 물성 사용 | 실제는 흡습 상태 | DAM 물성 기준 설계 |
| 재생재 무제한 사용 | VOC/냄새/물성 저하 | 재생재 비율 제한 (보통 20% 이하) |
| PC 건조 생략 | 가수분해, 실버 | 반드시 3-4시간 건조 (120°C) |
| 얇은 ABS 도금 | 변형/벗겨짐 | 두께 확보, 리브 배치 고려 |
| POM 도장 시도 | 밀착 불가 | POM은 무도장, 성형색 사용 |
