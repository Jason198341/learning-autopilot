# ANALOGY MAP

> 다른 분야와 연결하여 내장재 플라스틱 이해를 확장합니다.

## Primary Analogy

**This concept is like...**

```
[내장재 플라스틱 선정] ≈ [요리 재료 선택]
```

### Why This Analogy Works

| 플라스틱 선정 | 요리 재료 선택 |
|---------------|----------------|
| 재질 (PP, ABS, PC) | 재료 (고기, 생선, 채소) |
| 컴파운딩 (탈크, 유리섬유) | 양념/조미료 |
| 가공 방법 (사출, 압출) | 조리법 (굽기, 삶기, 튀기기) |
| 표면처리 (도장, IMD) | 플레이팅/장식 |
| 품질 기준 (VOC, 내열) | 맛/위생 기준 |
| 원가 | 식자재 가격 |

### 구체적 비유

```
PP = 두부
├── 저렴함
├── 담백함 (무특징)
├── 다른 재료와 잘 어울림 (블렌딩)
└── 그 자체로는 밋밋 (강화재 필요)

ABS = 삼겹살
├── 적당한 가격에 좋은 품질
├── 다재다능 (구이, 찜, 볶음)
├── 그 자체로 맛있음 (외관 우수)
└── 기름기 주의 (도금 조건)

PC = 한우 등심
├── 프리미엄 가격
├── 뛰어난 품질 (투명, 강함)
├── 섬세한 조리 필요 (건조, 온도 관리)
└── 특별한 요리에 적합 (광학 부품)

POM = 치즈
├── 특유의 질감 (저마찰)
├── 가공이 쉬움
├── 특정 용도에 딱 맞음 (기어)
└── 열에 주의 (분해)

PA = 닭가슴살
├── 고단백 = 고강도
├── 수분 관리 중요 (흡습)
├── 다양한 조리법 (GF 강화)
└── 가격 대비 고성능
```

### Where This Analogy Breaks

- 플라스틱은 한번 성형하면 형태 고정 (요리는 재가열 가능)
- 플라스틱 블렌딩은 분자 수준 (요리는 물리적 혼합)
- 플라스틱은 15년 내구성 (요리는 일시적)

## Cross-Domain Connections

| Domain | Similar Concept | Connection |
|--------|-----------------|------------|
| **주방용품** | PP=밀폐용기, PC=물병 | 같은 소재, 유사 요구사항 |
| **가전제품** | TV 베젤, 냉장고 내장 | 유사 외관품질/내구성 기준 |
| **의료기기** | 멸균/생체적합성 플라스틱 | 더 엄격한 품질 기준 |
| **항공기** | 고성능 엔프라, 복합재 | 극한 요구사항의 확장 |
| **건축** | 창호 프로파일, 바닥재 | 대형/장수명 플라스틱 |
| **패션** | 합성섬유, 인조가죽 | 같은 고분자, 다른 형태 |

## Pattern Recognition

### Pattern 1: Cost-Performance Trade-off

```
    가격
     ↑
     │         ● PC
     │     ● PA-GF
     │   ● PC/ABS
     │ ● POM
     │● ABS
     │
●PP ─┼──────────────────→ 성능
     │

모든 산업에서 반복되는 패턴:
- 범용품 → 중급품 → 고급품
- 자동차, 가전, 의류 모두 동일
```

### Pattern 2: 기능 분화 (Specialization)

```
     범용 소재 (PP)
           │
     ┌─────┼─────┐
     │     │     │
     ▼     ▼     ▼
  고강도  저비용  소프트
  PP-GF   PP    TPO

마치...
     밀가루
        │
   ┌────┼────┐
   │    │    │
   빵   면   과자

→ 기본 소재에서 특화된 변종이 파생
```

### Pattern 3: 복합화 (Blending/Composite)

```
재질 A + 재질 B = 새로운 특성

PC + ABS = PC/ABS (강도 + 가공성)
PP + 탈크 = PP-TD (강성 + 저비용)
PP + EPDM = TPO (유연성 + 재활용)

마치...
홍차 + 우유 = 밀크티 (새로운 맛)
```

## Metaphors

### Technical Metaphor
> "내장재 플라스틱 선정은 **교향곡 악기 편성**과 같다. 각 파트(부품)에 맞는 악기(재질)를 배치하고, 전체 조화(원가/품질 밸런스)를 이뤄야 한다. PP는 바이올린(다수/기본), PC는 첼로(소수/핵심), TPO는 타악기(특수 효과)."

### Everyday Metaphor
> "차량 내장재는 **집 인테리어**와 같다. 바닥(플로어)에는 튼튼한 재료, 벽(도어트림)에는 보기 좋은 마감, 테이블(콘솔)에는 실용적인 소재. 예산 내에서 적재적소에 배치하는 것이 핵심."

### Teaching Metaphor
> "플라스틱을 고르는 건 **운동선수 선발**과 비슷해요. PP는 유연한 마라토너(범용), ABS는 균형 잡힌 10종경기 선수(외관), PC는 파워 역도선수(강도), POM은 정밀 사격수(정밀 기계). 포지션(부품)에 맞는 선수(재질)를 뽑아야 팀(차량)이 이깁니다."

## Visual Analogies

### 재질 = 직업

```
플라스틱 세계            사회
═══════════════       ═══════════

PP                →    사무직 (가장 많음, 범용)
ABS               →    영업직 (외부 대면, 외모 중요)
PC                →    기술전문직 (고급, 전문성)
POM               →    장인 (특수 기술, 정밀)
PA                →    운동선수 (고강도, 관리 필요)
TPO               →    서비스직 (부드러움, 접촉)
```

### 가격 vs 특성 스펙트럼

```
저가 ◄─────────────────────────────────► 고가

PP ────── ABS ────── PC/ABS ────── PC
│          │           │           │
범용       외관        균형       특수

건축 비유:
합판 ──── 원목마루 ──── 타일 ──── 대리석
```

## Historical Parallels

- **철 → 알루미늄 → 플라스틱**: 자동차 소재 진화
  - 1900년대: 목재/철
  - 1950년대: 스틸 대시보드
  - 1970년대: 플라스틱 도입
  - 2000년대: 엔지니어링 플라스틱 확대
  - 현재: 친환경 소재 전환

- **아날로그 → 디지털**: 표면처리 진화
  - 과거: 습식 도장 (수작업 의존)
  - 현재: IMD/IML (디지털 인쇄)
  - 미래: 3D 프린팅 직접 제조

## Transfer Learning

### Skills that transfer:

- **재료 물성 이해**: 어떤 산업이든 소재 선정 원리는 유사
- **Trade-off 분석**: 비용-성능 균형은 보편적
- **품질 관리**: VOC, 내구성 시험 → 다른 제품에도 적용

### Insights that apply elsewhere:

- "목적에 맞는 최적 소재가 있다" → 도구 선택의 원리
- "복합화로 약점 보완" → 팀 구성의 원리
- "비용 vs 품질 트레이드오프" → 모든 구매 의사결정

## Unique Aspects

### What makes automotive interior plastics unique:

1. **다중 요구사항 동시 충족**
   - 기계적 강도 + 외관 + 촉감 + 안전 + 환경

2. **극한 환경 내구성**
   - -40°C ~ +110°C, 15년, 자외선

3. **대량 생산 + 고품질**
   - 수십만 대 동일 품질 유지

4. **법규 충족**
   - VOC, 난연, 재활용률, 충돌 안전

5. **감성 품질**
   - 수치로 측정 어려운 "고급스러움"
