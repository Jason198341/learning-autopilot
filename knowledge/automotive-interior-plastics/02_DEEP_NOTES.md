## Layer 1: 범용 플라스틱 (Commodity Plastics)

### 1.1 PP (Polypropylene) - 폴리프로필렌

> 내장재 사용량 1위. 전체 내장 플라스틱의 40~50% 차지

#### 기본 물성

| 특성 | 단위 | 순수 PP | PP-TD20 | PP-LGF30 |
|------|------|---------|---------|----------|
| 비중 | - | 0.90 | 1.04 | 1.12 |
| 인장강도 | MPa | 30-35 | 35-40 | 70-90 |
| 굴곡강도 | MPa | 40-50 | 55-65 | 110-130 |
| 굴곡탄성률 | GPa | 1.2-1.5 | 2.5-3.0 | 5.0-6.5 |
| Izod 충격 | J/m | 30-50 | 40-60 | 80-120 |
| HDT (1.8MPa) | °C | 55-60 | 90-110 | 140-155 |
| 수축률 | % | 1.5-2.5 | 0.8-1.2 | 0.3-0.6 |
| MFI | g/10min | 10-30 | 8-20 | 5-15 |

#### 개질 종류

```
┌─────────────────────────────────────────────────────────────┐
│                    PP 컴파운드 종류                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  PP (순수)                                                  │
│    │                                                        │
│    ├── PP-TD (Talc filled) ────────────────────────────────│
│    │      ├── PP-TD10: 탈크 10% - 기본 보강                 │
│    │      ├── PP-TD20: 탈크 20% - 표준 (IP, 도어트림)       │
│    │      └── PP-TD30: 탈크 30% - 고강성                    │
│    │                                                        │
│    ├── PP-GF (Glass fiber filled) ─────────────────────────│
│    │      ├── PP-GF20: 유리섬유 20% - 고강도                │
│    │      └── PP-GF30: 유리섬유 30% - 구조용                │
│    │                                                        │
│    ├── PP-LGF (Long Glass Fiber) ──────────────────────────│
│    │      └── PP-LGF30: 장섬유 30% - 최고 강도/내열         │
│    │                                                        │
│    └── PP-EPDM (Rubber modified) ──────────────────────────│
│           └── 내충격 개선, 범퍼/에너지 흡수                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 장단점

| 장점 | 단점 |
|------|------|
| ✅ 최저가 (kg당 1,500~2,000원) | ❌ 도장 밀착 불량 (무극성) |
| ✅ 최경량 (비중 0.9~1.1) | ❌ 내열성 한계 (순수 PP) |
| ✅ 내약품성 우수 | ❌ 저온 충격 취약 |
| ✅ 재활용 용이 | ❌ 표면 경도 낮음 |
| ✅ 대형 부품 성형 용이 | ❌ UV 열화 (안정제 필수) |

#### 적용 부품
- 인스트루먼트 패널 기재 (PP-LGF30)
- 도어 트림 기재 (PP-TD20)
- 필러 트림 (PP-TD20)
- 콘솔 박스 (PP-TD20)
- 글로브 박스 (PP-TD10)
- 트렁크 트림 (PP-TD10)

### 1.2 ABS (Acrylonitrile Butadiene Styrene)

> 외관 부품의 대명사. 도장/도금 적합성 최고

#### 기본 물성

| 특성 | 단위 | 일반 ABS | 내열 ABS | 도금 ABS |
|------|------|----------|----------|----------|
| 비중 | - | 1.04-1.06 | 1.05-1.08 | 1.04-1.06 |
| 인장강도 | MPa | 40-50 | 45-55 | 40-48 |
| 굴곡강도 | MPa | 60-80 | 70-90 | 60-75 |
| 굴곡탄성률 | GPa | 2.0-2.5 | 2.3-2.8 | 2.0-2.4 |
| Izod 충격 | J/m | 200-400 | 150-300 | 180-350 |
| HDT (1.8MPa) | °C | 85-100 | 100-115 | 85-95 |
| 수축률 | % | 0.4-0.7 | 0.4-0.6 | 0.4-0.7 |

#### 3성분 역할

```
ABS = Acrylonitrile + Butadiene + Styrene
          │              │           │
          ▼              ▼           ▼
      내열성/내약품    내충격성     가공성/광택
       (20-30%)       (15-30%)     (40-60%)

고내열 ABS: Acrylonitrile ↑
고충격 ABS: Butadiene ↑
고광택 ABS: Styrene ↑
```

#### 장단점

| 장점 | 단점 |
|------|------|
| ✅ 도장 밀착 우수 (극성) | ❌ PP 대비 2배 비용 |
| ✅ 도금 가능 | ❌ 내후성 취약 (UV) |
| ✅ 고광택 가능 | ❌ 내약품성 보통 |
| ✅ 치수 안정성 양호 | ❌ PP 대비 무거움 |
| ✅ 2차 가공성 우수 | ❌ 연소 시 유해가스 |

#### 적용 부품
- 센터 페시아 (도장)
- 에어컨 베젤 (무광/유광)
- 스위치 베젤
- 오디오 패널
- 도금 장식품 (도어 핸들, 레지스터)

### 2.2 PC/ABS (Polycarbonate/ABS Blend)

> PC의 강도 + ABS의 가공성. 가장 인기 있는 블렌드

#### 기본 물성

| 특성 | 단위 | PC/ABS 일반 | PC/ABS 내열 | PC/ABS 도금 |
|------|------|-------------|-------------|-------------|
| 비중 | - | 1.10-1.15 | 1.12-1.17 | 1.10-1.14 |
| 인장강도 | MPa | 50-60 | 55-65 | 48-55 |
| 굴곡강도 | MPa | 75-90 | 80-95 | 70-85 |
| 굴곡탄성률 | GPa | 2.2-2.6 | 2.4-2.8 | 2.1-2.5 |
| Izod 충격 | J/m | 400-550 | 350-480 | 380-500 |
| HDT (1.8MPa) | °C | 100-115 | 115-130 | 95-105 |
| 수축률 | % | 0.5-0.7 | 0.4-0.6 | 0.5-0.7 |

#### PC:ABS 비율 영향

```
PC:ABS = 70:30  →  고내열, 고강도, 가공성↓
PC:ABS = 60:40  →  균형형 (가장 많이 사용)
PC:ABS = 50:50  →  경제형
PC:ABS = 40:60  →  고가공성, 내열↓

일반적으로: 내열 필요 → PC↑, 가공성 필요 → ABS↑
```

#### 적용 부품
- 센터 페시아 (고급)
- 클러스터 하우징
- 에어벤트 베젤
- 스티어링 휠 커버
- 도어 핸들 베이스

### 2.3 POM (Polyoxymethylene) - 폴리아세탈

> "플라스틱 기어의 왕". 저마찰, 고강도, 내피로

#### 기본 물성

| 특성 | 단위 | 호모 POM | 코폴리머 POM |
|------|------|----------|--------------|
| 비중 | - | 1.42 | 1.41 |
| 인장강도 | MPa | 65-70 | 60-65 |
| 굴곡강도 | MPa | 95-100 | 85-95 |
| 굴곡탄성률 | GPa | 2.8-3.2 | 2.5-2.9 |
| Izod 충격 | J/m | 70-90 | 50-70 |
| HDT (1.8MPa) | °C | 110-125 | 100-115 |
| 마찰계수 | - | 0.15-0.35 | 0.20-0.40 |
| 수축률 | % | 1.8-2.3 | 1.5-2.0 |

#### 호모 vs 코폴리머

```
호모 POM (Delrin®):
├── 높은 결정화도 → 고강도/고강성
├── 중심선 보이드 발생 가능
└── 열안정성 낮음 (포름알데히드 발생)

코폴리머 POM (Duracon®, Celcon®):
├── 더 나은 열안정성
├── 중심선 보이드 적음
└── 약간 낮은 강도
```

#### 적용 부품
- 클립/패스너
- 기어 (윈도우, 시트, 미러)
- 도어 체커
- 연료캡 힌지
- 글로브박스 래치

## Layer 3: 특수/기능성 소재

### 3.1 TPO (Thermoplastic Olefin)

> 에어백 커버와 소프트 터치의 표준

#### 기본 물성

| 특성 | 단위 | TPO 일반 | TPO 고유연 | TPO 에어백 |
|------|------|----------|------------|------------|
| 비중 | - | 0.90-0.95 | 0.88-0.92 | 0.92-0.98 |
| 인장강도 | MPa | 10-20 | 6-12 | 12-18 |
| 파단신율 | % | 300-500 | 400-600 | 350-500 |
| Shore A 경도 | - | 60-90 | 50-70 | 70-85 |
| HDT (0.45MPa) | °C | 50-70 | 45-60 | 55-75 |
| 저온취성 | °C | -40 이하 | -50 이하 | -40 이하 |

#### TPO 특성

```
TPO = PP + EPDM Rubber + Filler

장점:
├── PP와 재활용 호환 (같은 올레핀계)
├── 저비중 (0.9 수준)
├── 소프트 터치 (Shore A 60-90)
├── 저온 충격성 우수
└── 도장 없이 성형색 적용 가능 (In-Mold Grain)

적용:
├── 에어백 커버 ★ (저온에서 안전하게 전개)
├── IP 스킨 (슬러시 성형)
├── 도어 암레스트 스킨
└── 콘솔 소프트 패드
```

### 4.2 Fogging (포깅)

```
Fogging 테스트 (DIN 75201):
├── 유리판에 휘발 성분 응축량 측정
├── 100°C / 16시간 가열
├── Gravimetric: < 2 mg (무게 증가)
├── Reflectometric: > 70% (투과율 유지)
└── 주로 가소제, 저분자량 성분이 원인
```

### 4.3 내광성 (Light Fastness)

```
내광성 테스트 (SAE J2412 등):
├── 제논램프 (Xenon arc) 조사
├── 총 조사량: 200-1000 kJ/m² (OEM별 상이)
├── 그레이스케일 평가: 등급 4 이상 요구
├── △E (색차): < 1.5~3.0 (OEM별 상이)
└── PP, ABS는 UV 안정제 필수
```
