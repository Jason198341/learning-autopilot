# IIT Bombay 협업 기술 - 자동차 내장재 혁신 소재 완전 해설서

## 핵심 개념 요약

IIT Bombay와의 기술 협업은 **친환경 천연섬유 복합소재** 및 **탄소섬유 강화 플라스틱(CFRP)** 기반의 차세대 자동차 내장재 개발을 목표로 합니다. 4개의 특허 기술을 통해 기존 PP/ABS/PC 소재 대비 **경량화, 친환경성, 기계적 물성 향상**을 달성합니다.

# 1. 자동차 내장재 기초

## 1.1 자동차 내장재란?

자동차 **내장재(Interior Materials)**는 차량 내부에 사용되는 모든 재료를 말합니다.

```
┌─────────────────────────────────────────────────────────┐
│                    자동차 내장재 종류                      │
├─────────────────────────────────────────────────────────┤
│  • 대시보드 (Dashboard) - 계기판 주변 플라스틱           │
│  • 도어트림 (Door Trim) - 문 안쪽 패널                  │
│  • 센터콘솔 (Center Console) - 기어박스 주변            │
│  • 시트 (Seat) - 좌석 프레임 및 커버                    │
│  • 헤드라이너 (Headliner) - 천장 내장재                 │
│  • 필러트림 (Pillar Trim) - 기둥 부분 커버              │
│  • 카페트 (Carpet) - 바닥재                            │
└─────────────────────────────────────────────────────────┘
```

## 1.2 왜 새로운 내장재가 필요한가?

### 현재 자동차 산업의 3가지 도전 과제

| 도전 과제 | 설명 | IIT 기술의 해결책 |
|-----------|------|-------------------|
| 환경 규제 강화 | 플라스틱 사용 제한, 재활용 의무화 | 천연섬유 복합재 (바이오 기반) |
| 경량화 요구 | 전기차 시대, 배터리 무게 보상 필요 | 가벼우면서 강한 복합재 |
| 비용 절감 | 원자재 가격 상승 대응 | 저렴한 천연섬유 활용 |

# 2. 기초 과학 개념 완전 정복

## 2.1 원자와 분자 - 모든 물질의 시작점

### 원자(Atom)란?

원자는 물질을 구성하는 가장 작은 단위입니다. 너무 작아서 눈에 보이지 않습니다.

```
            원자의 구조

            ● ● ●      ← 전자(Electron): 음(-)전하
           ●     ●        핵 주위를 빠르게 돌고 있음
          ●   ◉   ●
           ●     ●     ◉ = 원자핵(Nucleus)
            ● ● ●          • 양성자(+): 양전하
                           • 중성자(0): 전하 없음
```

**비유로 이해하기:**
- 원자핵 = 태양
- 전자 = 태양 주위를 도는 행성들
- 원자 전체 크기가 축구장이라면, 원자핵은 축구공 정도!

### 분자(Molecule)란?

분자는 원자들이 결합해서 만들어진 것입니다.

**물 분자 (H₂O)의 예:**

```
    H (수소 원자)
     \
      O (산소 원자)
     /
    H (수소 원자)

→ 수소 2개 + 산소 1개가 결합 = 물 분자 1개
```

---

## 2.2 고분자(Polymer) - 플라스틱의 정체

### 고분자란 무엇인가?

**고분자(高分子, Polymer)**는 "높을 고(高)" + "분자(分子)"로, 아주 큰 분자라는 뜻입니다.

```
┌─────────────────────────────────────────────────────────┐
│                    고분자의 구조                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  단량체(Monomer) = 작은 블록 하나                        │
│       ■                                                │
│                                                         │
│  고분자(Polymer) = 블록이 수천~수만 개 연결된 것          │
│       ■-■-■-■-■-■-■-■-■-■-■-■-■-■-■-■...              │
│                                                         │
│  레고 블록을 상상해보세요!                                │
│  블록 1개 = 단량체                                       │
│  블록 수천 개로 만든 거대한 성 = 고분자                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 고분자의 종류

| 종류 | 특징 | 예시 |
|------|------|------|
| 열가소성(Thermoplastic) | 열을 가하면 녹음, 다시 굳힘 가능 | PE, PP, HDPE |
| 열경화성(Thermoset) | 한 번 굳으면 다시 녹지 않음 | 에폭시, 페놀 |

## 2.3 HDPE (고밀도 폴리에틸렌) 완전 해부

### HDPE란?

**HDPE = High-Density Polyethylene = 고밀도 폴리에틸렌**

### 분자 구조 비교

```
LDPE (저밀도 폴리에틸렌):
    ─────┬─────┬─────┬─────
         │     │     │        ← 가지(branch)가 많음
    ─────┴─────┴─────┴─────      → 분자들이 느슨하게 배열
                                  → 밀도가 낮음, 부드러움

HDPE (고밀도 폴리에틸렌):
    ─────────────────────────  ← 가지가 거의 없음
    ─────────────────────────    → 분자들이 촘촘하게 배열
    ─────────────────────────    → 밀도가 높음, 단단함
```

### HDPE의 화학식

**에틸렌(Ethylene) 단량체:**

```
    H   H
    │   │
    C = C        ← 탄소(C) 2개가 이중결합으로 연결
    │   │          수소(H) 4개가 붙어있음
    H   H
```

**폴리에틸렌(Polyethylene):**

```
    H H   H H   H H   H H
    │ │   │ │   │ │   │ │
  ─C─C─ ─C─C─ ─C─C─ ─C─C─ ─ ...
    │ │   │ │   │ │   │ │
    H H   H H   H H   H H

→ 에틸렌이 수천~수만 개 연결된 것!
→ 화학식: (C₂H₄)n  (n = 반복 횟수, 보통 10,000~100,000)
```

### HDPE의 물리적 특성

| 특성 | 값 | 의미 |
|------|-----|------|
| 밀도 | 0.94~0.97 g/cm³ | 물(1.0)보다 가벼워서 물에 뜸 |
| 녹는점 | 130~137°C | 끓는 물(100°C)에선 안 녹음 |
| 인장강도 | 20~30 MPa | 꽤 강함 (당기는 힘에 버팀) |
| 결정화도 | 70~80% | 분자 배열이 규칙적 = 단단함 |

## 2.4 섬유(Fiber)의 과학

### 섬유란?

섬유는 가늘고 긴 실 모양의 물질입니다.

### 섬유의 분류

```
섬유(Fiber)
├── 천연섬유 (Natural Fiber)
│   ├── 식물성: 면(Cotton), 마(Hemp), 사카룸 문자
│   ├── 동물성: 양모(Wool), 실크(Silk)
│   └── 광물성: 석면(Asbestos) - 현재 금지
│
└── 인조섬유 (Synthetic Fiber)
    ├── 합성섬유: 나일론, 폴리에스터, 탄소섬유
    └── 재생섬유: 레이온, 텐셀
```

### 사카룸 문자(Saccharum Munja) 섬유

```
┌─────────────────────────────────────────────────────────┐
│              사카룸 문자(Saccharum Munja)                 │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  다른 이름: 카나(Kana), 사르칸다(Sarkanda), 문자(Munja)    │
│                                                         │
│  원산지: 인도, 파키스탄, 네팔 등 남아시아 지역             │
│                                                         │
│  특징:                                                   │
│  • 사탕수수과(Saccharum) 식물의 일종                     │
│  • 키가 2~4m까지 자라는 다년생 풀                        │
│  • 인도에서 지붕, 바구니, 밧줄 만드는 데 전통적 사용      │
│  • 매우 저렴하고 풍부함 (잡초처럼 자람)                   │
│  • 연간 수확 가능 (지속 가능한 자원)                     │
│                                                         │
│  화학 성분:                                              │
│  • 셀룰로오스(Cellulose): 45~55%                        │
│  • 헤미셀룰로오스(Hemicellulose): 20~25%                │
│  • 리그닌(Lignin): 15~20%                               │
│  • 기타 (왁스, 펙틴 등): 5~10%                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 셀룰로오스의 분자 구조

셀룰로오스(Cellulose) = 식물 세포벽의 주성분

```
        CH₂OH           CH₂OH           CH₂OH
           │              │              │
       ┌───O───┐     ┌───O───┐     ┌───O───┐
       │       │     │       │     │       │
   ────O       O─────O       O─────O       O────
       │       │     │       │     │       │
       └───────┘     └───────┘     └───────┘
         │  │          │  │          │  │
         OH OH         OH OH         OH OH

→ 포도당(Glucose) 분자가 수천 개 연결된 것
→ 화학식: (C₆H₁₀O₅)n
→ 나무, 종이, 면의 주성분과 동일!
```

### 섬유 종류별 비교

| 종류 | 예시 | 강도 | 가격 |
|------|------|------|------|
| 천연섬유 | 황(Saccharum Munja), 대마, 아마 | 중간 | 저가 |
| 유리섬유 | E-glass | 높음 | 중가 |
| 탄소섬유 | T700, T800 | 매우 높음 | 고가 |

## 2.5 복합재료(Composite)의 원리

### 복합재료란?

복합재료는 두 가지 이상의 서로 다른 재료를 합쳐서 각각의 단점은 줄이고 장점은 살린 새로운 재료입니다.

```
┌─────────────────────────────────────────────────────────┐
│                 복합재료의 기본 구조                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│    ═══════════════════════════════════  ← 섬유(강화재)   │
│    ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  ← 매트릭스      │
│    ═══════════════════════════════════  ← 섬유(강화재)   │
│    ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░  ← 매트릭스      │
│    ═══════════════════════════════════  ← 섬유(강화재)   │
│                                                         │
│    • 매트릭스(Matrix): 섬유를 감싸고 지지하는 재료        │
│      - 플라스틱(HDPE, 에폭시 등)                         │
│      - 힘을 섬유로 전달하는 역할                         │
│                                                         │
│    • 강화재(Reinforcement): 실제로 힘을 버티는 재료      │
│      - 섬유(유리섬유, 탄소섬유, 천연섬유 등)              │
│      - 복합재료의 강도를 결정                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 일상생활의 복합재료 예시

| 복합재료 | 매트릭스 | 강화재 | 용도 |
|----------|----------|--------|------|
| 철근 콘크리트 | 콘크리트 | 철근 | 건물, 다리 |
| 유리섬유 강화 플라스틱(FRP) | 폴리에스터 | 유리섬유 | 욕조, 보트 |
| 탄소섬유 강화 플라스틱(CFRP) | 에폭시 | 탄소섬유 | 비행기, 스포츠카 |
| IIT 제안 복합재 | HDPE | 사카룸 문자 | 자동차 내장재 |

# 3. 4대 핵심 특허 기술

## 특허 1: Saccharum Munja + HDPE 복합소재

### 특허 정보
- **특허번호**: IN202421056908A
- **발명자**: 아리아 니틴 쿠마르, 싱 아파르나
- **기관**: 인도 봄베이 공과대학교 (IIT Bombay)

### 기술 개요
- **Saccharum Munja**: 인도 자생 갈대 유사 식물
- 천연섬유와 HDPE를 혼합한 친환경 복합소재
- 자동차 내장재 (도어 패널, 시트 백 등) 적용

---

### 3.1 핵심 기술 설명 - 제조 공정 6단계

#### [단계 a] 원료 혼합

```
┌─────────────────────────────────────────────────────────┐
│  잘게 자른 사카룸 문자 섬유 + HDPE 펠릿 혼합             │
│                                                         │
│  ★ 펠릿(Pellet)이란?                                    │
│    - 작은 알갱이 형태의 플라스틱 원료                     │
│    - 보통 직경 3~5mm의 원통형 또는 구형                   │
│    - 운반, 보관, 투입이 편리함                           │
│                                                         │
│  ★ 왜 섬유를 잘게 자르나?                               │
│    - 너무 길면 기계에 걸림                               │
│    - 짧은 섬유가 플라스틱과 잘 섞임                      │
│    - 보통 3~6mm 길이로 절단                             │
└─────────────────────────────────────────────────────────┘
```

#### [단계 b] 트윈 스크류 압출기 투입

```
┌─────────────────────────────────────────────────────────┐
│  ★ 트윈 스크류 압출기(Twin Screw Extruder)란?           │
│                                                         │
│        ┌─────────────────────────────────────┐          │
│        │  ≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋→   │ 출구     │
│  호퍼 →│  ≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋≋→   │→         │
│  (투입구)└─────────────────────────────────────┘  (다이)  │
│              ↑ 2개의 스크류가 서로 맞물려 회전           │
│                                                         │
│  • 스크류 = 나사 모양의 회전축                           │
│  • 싱글 스크류: 스크류 1개 (일반적)                      │
│  • 트윈 스크류: 스크류 2개 (혼합 능력 우수)              │
│                                                         │
│  트윈 스크류의 장점:                                     │
│  1. 섬유와 플라스틱을 더 균일하게 혼합                   │
│  2. 섬유 손상이 적음                                     │
│  3. 연속 생산 가능                                       │
└─────────────────────────────────────────────────────────┘
```

#### [단계 c] 150~200°C에서 처리

```
┌─────────────────────────────────────────────────────────┐
│  ★ 왜 150~200°C인가?                                   │
│                                                         │
│  • HDPE 녹는점: 130~137°C                               │
│    → 150°C 이상이면 HDPE가 완전히 녹아 액체 상태         │
│                                                         │
│  • 사카룸 문자 섬유 분해 온도: 약 220~250°C              │
│    → 200°C 이하면 섬유가 타지 않고 유지됨                │
│                                                         │
│  ┌────────────────────────────────────────────┐         │
│  │  100°C    150°C    200°C    250°C          │         │
│  │    │        │        │        │            │         │
│  │    ├────────┼────────┼────────┤            │         │
│  │         [적정 온도 범위]   │                │         │
│  │    │   HDPE 녹음    │ 섬유 손상 시작        │         │
│  └────────────────────────────────────────────┘         │
│                                                         │
│  온도가 중요한 이유:                                     │
│  - 너무 낮으면: HDPE가 안 녹아 섬유 코팅 불량            │
│  - 너무 높으면: 섬유가 타서 검게 변하고 강도 저하         │
└─────────────────────────────────────────────────────────┘
```

#### [단계 d] 출구 영역으로 운반

```
┌─────────────────────────────────────────────────────────┐
│  스크류 회전에 의해 혼합물이 앞으로 이동                   │
│  이 과정에서:                                            │
│  • 섬유와 HDPE가 완전히 혼합됨                           │
│  • 공기 방울이 제거됨                                    │
│  • 균일한 온도로 유지됨                                  │
└─────────────────────────────────────────────────────────┘
```

#### [단계 e] 사출 성형기로 이송

```
┌─────────────────────────────────────────────────────────┐
│  ★ 사출 성형(Injection Molding)이란?                    │
│                                                         │
│      호퍼(원료투입)                                      │
│          ↓                                              │
│    ┌─────────────┐                                      │
│    │ ≋≋≋≋≋≋≋≋≋≋≋ │→ 노즐 → ┌──────────┐               │
│    │  가열 실린더 │         │  금형    │               │
│    └─────────────┘         │ (틀)    │               │
│                            │          │               │
│                            │ 제품모양 │               │
│                            └──────────┘               │
│                                                         │
│  • 녹은 플라스틱을 틀(금형)에 주입                       │
│  • 틀 모양대로 제품이 만들어짐                           │
│  • 대시보드, 도어트림 등 복잡한 형상 제작 가능            │
└─────────────────────────────────────────────────────────┘
```

#### [단계 f] 최종 성형 조건

```
┌─────────────────────────────────────────────────────────┐
│  • 금형 온도: 45~55°C                                   │
│    → 금형이 너무 뜨거우면 제품이 잘 안 굳음               │
│    → 너무 차가우면 플라스틱이 금형에 닿자마자 굳어버림    │
│                                                         │
│  • 사출 압력: 280~320 bar                               │
│    → bar = 압력 단위 (1 bar ≈ 대기압)                   │
│    → 280 bar = 대기압의 280배!                          │
│    → 자동차 타이어 공기압(약 2.5 bar)의 약 100배         │
│                                                         │
│    왜 이렇게 높은 압력이 필요한가?                       │
│    • 금형 구석구석까지 재료를 밀어넣기 위해              │
│    • 섬유가 있어서 점도가 높아졌기 때문                  │
│    • 기포 없는 제품을 만들기 위해                       │
└─────────────────────────────────────────────────────────┘
```

### 3.2 과학적 원리 상세 설명

#### 섬유와 매트릭스의 결합 원리

```
┌─────────────────────────────────────────────────────────┐
│              섬유-매트릭스 계면 결합                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  문제점: 천연섬유(친수성) + HDPE(소수성) = 안 붙음!       │
│                                                         │
│  ★ 친수성(Hydrophilic)이란?                             │
│    - 물을 좋아하는 성질                                  │
│    - 천연섬유의 셀룰로오스는 -OH기(수산기)가 많음         │
│    - OH기가 물 분자와 수소결합을 형성                    │
│                                                         │
│        H                                                │
│        │                                                │
│    ─O─H ··· O─H    ← 수소결합(점선)                    │
│            │                                            │
│            H (물)                                       │
│                                                         │
│  ★ 소수성(Hydrophobic)이란?                             │
│    - 물을 싫어하는 성질                                  │
│    - HDPE는 탄소(C)와 수소(H)만 있음                    │
│    - 극성이 없어서 물과 결합 안 됨                       │
│                                                         │
│  해결책: 기계적 결합 (Mechanical Interlocking)           │
│                                                         │
│       섬유 표면                                          │
│       ╱╲╱╲╱╲╱╲╱╲╱╲     ← 섬유 표면의 요철               │
│      ████████████████   ← HDPE가 요철 사이로 침투        │
│                                                         │
│  트윈 스크류 압출기의 역할:                              │
│  • 강한 전단력(Shear Force)으로 섬유 표면 활성화         │
│  • HDPE를 섬유 깊숙이 침투시킴                          │
│  • 결합력 향상                                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 복합재료의 강도 증가 원리

```
┌─────────────────────────────────────────────────────────┐
│            응력 전달 메커니즘 (Stress Transfer)           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  외부에서 힘이 가해지면:                                  │
│                                                         │
│     힘 →  ████████████████████████  ← 힘               │
│           ════════════════════════   (섬유)             │
│           ████████████████████████                      │
│           ════════════════════════   (섬유)             │
│           ████████████████████████                      │
│                                                         │
│  1. 먼저 매트릭스(HDPE)가 힘을 받음                      │
│  2. 매트릭스가 섬유로 힘을 전달                          │
│  3. 섬유가 실제로 힘을 버팀                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 재료별 인장강도 비교

| 재료 | 인장강도(MPa) | 비교 |
|------|---------------|------|
| HDPE | 20~30 | 기준 |
| 사카룸 문자 섬유 | 100~200 | 5~10배 강함 |
| 탄소섬유 | 3000~7000 | 100~300배 강함 |

→ **약한 HDPE + 강한 섬유 = 중간 정도의 강한 복합재!**

### 4.1 3D 프린팅 기술 설명

#### 3D 프린팅(적층 제조)이란?

```
┌─────────────────────────────────────────────────────────┐
│               3D 프린팅의 원리 (FDM 방식)                 │
├─────────────────────────────────────────────────────────┤
│                                                         │
│      필라멘트 (실 형태의 재료)                           │
│           │                                             │
│           ↓                                             │
│      ┌─────────┐                                        │
│      │ 노즐    │ ← 200~250°C로 가열                     │
│      │(가열부) │                                        │
│      └────┬────┘                                        │
│           │ 녹은 재료                                    │
│           ↓                                             │
│      ════════════  ← 1층                                │
│      ════════════  ← 2층                                │
│      ════════════  ← 3층                                │
│      ▓▓▓▓▓▓▓▓▓▓▓▓  ← 빌드 플레이트(바닥판)              │
│                                                         │
│  • 한 층씩 쌓아서 3차원 물체를 만듦                      │
│  • 층 두께: 보통 0.1~0.3mm                              │
│  • 복잡한 형상도 금형 없이 제작 가능                     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### HDPE 3D 프린팅의 문제점

```
┌─────────────────────────────────────────────────────────┐
│              HDPE 3D 프린팅이 어려운 이유                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  문제 1: 수축(Shrinkage)                                │
│  • HDPE는 냉각 시 부피가 2~4% 수축                      │
│  • 제품 치수가 달라짐                                   │
│                                                         │
│  문제 2: 뒤틀림(Warping)                                │
│  • 층마다 냉각 속도가 달라서 발생                        │
│  • 큰 부품일수록 심함                                   │
│                                                         │
│  문제 3: 기포 형성(Void Formation)                      │
│  • 층 사이에 공기가 갇힘                                │
│  • 강도가 크게 약해짐                                   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 사카룸 문자 섬유가 문제를 해결하는 원리

| 문제 | 해결 원리 | 효과 |
|------|-----------|------|
| 수축 | 섬유는 냉각해도 수축 안 함, HDPE 수축을 물리적으로 막음 | 수축률 4% → 1~2% |
| 뒤틀림 | 섬유가 수축 방향의 힘을 분산 | 뒤틀림 현저히 감소 |
| 기포 | 섬유가 층 사이를 가교(Bridge) | 층간 접착력 향상 |
