# IIT Bombay 협업 기술 - 자동차 내장재 혁신 소재

## 핵심 개념 요약

IIT Bombay와의 기술 협업은 **친환경 천연섬유 복합소재** 및 **탄소섬유 강화 플라스틱(CFRP)** 기반의 차세대 자동차 내장재 개발을 목표로 합니다. 4개의 특허 기술을 통해 기존 PP/ABS/PC 소재 대비 **경량화, 친환경성, 기계적 물성 향상**을 달성합니다.

---

## 기초 과학 개념

### 1. 원자와 분자
- **원자**: 물질의 최소 단위 (양성자, 중성자, 전자로 구성)
- **분자**: 원자들이 화학 결합으로 연결된 구조
- **예시**: 물(H₂O) = 수소 2개 + 산소 1개

### 2. 고분자(Polymer)
- **정의**: "많은(Poly) 부분(mer)"이 반복 연결된 거대 분자
- **예시**: 플라스틱, 고무, 단백질, DNA
- **구조**: 단량체(monomer)가 수천~수만 개 연결

### 3. HDPE (고밀도 폴리에틸렌)
| 특성 | 값 |
|------|-----|
| 밀도 | 0.93~0.97 g/cm³ |
| 인장강도 | 20~37 MPa |
| 내열성 | 80~120°C |
| 특징 | 우유병, 파이프, 장난감에 사용 |

### 4. 섬유(Fiber) 종류
| 종류 | 예시 | 강도 | 가격 |
|------|------|------|------|
| 천연섬유 | 황, 대마, 아마 | 중간 | 저가 |
| 유리섬유 | E-glass | 높음 | 중가 |
| 탄소섬유 | T700, T800 | 매우 높음 | 고가 |

### 5. 복합재료(Composite)
- **정의**: 두 가지 이상 재료를 결합하여 새로운 특성 창출
- **구성**: 기지재(Matrix) + 강화재(Reinforcement)
- **예시**: 콘크리트(시멘트+자갈), CFRP(에폭시+탄소섬유)

## 4대 핵심 특허 기술

### 특허 1: Saccharum Munja + HDPE 복합소재
**출원번호**: IN202421056908A

#### 기술 개요
- **Saccharum Munja**: 인도 자생 갈대 유사 식물
- 천연섬유와 HDPE를 혼합한 친환경 복합소재
- 자동차 내장재 (도어 패널, 시트 백 등) 적용

#### 제조 공정
1. 천연섬유 전처리 (알칼리 처리)
2. 섬유-HDPE 혼합 (압출기)
3. 펠렛 제조
4. 사출/압축 성형

#### 기대 효과
- 기존 PP 대비 10~15% 경량화
- 탄소 발자국 30% 감소
- 원가 5~10% 절감 가능

### 특허 2: 3D 프린팅 친환경 복합소재
**출원번호**: IN202521035215A

#### 기술 개요
- 천연섬유 강화 바이오 기반 필라멘트
- FDM/FFF 방식 3D 프린팅 적용
- 시제품/소량 생산에 최적화

#### 핵심 구성
| 성분 | 비율 | 역할 |
|------|------|------|
| PLA/PBS | 60~70% | 기지재 |
| 천연섬유 | 20~30% | 강화재 |
| 첨가제 | 5~10% | 커플링제, 가소제 |

#### 적용 분야
- 시제품 신속 제작
- 맞춤형 내장 부품
- 소량 다품종 생산

---

### 특허 3: CFRP 굽힘 강도 향상
**출원번호**: IN202321047360A

#### 기술 개요
- 탄소섬유 복합재의 굽힘 강도 30% 이상 향상
- 적층 구조 및 계면 최적화
- 구조 부품 적용 가능

#### 핵심 기술
1. **적층 각도 최적화**: [0°/90°/±45°] 하이브리드 적층
2. **계면 개선**: 탄소섬유 표면 처리 (산화, 플라즈마)
3. **수지 시스템**: 고인성 에폭시 + 강인화제

#### 성능 비교
| 항목 | 기존 CFRP | 개선 CFRP |
|------|-----------|-----------|
| 굽힘 강도 | 800 MPa | 1,040+ MPa |
| 굽힘 탄성률 | 50 GPa | 55+ GPa |
| 층간 전단강도 | 45 MPa | 55+ MPa |

### 특허 4: HL+VARTM 하이브리드 제조
**출원번호**: IN202321066950A

#### 기술 개요
- Hand Layup + VARTM 결합 공정
- 대형 복잡 형상 CFRP 부품 제조
- 품질 균일성 및 생산성 향상

#### 공정 순서
```
1. 몰드 이형 처리
2. 겔코트 도포
3. Hand Layup (외층 적층)
4. 진공 백 밀봉
5. VARTM 수지 주입
6. 경화 (상온 or 가열)
7. 탈형 및 후가공
```

#### 장점
| 항목 | Hand Layup Only | HL+VARTM |
|------|-----------------|----------|
| 섬유 체적 분율 | 30~40% | 50~60% |
| 기공률 | 3~5% | <1% |
| 두께 균일성 | ±15% | ±5% |
| 생산 시간 | 기준 | 20% 단축 |

## 기존 내장재 소재 비교

### 현재 사용 소재
| 소재 | 용도 | 장점 | 단점 |
|------|------|------|------|
| PP | 도어 트림, 대시보드 | 저가, 가공 용이 | 낮은 강도 |
| ABS | 콘솔, 트림 | 우수한 충격 강도 | 내열성 한계 |
| PC/ABS | 계기판 | 고강도, 내열 | 고가 |
| PU 폼 | 시트 쿠션 | 편안함 | 재활용 어려움 |

### IIT 기술 장점
1. **경량화**: 10~20% 중량 감소
2. **친환경**: 탄소 배출 30~50% 감소
3. **지속가능성**: 천연섬유/바이오 수지 활용
4. **성능**: 기계적 물성 유지/향상

## SWOT 분석

### Strengths (강점)
- IIT Bombay의 재료공학 전문성
- 인도 정부의 친환경 정책 지원
- 특허로 보호된 독자 기술
- 현지 원료 (Saccharum Munja) 활용

### Weaknesses (약점)
- 양산 검증 필요
- 초기 설비 투자 필요
- 품질 일관성 확보 과제

### Opportunities (기회)
- 인도 자동차 시장 성장
- CAFE Phase 3 경량화 요구
- ESG 경영 트렌드
- 전기차 배터리 중량 상쇄 필요

### Threats (위협)
- 경쟁사 유사 기술 개발
- 천연섬유 공급 안정성
- 글로벌 OEM 인증 기준

## 용어 사전

| 용어 | 영문 | 설명 |
|------|------|------|
| 기지재 | Matrix | 복합재에서 섬유를 감싸는 재료 |
| 강화재 | Reinforcement | 복합재에서 강도를 담당하는 재료 |
| 계면 | Interface | 기지재와 강화재의 접촉면 |
| 커플링제 | Coupling Agent | 계면 결합력 향상 첨가제 |
| 적층 | Lamination | 여러 층을 쌓아 올리는 것 |
| 함침 | Impregnation | 섬유에 수지를 스며들게 하는 것 |
| 경화 | Curing | 수지가 굳어지는 화학 반응 |
| 탈형 | Demolding | 경화 후 몰드에서 분리 |

- 친환경 소재 시장 선도
