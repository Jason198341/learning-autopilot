> 작성일: 2025-02-04
> 수준: 세계 최고 전문가급 내용 + 대학 1학년 설명 방식

## 1. 개요

### 1.1 학습 주제
**자동차 칵핏 설계 (Automotive Cockpit Design)**
- 운전자와 차량이 만나는 모든 인터페이스의 설계 철학과 실무

### 1.2 설명 철학
```
전문가 수준 내용 ──→ 신입생 수준 설명
"왜?"를 끊임없이 설명
비유와 일상 예시 적극 활용
전문용어는 반드시 풀어서 설명
```

### 1.3 학습 목표
- [ ] 칵핏의 정의와 구성 요소를 설명할 수 있다
- [ ] 인간공학 기반 설계 원리를 이해한다
- [ ] IP/콘솔/스티어링 설계 핵심을 파악한다
- [ ] HMI(Human-Machine Interface) 트렌드를 안다
- [ ] 안전 규제와 설계 제약을 이해한다

### 1.4 범위
| 포함 | 제외 |
|------|------|
| 인스트루먼트 패널 (IP) | 시트 설계 상세 |
| 센터 콘솔/페시아 | 도어 트림 상세 |
| 스티어링 휠/칼럼 | 헤드라이너 |
| 클러스터/AVN | 외장 설계 |
| 페달/조작계 | 파워트레인 |
| 에어백 연동 설계 | |

**심층 노트 - "진짜 노하우"**

  - 시야각과 사각지대
- [ ] **IP 설계**
  - 단면 구성 (상단/중단/하단)

- [ ] **클러스터 설계**
  - 아날로그 → 디지털 전환

- [ ] **센터 페시아 설계**
  - AVN 화면 크기/각도
  - 공조 조작부 배치
  - 블라인드 조작 가능성
- [ ] **스티어링 휠 설계**
  - 그립 단면과 인간공학
  - 스위치 배치 원칙
  - 에어백 모듈 통합
- [ ] **콘솔 설계**
  - 변속기 타입별 차이
  - 암레스트 높이/위치
  - 수납공간 최적화
