## ✅ Resolved Questions

### Q: 왜 운전석이 왼쪽/오른쪽으로 나뉘나요?

**Date resolved:** 2025-02-04

**Answer:**
```
좌측통행 국가 (영국, 일본, 호주)
→ 운전석이 오른쪽 (RHD: Right-Hand Drive)
→ 이유: 마주오는 차를 잘 보려고

우측통행 국가 (한국, 미국, 유럽)
→ 운전석이 왼쪽 (LHD: Left-Hand Drive)
→ 이유: 마주오는 차를 잘 보려고

역사적 배경:
· 마차 시대: 채찍 든 손이 반대편 마차 안 때리게
· 나폴레옹: 우측통행 전파 (적국 영국과 반대로)
```

### Q: 에어백은 왜 그렇게 빨리 펴지나요?

**Date resolved:** 2025-02-04

**Answer:**
```
충돌 순서:
0 ms:     충돌 감지
15 ms:    에어백 전개 시작
30 ms:    에어백 완전 전개 ← 0.03초!
100 ms:   운전자 머리가 에어백에 닿음

왜 빨라야 하나?
· 시속 50km 충돌 = 0.1초 만에 운전자가 핸들에 부딪힘
· 에어백이 느리면 이미 부딪힌 후에 펴짐 = 소용없음
· 그래서 0.03초 만에 펴져야 함!

설계 영향:
· 에어백 전개 공간 확보 필수
· 커버 재질 = 순간적으로 찢어져야 함
· 경고: 에어백 근처에 물건 두지 말 것
```

### Q: 클러스터는 왜 핸들 뒤에 있나요?

**Date resolved:** 2025-02-04

**Answer:**
```
이유 1: 시선 이동 최소화
    ┌────┐
    │도로 │ ← 여기서 가장 가까운 곳 = 핸들 뒤
    └────┘
      ↓
    ┌────┐
    │클러│ ← 살짝만 내려보면 됨
    └────┘

이유 2: 가림 방지
    핸들 위로 보이도록 배치
    (핸들에 가려지면 안 됨)

이유 3: 인간공학
    눈높이에서 15~25° 아래
    = 자연스러운 시선 각도
```
