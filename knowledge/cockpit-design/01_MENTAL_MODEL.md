## 🖼️ 칵핏 = 운전자의 사무실

### 비유로 이해하기

```
일반 사무실                     자동차 칵핏
═══════════                    ═══════════

모니터  ─────────────────→  클러스터/HUD
(정보 확인)                    (속도, 연료 확인)

키보드/마우스 ────────────→  스티어링/페달
(입력 도구)                    (조작 도구)

책상 위 물건 ─────────────→  센터페시아/콘솔
(자주 쓰는 것)                 (에어컨, 내비)

서랍/수납 ───────────────→  글로브박스/콘솔박스
(덜 쓰는 것)                   (서류, 충전기)

의자 ────────────────────→  시트
(앉는 곳)                      (앉는 곳)
```

> 💡 사무실에서 가장 중요한 건? **내 손이 편하게 닿는 배치!**
> 칵핏도 마찬가지예요!

## 📐 칵핏의 3개 영역

### 기능별로 나누면?

```
┌─────────────────────────────────────────────────────┐
│                    < 칵핏 3개 영역 >                   │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ╔═══════════════════════════════════════════════╗ │
│  ║  1️⃣ 정보 영역 (INFORMATION)                    ║ │
│  ║     "운전에 필요한 정보를 본다"                   ║ │
│  ║     → 클러스터, HUD, 경고등                     ║ │
│  ╚═══════════════════════════════════════════════╝ │
│                         ↓                           │
│  ╔═══════════════════════════════════════════════╗ │
│  ║  2️⃣ 조작 영역 (CONTROL)                        ║ │
│  ║     "차를 직접 조종한다"                         ║ │
│  ║     → 스티어링, 페달, 변속기, 버튼               ║ │
│  ╚═══════════════════════════════════════════════╝ │
│                         ↓                           │
│  ╔═══════════════════════════════════════════════╗ │
│  ║  3️⃣ 편의 영역 (COMFORT)                        ║ │
│  ║     "운전을 편하게 한다"                         ║ │
│  ║     → 에어컨, 오디오, 수납, 컵홀더               ║ │
│  ╚═══════════════════════════════════════════════╝ │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 각 영역의 우선순위

| 순위 | 영역 | 이유 | 위치 |
|------|------|------|------|
| 1 | 정보 | 모르면 사고 | 눈높이 정면 |
| 2 | 조작 | 빨라야 안전 | 손 닿는 곳 |
| 3 | 편의 | 없어도 운전 가능 | 나머지 공간 |

## 👁️ 운전자의 눈은 어디를 볼까?

### Eye Scanning Pattern (시선 흐름)

```
운전 중 눈의 움직임 (%)
═══════════════════════════════════════════

    ┌─────────────────────────────────────┐
    │                                     │
    │         ┌─────────────────┐         │
    │         │    전방 도로     │         │
    │         │      70%        │  ← 대부분 여기!
    │         └─────────────────┘         │
    │                                     │
    │  ┌─────────┐       ┌─────────┐     │
    │  │ 좌측미러 │       │ 우측미러 │     │
    │  │   8%    │       │   5%    │     │
    │  └─────────┘       └─────────┘     │
    │                                     │
    │         ┌─────────────────┐         │
    │         │    클러스터     │         │
    │         │      10%       │  ← 속도 확인
    │         └─────────────────┘         │
    │                                     │
    │         ┌─────────────────┐         │
    │         │   센터 디스플레이 │         │
    │         │      5%        │  ← 내비 등
    │         └─────────────────┘         │
    │                                     │
    │         ┌─────────────────┐         │
    │         │   룸미러        │         │
    │         │      2%        │         │
    │         └─────────────────┘         │
    │                                     │
    └─────────────────────────────────────┘

※ 중요: 전방에서 눈을 떼는 시간을 최소화해야 안전!
```

### 시선 이동 거리 규칙

> 🚨 **2초 룰**: 전방에서 눈을 떼는 시간 = 최대 2초
> 시속 100km → 2초 = 약 56m 이동!

```
좋은 설계: 시선 이동 거리 ↓ = 빨리 보고 돌아옴
나쁜 설계: 시선 이동 거리 ↑ = 오래 봐야 함 → 위험!

클러스터 위치가 아래쪽? → 고개 숙여야 함 → 위험
클러스터 위치가 눈높이? → 살짝만 내려봄 → 안전
```

## ✋ 손 닿는 범위 (Reach Envelope)

### 팔 뻗으면 어디까지 닿을까?

```
                    [ 운전자 상체 ]
                          │
            ┌─────────────┴─────────────┐
            │                           │
   ┌────────┴────────┐       ┌────────┴────────┐
   │   왼손 영역      │       │    오른손 영역    │
   │  (스티어링)      │       │  (변속기/콘솔)    │
   └────────┬────────┘       └────────┬────────┘
            │                           │
            ▼                           ▼

    ╔═══════════════════════════════════════════╗
    ║           리치 영역 (Reach Envelope)        ║
    ╠═══════════════════════════════════════════╣
    ║                                           ║
    ║    Zone A        Zone B        Zone C     ║
    ║  ┌───────┐    ┌───────┐    ┌───────┐    ║
    ║  │ 0-20cm │    │20-40cm│    │40-60cm│    ║
    ║  │       │    │       │    │       │    ║
    ║  │ 자주  │    │ 가끔  │    │ 드물게 │    ║
    ║  │ 쓰는것│    │ 쓰는것│    │ 쓰는것 │    ║
    ║  └───────┘    └───────┘    └───────┘    ║
    ║                                           ║
    ║  예: 핸들    예: 에어컨   예: 글로브박스   ║
    ║      볼륨        내비         수납        ║
    ║                                           ║
    ╚═══════════════════════════════════════════╝
```

### Zone별 배치 원칙

| Zone | 거리 | 배치할 것 | 이유 |
|------|------|-----------|------|
| **A** | 0-20cm | 핸들, 주행 버튼 | 눈 안 떼고 조작 |
| **B** | 20-40cm | 에어컨, 오디오 | 가끔 조작 |
| **C** | 40-60cm | 내비 화면, 수납 | 정차 시 조작 |

```

══════════════════════════════════════════════
   "어떤 차를 만들 건가?"

   └──────────────┘
              ↓

Phase 3: 스타일링 (Styling)
══════════════════════════════════════════════
   "어떻게 생겼을 건가?"

   ┌──────────────┐
   │ 콘셉트 스케치  │ → 여러 아이디어

   │ 디지털 모델   │ → 3D로 그림
   └──────────────┘
              ↓

Phase 4: 설계 (Engineering)
══════════════════════════════════════════════
   "진짜로 만들 수 있는가?"

   ┌──────────────┐
   │ 부품 설계     │ → 각 부품 도면
   │ 구조 해석     │ → 강도, 충돌 시뮬레이션
   │ 인간공학 검증  │ → 사람이 쓰기 편한가?
   └──────────────┘
              ↓

Phase 5: 시작 (Prototype)
══════════════════════════════════════════════
   "실제로 만들어서 테스트"

   ┌──────────────┐
   │ 시작품 제작   │ → 실제 차에 조립
   │ 평가/개선     │ → 문제점 발견 → 수정
   │ 양산 준비     │ → 공장에서 만들 준비
   └──────────────┘
```

## 🧭 Mental Shortcuts (빠른 판단법)

### 1️⃣ "3초 룰"
```
처음 보는 기능을 3초 안에 이해 못하면?
→ 나쁜 설계!

예: 에어컨 버튼이 어디 있는지 3초 안에 못 찾으면 실패
```

### 2️⃣ "할머니 테스트"
```
내 할머니도 조작할 수 있으면?
→ 좋은 설계!

예: 스마트폰 같은 제스처보다 → 눌러서 딸깍 소리 나는 버튼
```

### 3️⃣ "긴급 상황 테스트"
```
위험 경고가 떴을 때 바로 알아볼 수 있으면?
→ 안전한 설계!

예: 빨간색 경고등 = 즉시 정차
    주황색 경고등 = 빨리 점검
```

### 4️⃣ "왼손 오른손 법칙"
```
운전 중 자주 쓰는 것 → 왼손 (핸들에서 가까움)
정차 중 쓰는 것 → 오른손 (콘솔 쪽)

예: 와이퍼, 방향지시등 → 핸들 레버 (왼손)
    에어컨, 내비 → 센터 페시아 (오른손)
```
