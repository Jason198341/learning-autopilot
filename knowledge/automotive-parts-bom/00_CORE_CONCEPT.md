# CORE CONCEPT: 자동차 부품 BOM (Bill of Materials)

> 자동차를 구성하는 핵심 부품들의 구조, 재질, 기능 총정리

---

## 🎯 한 문장 정의

**자동차 부품 BOM**은 차량을 구성하는 모든 부품의 목록과 구조를 체계화한 것으로, 각 부품의 재질, 무게, 원가, 공급업체, 조립 위치 등을 포함합니다.

---

## 📋 부품 카테고리 분류

```
자동차 부품 대분류:
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   1. 외장 부품 (Exterior)                                   │
│      ├─ 범퍼 (Bumper)                                      │
│      ├─ 펜더 (Fender)                                      │
│      ├─ 도어 (Door)                                        │
│      ├─ 후드/트렁크 (Hood/Trunk)                           │
│      ├─ 연료도어 (Fuel Door)                               │
│      ├─ 아웃사이드 핸들 (Outside Handle)                   │
│      ├─ 사이드 미러 (Side Mirror)                          │
│      └─ 언더커버 (Undercover)                              │
│                                                             │
│   2. 내장 부품 (Interior)                                   │
│      ├─ IP (Instrument Panel)                              │
│      ├─ 도어 트림 (Door Trim)                              │
│      ├─ 필러 트림 (Pillar Trim)                            │
│      ├─ 헤드라이닝 (Headlining)                            │

│      ├─ 콘솔 (Console)                                     │
│      └─ 시트 (Seat)                                        │
│                                                             │
│   3. 안전 부품 (Safety)                                     │
│      ├─ 시트벨트 (Seat Belt)                               │

│      └─ 충격흡수재 (Crash Pad)                             │
│                                                             │
│   4. 연료계 부품 (Fuel System)                              │
│      ├─ 연료탱크 (Fuel Tank)                               │

│      ├─ 윈드실드 (Windshield)                              │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐

│   ─────────────────────────────────────────────────         │

│   · ECE R16 (유럽)                                         │

│                                                             │

```
